# cineskin - CIN_027

## 概要

映画スキンデザインとビデオエフェクトのポストプロダクションを、ワンストップで実現するプロフェッショナルツール。cineskinは、映像制作の現場で求められる高精度なカラーグレーディング、テクスチャ合成、エフェクト管理を、直感的なUIと強力なバックエンドで支える。特に、映画的な質感（フィルムグレイン、フレア、ヴィネット）をリアルタイムに適用できる点が、他ツールとの決定的な差別化要因だ。

## コア機能

### 1. ダイナミックフィルムグレインエンジン
単なるノイズ追加ではなく、フィルムストックごとの粒子構造（Kodak Vision3、Fuji Eternaなど）を再現。ISO感度、露光時間、現像処理パラメータを調整可能。4K/8K解像度でも、フレーム間の一貫性を保ちながら、自然な粒状感を生成する。

### 2. マルチレイヤーエフェクトスタッキング
ベースレイヤー（カラーグレーディング）、エフェクトレイヤー（レンズフレア、ブラー、グリッチ）、マスクレイヤー（アルファチャンネル制御）を、非破壊で重ね合わせ。各レイヤーはブレンドモード（スクリーン、オーバーレイ、ソフトライトなど）と不透明度を個別設定可能。GPUアクセラレーション対応で、プレビューもスムーズ。

### 3. リアルタイムプリセットスワップ
「ノワール」「サイバーパンク」「レトロ80s」などの映画ジャンル別プリセットを、ワンクリックで適用。プリセットは、カーブ、LUT、テクスチャ、エフェクトパラメータをセットで保存。さらに、プリセット間のクロスフェード（0.5秒〜5秒）を設定可能で、トランジション効果としても利用できる。

## 使い方

### 1. プロジェクトセットアップとLUT適用
ターミナルで`cineskin init`を実行し、プロジェクトディレクトリを作成。次に、`cineskin import --source /path/to/footage.mp4`でソースファイルをインポート。最後に、`cineskin lut apply --lut ./luts/vision3_250D.cube`で、Cinema-gradeのLUTをベースレイヤーに適用。この時、`--strength 0.8`で強度を調整可能。

### 2. エフェクトレイヤーの追加とマスク
`cineskin layer add --type effect --name "flair"`でエフェクトレイヤーを作成。次に、`cineskin effect set --layer flair --param "lens_flare_type:anamorphic" --param "intensity:0.6"`で、アナモルフィックレンズフレアを設定。さらに、`cineskin mask create --layer flair --shape ellipse --center 0.5,0.5 --radius 0.3,0.2`で、楕円マスクを適用し、フレアの発生範囲を限定。

### 3. プリセットのカスタム保存とエクスポート
`cineskin preset save --name "my_noir" --layers base,flair,texture`で、現在のレイヤー構成をプリセットとして保存。次に、`cineskin preset apply --name "my_noir" --crossfade 2.0`で、2秒のクロスフェードでプリセットを適用。最後に、`cineskin export --format prores --codec 422HQ --output ./output/final.mov`で、ProRes 422HQでエクスポート。`--bitrate 50M`でビットレートも指定可能。

## ベストプラクティス

### 1. レイヤー管理は「ベース→エフェクト→マスク」の3層構造を徹底
ポストプロダクションの現場では、後から修正が効く非破壊ワークフローが必須。ベースレイヤーでカラーグレーディング、エフェクトレイヤーで質感やアーティファクト、マスクレイヤーで部分制御と、役割を明確に分けることで、チームでの共同作業やクライアントからの修正依頼に即応できる。特に、エフェクトレイヤーは必ずベースレイヤーの上に配置し、マスクレイヤーは最上位に置くこと。

### 2. プリセットは「シーン単位」で保存し、トランジションを活用
映画制作では、シーンごとに統一感のあるルックが必要。cineskinのプリセットをシーン単位で保存し、シーンチェンジの際にクロスフェードを設定することで、自然なルックの遷移を実現。例えば、昼のシーンから夜のシーンに切り替わる場合、プリセットを2秒でクロスフェードすることで、観客に違和感を与えない。さらに、プリセット名に「scene_01_day」「scene_02_night」と命名規則を設けると、後からの検索性が向上する。
