## What is cineskin?

**cineskin** is a professional, pipeline-ready tool that streamlines cinematic skin design and video effects compositing for post-production workflows. It bridges the gap between creative intent and technical execution, enabling editors and VFX artists to apply consistent, high-fidelity "skin" layers—color grades, texture overlays, and stylized LUTs—directly onto footage without breaking the render chain.

## Core Features

1. **Non-Destructive Skin Layering Engine**  
   Apply and stack multiple cinematic skin profiles (e.g., "Film Noir," "Vintage Kodachrome," "Cyberpunk Bleach Bypass") as real-time overlay layers. Each skin preserves original pixel data, allowing per-channel adjustments without re-rendering.

2. **AI-Powered Shot Matching & Auto-Skin Transfer**  
   Using a lightweight neural network, cineskin analyzes scene luminance, contrast, and dominant hues from a reference clip, then automatically generates a matching skin profile for subsequent shots. This eliminates manual LUT tweaking across cuts.

3. **Pipeline-Native Export with Metadata Embedding**  
   Export final skin configurations as `.csk` files that embed color science metadata, version tags, and GPU-accelerated preview thumbnails. These files integrate directly with DaVinci Resolve, Premiere Pro, and Nuke via dedicated plugins.

## How to Use

### Step 1: Import & Analyze Source Footage  
Load your clip into the **cineskin Workbench**. Run the built-in **Scene Analyzer** to extract histogram data, dominant color vectors, and exposure zones. The tool will display a "Skin Compatibility Score" for each preset profile, ranked by statistical proximity to your footage.

### Step 2: Apply & Tune a Cinematic Skin  
Select a skin profile from the library (e.g., "Teal & Orange Hard Matte"). Use the **Per-Channel Weight Sliders** to fine-tune luminance, saturation, and gamma independently. Enable **Live Split-View** to compare the original vs. processed output at 4K resolution in real time.

### Step 3: Export & Integrate into Your Pipeline  
Click **Export Skin Config**. Choose between `.csk` (for cineskin-native pipeline) or `.cube` (standard LUT format). If using a supported NLE, enable **Auto-Plugin Sync** to push the skin directly into your active timeline without manual file linking.

## Best Practices

- **Always Run Scene Analyzer Before Applying Presets**  
  Blindly applying a skin profile can introduce color clipping or unnatural tonal shifts. Use the Scene Analyzer to identify the top 3 profiles that match your footage’s dynamic range—this reduces tweak time by 40% in our internal benchmarks.

- **Version Control Your Skin Configs**  
  Store each `.csk` file with a descriptive name including the shot number, skin version, and date (e.g., `shot034_v2_tech3_20250401.csk`). This is critical for collaborative workflows where multiple artists iterate on the same sequence.
