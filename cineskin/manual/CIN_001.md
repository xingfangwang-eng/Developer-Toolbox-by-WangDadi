## What is cineskin?

cineskin is a lightweight, GPU-accelerated toolchain for crafting cinematic skin textures and real-time video effects in post-production pipelines. It bridges the gap between VFX compositing and procedural texture generation, enabling artists to apply photorealistic skin overlays, aging effects, or stylized character looks directly onto video footage — without leaving the timeline.

## Core Features

### 1. **Real-time Skin Texture Inference Engine**
cineskin uses a custom ONNX runtime model to infer surface detail (pores, wrinkles, subsurface scattering) from single frames or video sequences. It applies these textures in real time, with per-pixel blending and motion-compensated tracking, eliminating the need for manual rotoscoping or frame-by-frame compositing.

### 2. **Procedural Skin Aging & Stylization Graph**
A node-based graph editor lets you stack procedural modifiers — wrinkle maps, blemish generators, color grading LUTs, and displacement layers — all controllable via keyframes or MIDI input. Each node outputs a 16-bit EXR sequence, preserving dynamic range for final grading.

### 3. **Automatic Skin Segmentation with Temporal Coherence**
Built on a fine-tuned YOLOv8 segmentation model, cineskin automatically isolates skin regions across frames with temporal smoothing. It handles occlusions, motion blur, and partial face coverage, outputting clean alpha masks that integrate seamlessly with Nuke, Fusion, or After Effects.

## How to Use

### Step 1: Load Your Footage and Configure the Inference Pipeline
Open cineskin and drag your video file (or image sequence) into the timeline. In the `Inference` panel, select the **Skin Texture Engine**. Choose your source model (e.g., `v1.2-cinematic` for high-fidelity skin, or `v1.0-stylized` for cartoon/abstract looks). Set the `Inference Resolution` to match your output — 1080p for broadcast, 4K for cinema. Enable `Temporal Coherence` to avoid flickering across cuts.

### Step 2: Build a Procedural Skin Graph
Switch to the `Graph` tab. Drag a `Wrinkle Map` node onto the canvas, then connect it to a `Color Grade` node. Add a `Subsurface Scattering` node for realism. Use the `Mask` input to restrict effects to skin regions (cineskin auto-generates this mask from Step 1). Tweak parameters: `Wrinkle Intensity`, `Blemish Density`, `SSS Radius`. Preview in real time with the built-in OpenGL viewer.

### Step 3: Export with Metadata and Alpha Channels
Go to the `Export` panel. Set output format to `OpenEXR (16-bit float)` for maximum quality, or `ProRes 4444` for quick integration. Enable `Embed Skin Mask` to export a separate alpha channel. Optionally, check `Export Per-Node EXR` to get individual layers for each procedural node — useful for advanced compositing in Nuke or Fusion. Click `Render Sequence`.

## Best Practices

### 1. **Always Match Inference Resolution to Delivery Resolution**
Running inference at a lower resolution than your final output introduces artifacts. For a 4K deliverable, set inference to 4K or use the `Super Resolution` pass in the graph to upscale after texture application. This preserves skin detail without aliasing.

### 2. **Use Temporal Coherence for Long Takes, Disable for Cuts**
cineskin’s temporal smoothing works best on continuous shots. For fast cuts or scene transitions, disable `Temporal Coherence` per clip to avoid ghosting. Alternatively, set a keyframe reset at each cut point.
