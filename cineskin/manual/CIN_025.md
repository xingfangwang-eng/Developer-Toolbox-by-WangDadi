# cineskin - CIN_025

## Overview

cineskin is a next-gen **AI-driven film-grade skin retouching & color grading engine** for post-production pipelines. It automates actor skin texture enhancement, dynamic color matching, and shot-by-shot consistency without manual LUT stacking. Built for editors, colorists, and VFX artists who hate wasting time on tedious frame-by-frame corrections.

## Core Features

### 1. Temporal Skin Texture Reconstruction (TSTR)
cineskin’s proprietary **frame-interpolation-aware denoiser** preserves micro-details (pores, sweat, stubble) while removing compression artifacts across entire sequences. Unlike static filters, TSTR analyzes motion vectors to prevent “plastic face” artifacts during camera movement.

### 2. AI-Driven Scene-Linked Color Harmonization
Automatically extracts dominant palettes from each scene’s background and applies **skin-tone-safe hue shifts** to match ambient lighting (e.g., tungsten vs. fluorescent). Supports ACES, Rec.709, and Log C input profiles with zero clipping in highlights or shadows.

### 3. Shot-to-Shot Metadata Bridge
Exports per-frame skin segmentation masks as OpenEXR/ACES containers—directly ingestible by Nuke, DaVinci Resolve, or After Effects. No more manual rotoscoping for skin isolation in complex VFX composites.

## How to Use

### Step 1: Ingest & Profile Your Footage
Drag a folder of source clips (ProRes, DNxHR, or RED RAW) into cineskin’s batch queue. Select your target output color space (e.g., Rec.709 for broadcast, DCI-P3 for cinema). The engine auto-detects skin regions per frame via a pre-trained **YOLOv8-seg model** calibrated on 50K+ IMDb-grade frames.

### Step 2: Apply Adaptive Skin Grading
In the **“Skin Zone” panel**, adjust three sliders: *Porosity* (fine detail retention), *Warmth Offset* (Kelvin shift for skin only), and *Shadow Lift* (underexposed face recovery). Each slider updates a real-time 4K preview with GPU-accelerated inference (CUDA/MPS). Toggle “Frame-Lock” to propagate changes across all shots with matching lighting conditions.

### Step 3: Export with Metadata
Click “Export Sequence” to generate a flat video file (H.264/H.265/ProRes) **plus** a sidecar `.cineskin` archive containing per-frame grading parameters. For compositing workflows, check “Export Skin Masks” to output 16-bit EXR sequences with alpha channels—ready for keying or relighting in Fusion or Flame.

## Best Practices

### Practice 1: Pre-Grade Raw Footage in Log Before cineskin
For maximum dynamic range, apply a basic exposure/contrast correction in Resolve or Premiere *before* importing into cineskin. This prevents the skin engine from amplifying noise in underexposed shadows. Always set input gamma to “Log C” or “S-Log3” for accurate skin tone mapping.

### Practice 2: Use “Shot-Reference” Mode for Multi-Cam Edits
When cutting between cameras (e.g., A-cam ARRI Alexa vs. B-cam Sony VENICE), select a single reference frame with balanced skin tones in cineskin’s **Reference Library**. Apply “Match to Reference” to all clips—the AI adjusts per-camera color matrices to maintain consistent actor complexion across cuts.
