# cineskin - CIN_015

## Resumen

**cineskin** es un motor de postproducción cinematográfica impulsado por IA que transforma cualquier metraje en bruto en un acabado visual con paletas de color, texturas y estética de blockbuster. Este proyecto está diseñado para editores de video, coloristas y desarrolladores de SaaS que buscan integrar efectos de cine en tiempo real sin depender de suites complejas como DaVinci Resolve o After Effects. Con un enfoque en la optimización SEO para el mercado hispanohablante, cineskin se posiciona como la herramienta definitiva para el diseño de pieles cinematográficas y efectos de video avanzados.

## Características Principales

- **Motor de Mapeo de Texturas Cinematográficas (CTM):** Aplica automáticamente más de 50 perfiles de color y texturas de grano de película (35mm, 16mm, Super 8) a clips de video en tiempo real, utilizando redes neuronales convolucionales para preservar detalles de alto rango dinámico (HDR) sin pérdida de calidad.
- **Pipeline de Postproducción Basado en LUTs Dinámicos:** Genera y aplica LUTs (Look-Up Tables) adaptativos que se ajustan dinámicamente a la iluminación y el contraste de cada escena, eliminando la necesidad de corrección manual de color. Compatible con formatos RAW, ProRes y H.264.
- **API RESTful con Soporte de Batch Processing:** Integra cineskin en flujos de trabajo existentes mediante una API que permite procesar lotes de hasta 100 clips simultáneamente, con parámetros ajustables para exposición, saturación, viñeteado y aberración cromática.

## Cómo Usar

1. **Configuración del Pipeline de Color con LUTs Dinámicos:**  
   - Clona el repositorio y ejecuta `pip install -r requirements.txt` para instalar dependencias (OpenCV, NumPy, TensorFlow).  
   - En el archivo `config.yaml`, define la ruta de entrada (`input_dir`) y salida (`output_dir`).  
   - Ejecuta `python cineskin.py --profile neo_noir --lut custom_lut.cube` para aplicar un perfil de cine negro con un LUT personalizado. El motor analizará automáticamente el histograma de cada frame y ajustará el contraste local.

2. **Procesamiento por Lotes con la API REST:**  
   - Inicia el servidor con `uvicorn api:app --reload --port 8000`.  
   - Envía una solicitud POST a `/process` con un JSON que incluya `video_urls` (lista de URLs) y `profile` (ej. "vintage_70s").  
   - Recibirás un objeto JSON con las URLs de los videos procesados, cada uno con metadatos de tiempo de procesamiento y perfil aplicado. Ejemplo:  
     ```json
     {
       "status": "success",
       "processed_videos": [
         {"id": "clip_01", "url": "https://cdn.cineskin.io/neo_noir_01.mp4", "processing_time": "2.3s"}
       ]
     }
     ```

3. **Integración con Plugins de Edición de Video:**  
   - Descarga el plugin para Premiere Pro desde el directorio `/plugins/premiere`.  
   - Instálalo en la carpeta de extensiones de Adobe y reinicia el software.  
   - Selecciona un clip en la línea de tiempo, ve a "Ventana > Extensiones > cineskin", elige un perfil (ej. "Kodak Vision3 500T") y haz clic en "Aplicar". El plugin sincroniza los cambios con el motor local mediante WebSockets.

## Mejores Prácticas

- **Optimización de Recursos para Procesamiento por Lotes:** Para evitar cuellos de botella en la GPU, limita el número de hilos de procesamiento a la mitad de los núcleos disponibles. Usa `--max_threads 4` en el CLI y monitorea el uso de VRAM con `nvidia-smi`. Si trabajas con videos 4K, considera reducir la resolución de previsualización a 1080p durante el desarrollo.
- **Versionado de Perfiles y LUTs:** Mantén un repositorio Git con todos los perfiles de color y LUTs personalizados. Usa etiquetas semánticas (ej. `v1.2.3-neo_noir`) para rastrear cambios. En la API, incluye un campo `version` en las solicitudes para garantizar consistencia entre diferentes builds del proyecto.
