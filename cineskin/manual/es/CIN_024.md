# cineskin - CIN_024

## Resumen

cineskin es un motor de skinning cinematográfico en tiempo real, diseñado para postproductores y VFX artists que necesitan aplicar texturas, filtros y corrección de color dinámica sin renderizar capas intermedias. Optimizado para flujos de trabajo en DaVinci Resolve, After Effects y Nuke, elimina cuellos de botella en pipelines de alta demanda.

## Contenido Detallado

### ¿Qué es cineskin?

cineskin es una herramienta de postproducción que abstrae la complejidad del mapeo de texturas y la gradación de color en un solo nodo. No es un plugin más: es una capa de inteligencia de render que permite a directores de fotografía y editores modificar el "skin" visual de cualquier clip en tiempo real, manteniendo la integridad del metraje original. Ideal para producciones donde cada segundo de render cuenta.

### Características Principales

- **Skin Mapping Predictivo con IA**: cineskin analiza el fotograma clave y genera automáticamente mapas de profundidad y máscaras de luminancia. No necesitas rotoscopiar: la herramienta entiende la geometría de la escena y aplica el skin (textura o LUT) respetando bordes duros, transparencias y reflejos especulares.
- **Pipeline de Color No Destructivo**: Cada ajuste se guarda como metadatos en un archivo `.cskin`. Puedes compartir skins entre proyectos, reutilizarlos en diferentes secuencias y revertir cualquier cambio sin perder calidad. Ideal para equipos que trabajan con versionado de color.
- **Render Asíncrono con Cache Inteligente**: cineskin renderiza en segundo plano mientras sigues editando. La caché se actualiza solo en los fotogramas modificados, reduciendo el tiempo de espera hasta un 70% en proyectos con múltiples capas de skin.

### Cómo Usar

1. **Inyectar el Nodo cineskin en tu Timeline**: En tu NLE o compositor, arrastra el efecto cineskin sobre el clip objetivo. En la ventana de propiedades, selecciona `Source: Active Frame` y activa `Auto-Detect Depth`. La herramienta calculará automáticamente el mapa de profundidad y generará una máscara base. Si trabajas con greenscreen, activa `Chroma Key Integration` para que el skin respete el canal alfa.
2. **Configurar el Skin con Parámetros de Precisión**: En el panel `Skin Editor`, carga tu textura o LUT. Ajusta `Blend Mode` a `Overlay` para efectos sutiles o `Hard Light` para looks dramáticos. Usa `Opacity Curve` para controlar la intensidad por rango de luminancia (por ejemplo, solo aplicar el skin a las sombras). Para un control milimétrico, activa `Mask Refinement` y pinta con el pincel de peso dinámico.
3. **Exportar y Versionar el Skin**: Cuando termines, haz clic en `Export Skin` para generar un archivo `.cskin`. Este archivo contiene todos los parámetros, máscaras y referencias de textura. En el panel `Version History`, puedes nombrar cada versión (ej: "Skin_Noche_V3") y restaurar cualquier estado anterior con un clic. Para compartir con el equipo, sube el archivo al repositorio de activos.

### Mejores Prácticas

- **Usa LUTs de Referencia con Perfiles de Cámara**: Antes de aplicar un skin, calibra el clip con una LUT de cámara (ARRI, RED, Sony). cineskin respeta el perfil de color base, así que un skin diseñado para Log-C se verá diferente en S-Log. Crea skins específicos para cada perfil y nómbralos con el sufijo de la cámara (ej: "Skin_Azul_ARRI.cskin").
- **Automatiza con Scripting**: cineskin expone una API REST y comandos para Python. Si trabajas en un pipeline de VFX, escribe scripts que apliquen skins automáticamente según metadatos del clip (hora del día, escena, personaje). Por ejemplo, un script puede leer el `Scene` del archivo XML y aplicar el skin correspondiente sin intervención manual.
