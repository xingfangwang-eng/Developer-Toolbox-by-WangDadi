# cineskin - CIN_003

## Resumen

cineskin es una herramienta de postproducción cinematográfica que automatiza la aplicación de skins de película (LUTs, texturas, y efectos de grano) directamente sobre secuencias de video, eliminando la necesidad de software de edición pesado. Diseñada para flujos de trabajo ágiles, permite a editores y coloristas aplicar estilos visuales complejos en tiempo real, optimizando el rendimiento en proyectos de alta resolución (4K/8K). Con un enfoque en la coherencia de color y la reducción de artefactos, cineskin integra algoritmos de aprendizaje automático para adaptar los skins a diferentes condiciones de iluminación y metraje.

## Características Principales

- **Motor de Adaptación Contextual de LUTs**: No se limita a aplicar LUTs estáticos. cineskin analiza el histograma y la distribución de color de cada fotograma, ajustando dinámicamente la intensidad y la saturación del skin para evitar clipping y preservar detalles en sombras y altas luces. Esto es crítico para metraje de acción con cambios bruscos de iluminación.
- **Biblioteca de Skins con Mapeo de Textura 3D**: Cada skin incluye no solo una LUT, sino también mapas de textura (grano, aberración cromática, halación) que se aplican en un espacio de color 3D. Esto permite replicar con precisión la apariencia de películas analógicas (Kodak, Fuji) o estilos digitales (teal/orange, bleach bypass) sin artefactos de banding.
- **Optimización de Rendimiento con Caching Predictivo**: Utiliza un sistema de caché inteligente que precalcula los fotogramas más probables a ser renderizados basándose en la línea de tiempo del proyecto. Reduce el tiempo de exportación hasta un 40% en secuencias largas, evitando la recarga innecesaria de skins y texturas.

## Cómo Usar

1. **Integración del Pipeline de Color**: Importa tu secuencia de video (soportados: ProRes, DNxHD, H.264/5) mediante la CLI `cineskin import --input /ruta/al/video.mp4 --profile cinema`. Esto genera un archivo de metadatos (`.csk`) que mapea cada fotograma con su histograma original. Luego, ejecuta `cineskin apply --skin "Kodak 2383" --lut-intensity 0.85 --grain 0.3` para aplicar el skin. El motor ajustará automáticamente la intensidad del grano según la sensibilidad ISO detectada en los metadatos EXIF del archivo.
2. **Personalización de Skins con Ajustes Precisos**: Para modificar un skin existente, usa el modo interactivo: `cineskin edit --skin "Teal Orange" --curve shadows:-15,midtones:0,highlights:+10`. Esto permite modificar la curva de color sin alterar el mapeo de textura. Para guardar el skin personalizado como un nuevo perfil: `cineskin export --profile "Mi Estilo" --format .cskin`. Este archivo puede compartirse entre proyectos.
3. **Renderizado por Lotes con Validación de Calidad**: Para exportar múltiples versiones de un mismo metraje (por ejemplo, para pruebas A/B de estilos), usa `cineskin batch --input /ruta/a/carpeta --skins "Kodak 2383,Fuji 3510,Teal Orange" --output-dir /ruta/salida`. Cada skin generará una carpeta separada. Para verificar la consistencia del color, ejecuta `cineskin validate --input /ruta/salida --reference /ruta/original.mp4`, que genera un informe PDF con métricas de diferencia de color (ΔE, PSNR) y advertencias de clipping.

## Mejores Prácticas

- **Calibración Previa del Metraje**: Antes de aplicar cualquier skin, realiza una corrección de color básica (balance de blancos y exposición) en tu NLE o con `cineskin preprocess --auto-white`. Los skins de cineskin están diseñados para actuar sobre metraje neutro; aplicarlos sobre material sin corregir puede amplificar dominantes de color no deseadas. Para proyectos multicámara, unifica la temperatura de color de todas las fuentes antes de la aplicación.
- **Uso de Skins como Referencia, No como Solución Final**: Aunque cineskin ofrece resultados impresionantes, no reemplaza una corrección de color manual para proyectos de alto presupuesto. Utiliza los skins como punto de partida para la gradación final. Exporta el metraje con el skin aplicado como un archivo de referencia (por ejemplo, `proxies` en 1080p) y luego aplica el mismo skin como LUT en tu NLE (DaVinci Resolve, Premiere) para ajustes finos. Esto garantiza coherencia en la edición sin sacrificar el control creativo.
