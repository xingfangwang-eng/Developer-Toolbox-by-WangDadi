### What is zerosub?

zerosub is a professional tool that helps solve problems in related fields. It is designed to simplify the process of creating and managing subtitles for video files. The software is easy to use and provides comprehensive features for users who want to ensure that their videos have clear and accurate captions.

### Core Features

- **Subtitle Creation**: zerosub allows you to add text or speech to your video, providing an option to enhance the viewer's experience by adding narration to educational content.
  
- **Customizable Options**: It includes customizable options like color scheme, font size, duration for each subtitle track, and more.

- **Subtitles Management System**: The software is designed with a comprehensive subtitles management system which allows users to easily update or delete subtitles without affecting other elements of the video. This feature makes managing subtitles fast and efficient.

### How to Use

1. Step 1: Import your video file using the provided import button.
2. Step 2: Select subtitles for different segments within your video (such as captions, comments, etc.) by clicking on the appropriate track in the timeline.
3. Step 3: Customize the subtitle text and options as desired, then apply changes to the selected tracks.
