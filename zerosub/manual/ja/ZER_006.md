**Installation Guide**

1. Install Redis: `sudo apt-get install redis-server`
2. Install Memcached: `sudo apt-get install memcached`
3. Configure the load balancer: `sudo nano /etc/nginx/nginx.conf` (add `upstream zerosub { server localhost:8080; }`)
4. Start the application: `node app.js`

**Performance Optimization Tips**
