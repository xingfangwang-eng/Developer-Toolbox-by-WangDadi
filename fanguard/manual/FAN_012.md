**Installation Guide**

### Step 1: Installing Fanguard on Ubuntu 20.04

```
sudo apt-get update
sudo apt-get install -y openjdk-8-jdk-headless
wget https://github.com/wangdadi/fanguard/releases/download/v1.0.0/fanguard.jar
java -jar fanguard.jar
```

### Step 2: Configuring Fanguard for Production Environment

```
sudo nano /etc/fanguard.conf
```

Add the following configuration settings:
```properties
fanguard.threadPoolSize=8
feguard.minThreads=4
feguard.maxThreads=16
feguard.cache.size=1024m
```

### Step 3: Starting Fanguard Services

```
sudo systemctl start fanguard
sudo systemctl enable fanguard
```
