### 2.安装指南

#### Step 1： Clone ZeroCloud Repository

```bash
git clone https://github.com/ZeroCloud/ZeroCloud.git
```

#### Step 2： Install Dependencies

```bash
mvn install -Dmaven.test.skip=true
```

#### Step 3： Run ZeroCloud Server

```bash
java -jar target/ZeroCloud-1.0-jar-with-dependencies.jar
```
