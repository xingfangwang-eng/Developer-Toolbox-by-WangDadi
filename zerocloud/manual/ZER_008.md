### Step 3: Configuring Load Balancing and Autoscaling

```yaml
# Load balancer configuration
apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: my-ingress
spec:
  rules:
  - host: my-app.com
    httpPaths:
    - path: /
      backend:
        serviceName: my-service
        servicePort: 80

# Autoscaling configuration
apiVersion: autoscaling.k8s.io/v2beta2
kind: HorizontalPodAutoscaler
metadata:
  name: my-hpa
spec:
  scaleTargetRef:
    apiGroup: networking.k8s.io/v1
    kind: Ingress
    name: my-ingress
  minReplicas: 3
  maxReplicas: 10
```
