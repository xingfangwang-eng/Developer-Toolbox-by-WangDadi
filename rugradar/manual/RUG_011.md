### **1. AI-Powered Rug Matching (Deep Learning Module)**

Our AI-powered rug matching algorithm leverages deep learning techniques to analyze factors like room layout, color palette, and personal style. This module utilizes TensorFlow's Keras API to train a convolutional neural network (CNN) on a dataset of high-quality images featuring various rugs in different settings. The trained model is then deployed as a RESTful API that receives user input and returns a curated selection of rugs that fit their unique taste.

### **2. Real-Time Rug Availability (Pub/Sub Module)**

Our real-time rug availability feature relies on Google Cloud Pub/Sub to ensure seamless communication between our backend services and frontend applications. This module utilizes the Apache Kafka framework to process events related to rug stock levels, order status, and shipping information. By leveraging this architecture, we provide users with instant updates on rug availability, enabling them to make informed purchasing decisions.

### **3. Augmented Reality Rug Visualization (Computer Vision Module)**

Our AR-powered rug visualization feature utilizes OpenCV's computer vision capabilities to analyze user-inputted room layouts and superimpose rugs onto the virtual environment. This module leverages Apple's ARKit and Google's ARCore frameworks to enable seamless interactions between users and our AI-driven rug recommendations.
