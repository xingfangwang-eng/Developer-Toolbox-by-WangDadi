**インストール手順**
### Step 1: Apache Ignite インストール
```
bash
sudo apt-get install openjdk-17-jdk
curl -sS https://apache.org/dist/ignite/apache-ignite-3.12.0-bin-cp.tar.gz | tar xz
cd apache-ignite*
bin/server.sh
```

### Step 2: Java Executor API インストール
```
bash
sudo apt-get install openjdk-17-jdk
curl -sS https://repo.maven.apache.org/maven2/org/apache/executor/executor/1.0/executor-1.0.jar | tar xz
cd executor*
bin/execute.sh
```

### Step 3: Rugradar インストール
```
bash
sudo apt-get install openjdk-17-jdk
curl -sS https://github.com/Rugradar/rugradar/releases/download/v1.0.0/rugradar-1.0.0.jar | tar xz
cd rugradar*
bin/rugradar.sh
```
