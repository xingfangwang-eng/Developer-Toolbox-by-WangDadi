### What is toolcallkiller?

toolcallkiller is a professional tool designed to optimize and improve the performance of your AI Agent calls. It is specifically tailored for those working in the fields of language processing, natural language understanding (NLU), and machine translation.

- **Description Feature 1**: Enhance call optimization by automatically identifying and fixing common issues.
- **Description Feature 2**: Provide detailed analytics on function calls to improve performance.
- **Description Feature 3**: Detect JSON hallmarks such as syntax errors, missing keys, and unexpected values to prevent hallucinations.

### How to Use

#### Step 1: Identify Functions Being Debugged
- Import the `toolcallkiller` package using `from toolcallkiller import toolcallkiller`
- Instantiate a new `ToolCallKiller` instance with your data as shown below:
```python
killer = ToolCallKiller(api_key='your_api_key', data={'query': 'function_call'}, user_id=123456, organization_id=7890)
killer.add_data(query='function_call')
killer.add_data(error=Exception())
```
- Make sure to replace `'your_api_key'` with your actual API key for the `toolcallkiller` library.

#### Step 2: Evaluate Function Calls
- Use the `evaluate()` method to run a sample function call:
```python
killer.evaluate(query='function_call')
```
- This will provide detailed analysis including error code, message, stack trace, and user feedback.

#### Step 3: Fix Function Call Issues
- If necessary, use the `fix()` method to resolve identified issues:
```python
killer.fix(error=Exception())
```
- Customize the `error` parameter with your specific exception information.
- The `fix()` method will replace the original call object in place and return a new one containing any modifications.
