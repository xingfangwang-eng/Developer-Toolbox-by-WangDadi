# bbb26dramapredictor - BBB_010

## Overview

`bbb26dramapredictor` is a high-frequency sentiment & narrative drift analyzer built specifically for the Brazilian reality show Big Brother Brasil 26. It ingests real-time social signals (Twitter/X, Reddit, GShow polls) and applies a proprietary **Drama Entropy Index (DEI)** to forecast audience outrage spikes, alliance fractures, and Paredão elimination probabilities before they trend.

This tool is not a generic sentiment wrapper—it’s a **causal graph model** tuned to BBB26’s unique house dynamics, production editing patterns, and Brazilian meme culture.

## Core Features

### 1. Drama Entropy Index (DEI) with Causal Inference
Unlike simple sentiment polarity, DEI models the **conditional probability of narrative collapse** across 12 latent dimensions (e.g., “couple friction,” “production interference,” “fanbase raid”). The index is updated every 2 minutes via a sliding window of 5,000+ posts, weighted by user authority and virality velocity. Output is a float between 0 (boring consensus) and 1 (full meltdown).

### 2. Multi-Lingual Social Graph Overlap Detector
BBB26 generates content in Portuguese, Spanish, and English. This feature builds a cross-lingual embedding space (using fine-tuned Sentence-BERT + custom stopwords for Brazilian slang) and detects **semantic clusters** that emerge simultaneously across languages—e.g., a “falso” accusation in PT-BR that mirrors a “fake” narrative in EN. This is critical for spotting coordinated fan campaigns or production-driven edits.

### 3. Paredão Outcome Simulator (Monte Carlo + House Dynamics)
Before the official vote closes, this module runs 10,000 simulations using historical BBB voting patterns, current poll data from UOL and GShow, and a Markov chain model of house alliances. It outputs a **probability distribution** for each nominee, plus a “surprise factor” score (how much the result diverges from the current poll leader). The model recalibrates every 30 minutes.

## How to Use

### 1. Bootstrap the Data Pipeline with Real-Time Twitter Stream
```bash
python3 bbb26_stream.py --source twitter --keywords "BBB26,Paredão,eliminação" --lang pt --output ./data/raw/
```
This starts a filtered stream that writes JSON lines to disk. Use `--include-engagement` to capture retweet/like counts for DEI weighting. The script auto-rotates files every 10 minutes to prevent memory leaks.

### 2. Run the Drama Entropy Index on a Time Window
```python
from bbb26dramapredictor import DEI
dei = DEI(window_minutes=30, causal_depth=2)
score, breakdown = dei.analyze("./data/raw/2025-03-15_14-00.json")
print(f"DEI: {score:.3f} | Top driver: {breakdown['top_driver']}")
```
The `causal_depth` parameter controls how many chained events (e.g., “fight → tweet → poll shift”) the model considers. For production, set `depth=3` and run on a cron job every 5 minutes.

### 3. Generate a Paredão Simulation Report
```bash
python3 bbb26_simulate.py --nominees "João,Ana,Carlos" --polls ./data/polls/current.json --sims 10000 --output ./reports/paired_sim.html
```
This creates an interactive HTML report with a violin plot of vote distributions, a confusion matrix of alliance shifts, and a “dark horse” label for any nominee with >15% chance of upset. The simulation logs all random seeds for reproducibility.

## Industry Best Practices

### 1. Always Calibrate DEI Against Production Edits
BBB’s production team often inserts “flashback” segments or edits to steer narratives. Before trusting DEI spikes, cross-reference with the show’s live timeline (available via GloboPlay’s API). A high DEI that coincides with a flashback is likely **manufactured drama**—filter it out if you’re modeling organic sentiment.

### 2. Use Differential Privacy When Sharing Simulation Outputs
If you publish Paredão probabilities before the official vote, you risk influencing the audience (and violating Globo’s terms). Apply a Laplace noise mechanism to your output distributions (epsilon = 1.0) to prevent exact replication. This keeps your tool ethical and legally safe while still providing actionable insights.
