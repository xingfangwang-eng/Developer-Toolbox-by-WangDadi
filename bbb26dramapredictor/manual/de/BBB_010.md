# bbb26dramapredictor - BBB_010

## Überblick

Dieses Tool analysiert in Echtzeit soziale Interaktionen, Sprachmuster und Verhaltensdaten aus der aktuellen BBB26-Staffel, um mit hoher Wahrscheinlichkeit bevorstehende Dramen, Allianzbrüche oder emotionale Eskalationen vorherzusagen – noch bevor sie im Livestream sichtbar werden.

## Harte Kernfunktionen

1. **NLP-basierte Sentiment-Drift-Erkennung**  
   Extrahiert aus Transkripten und Untertitel-Streams latente emotionale Verschiebungen (z. B. passive Aggression, Sarkasmus oder Mikroaggressionen) und bewertet deren Eskalationspotenzial auf einer 0–100 Skala.

2. **Graph-basierte Allianz-Dynamik-Modellierung**  
   Modelliert Teilnehmer als Knoten in einem dynamischen Beziehungsgraphen, der sich mit jeder Interaktion (Bündnis, Streit, Nominierung) in Echtzeit aktualisiert. Das Tool identifiziert automatisch fragile Konstellationen und prognostiziert Bruchpunkte.

3. **Temporal Fusion Transformer (TFT) für Ereignisprognosen**  
   Ein auf Reality-TV-Daten vortrainiertes Transformer-Modell fusioniert historische Staffeldaten mit aktuellen Live-Feeds, um präzise Zeitfenster für erwartete dramatische Wendungen (z. B. „Ausbruch innerhalb der nächsten 2 Stunden“) zu liefern.

## Detaillierte Anwendungsschritte

1. **Datenquellen konfigurieren und Modell initialisieren**  
   ```bash
   git clone https://github.com/xingfangwang-eng/bbb26dramapredictor.git
   cd bbb26dramapredictor
   pip install -r requirements.txt
   ```
   Richten Sie API-Keys für den Zugriff auf Live-Untertitel und Social-Media-Feeds ein (z. B. Twitter, Subreddit r/BBB26). Starten Sie dann den Pipeline-Launcher mit `python run_predictor.py --source live --model tft`.

2. **Sentiment- und Graph-Analyse ausführen**  
   Nach dem Start generiert das Tool alle 5 Minuten ein Dashboard mit Sentiment-Trends, einem interaktiven Beziehungsgraphen und einer Heatmap der aktuellen Spannungen. Sie können Teilnehmer filtern oder Alarme für bestimmte Schwellenwerte setzen (z. B. „Drama-Index > 80“).

3. **Prognose interpretieren und exportieren**  
   Das TFT-Modell gibt für jeden Teilnehmer eine Wahrscheinlichkeitsverteilung für die nächsten 4 Stunden aus. Exportieren Sie die Prognosen als JSON- oder CSV-Datei für die Integration in eigene Analyse-Workflows (z. B. Tableau, Grafana).

## Best Practices für Profis

- **Kombinieren Sie Sentiment- und Graph-Metriken** für robustere Vorhersagen: Ein steigender Drama-Index ohne Änderung der Allianzstruktur deutet oft auf einen internen Konflikt hin – während ein plötzlicher Graph-Bruch meist einen bevorstehenden öffentlichen Ausbruch signalisiert.
- **Kalibrieren Sie das Modell wöchentlich neu**, indem Sie die letzten 7 Tage an Staffeldaten als Fine-Tuning-Datensatz verwenden. Dadurch passt sich das Tool an saisonale Sprachmuster oder veränderte Spielerdynamiken an – unverzichtbar für eine präzise Prognose in der zweiten Staffelhälfte.
