# bbb26dramapredictor - BBB_024

## 概要

本ツールは、ブラジルリアリティ番組「BBB26」のドラマ展開をリアルタイムで予測・分析するためのSaaS型エンジンです。視聴者データ、ソーシャルメディアのセンチメント、過去シーズンのパターンを機械学習モデルで統合し、次に起こる衝突・同盟・感情爆発を高精度で予測します。

## コア機能

### 1. ソーシャルセンチメント・リアルタイムエンジン
Twitter/X、Instagram、TikTokの投稿をストリーミング解析し、参加者ごとの感情スコア（怒り・喜び・嫉妬・共感）を秒単位で可視化。ネガティブバーストが検出されると、自動で「ドラマ警報」を発報します。

### 2. 同盟崩壊予測モデル（Alliance Fracture Index）
過去の会話ログ、投票履歴、ボディランゲージ解析（カメラフィードから抽出）を基に、参加者間の信頼度スコアを算出。閾値を下回ったペアを「崩壊リスク高」としてリストアップし、具体的なトリガーイベント（例：指名投票での裏切り）を予測します。

### 3. ストーリーアーク・シミュレーター
現在のゲーム状態を入力すると、3日後・7日後・最終週の3つのタイムラインで、最も起こり得るドラマ展開を確率付きで生成。各シナリオに対して、視聴率への影響度も予測します。

## 使い方

### ステップ1: データソースの接続と初期設定
まず、BBB26の公式ライブフィード（Globoplay API）と、Twitter/XのストリーミングAPIキーをダッシュボードに登録します。その後、過去3シーズンのアノテーションデータ（CSV形式）をアップロードし、モデルの転移学習を実行。初期キャリブレーションには約15分かかります。

```bash
# 推奨: バッチインポート用CLI
bbb26dramapredictor import --source twitter --since "2025-01-10T20:00:00Z" --lang pt
```

### ステップ2: ダッシュボードでリアルタイム監視
メイン画面では、参加者カードが感情スコアに応じて色分け表示されます（赤＝怒り、青＝悲しみ、黄＝喜び）。各カードをクリックすると、その参加者の直近24時間の感情トレンドグラフ、および関連ツイートのサンプルが表示されます。右上の「ドラマアラート」パネルには、現在進行中の緊張関係がリストアップされます。

### ステップ3: シミュレーション実行とレポート生成
「シミュレート」タブで、予測したい日数（例：72時間後）と、仮定イベント（例：「もし明日の指名でAがBを指名したら？」）を設定します。エンジンが1000回のモンテカルロシミュレーションを実行し、結果をPDFレポートとしてエクスポート可能。レポートには、各シナリオの確率、主要な転換点、視聴率予測が含まれます。

## ベストプラクティス

### 1. ノイズ除去フィルターを常に有効にせよ
ソーシャルデータにはbotやスパムが大量に混入します。必ず「Bot Detection Middleware」をONにし、アカウント作成日が30日未満のユーザー、フォロワー数50未満のアカウントを除外してください。これにより、センチメントスコアの精度が約22%向上します。

### 2. モデルのリトレーニングは週1回が最適
BBB26は毎週新たな参加者が追加され、同盟関係が激変します。毎週日曜日の朝6時（ブラジル時間）に、直近7日間のデータでモデルをリトレーニングするcronジョブを設定してください。推奨バッチサイズは5000サンプル、エポック数は3です。
