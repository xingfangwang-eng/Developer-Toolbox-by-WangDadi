# bbb26dramapredictor - BBB_021

## Resumen

Herramienta profesional de análisis predictivo diseñada específicamente para anticipar dinámicas de conflicto, alianzas y giros argumentales en reality shows como BBB26. Utiliza procesamiento de lenguaje natural (NLP) y modelos de machine learning entrenados con datos históricos de temporadas anteriores para ofrecer predicciones con alta precisión.

## Contenido Detallado

### ¿Qué es bbb26dramapredictor?

bbb26dramapredictor es un SaaS especializado en el análisis de patrones de comportamiento y narrativas televisivas en tiempo real. A diferencia de herramientas genéricas de análisis de sentimiento, este sistema está calibrado para detectar micro-expresiones lingüísticas, cambios de tono en conversaciones grupales y patrones de voto que predicen con hasta un 87% de precisión los próximos conflictos o eliminaciones en el programa.

### Características Principales

- **Análisis de Grafos de Tensión Social**: Mapea en tiempo real las relaciones entre participantes usando algoritmos de detección de comunidades (Louvain) y centralidad de intermediación. Identifica nodos de conflicto latente antes de que estallen en pantalla.

- **Modelo de Predicción de Giros Argumentales (Plot-Twist Engine)**: Entrenado con 14 temporadas de BBB y datos de 6 reality shows internacionales, utiliza transformers (BERT adaptado) para detectar anomalías narrativas. Predice con 72 horas de antelación eventos como "paredão sorpresa" o cambios drásticos de alianza.

- **Dashboard de Probabilidades Bayesiano**: Visualiza en vivo las probabilidades de cada participante de ser eliminado, ganar el reality o protagonizar un conflicto mayor. Actualiza las variables cada 5 minutos con datos extraídos de redes sociales, interacciones en vivo y patrones históricos.

### Cómo Usar

1. **Configurar el Crawler de Datos en Vivo**: Clona el repositorio y ejecuta `python setup_crawler.py --source twitter,bbb_live,reddit`. El sistema iniciará la recolección de menciones, hashtags y transcripciones del programa en tiempo real. Configura las credenciales de API en el archivo `.env` usando las variables `TWITTER_BEARER_TOKEN` y `BBB_LIVE_API_KEY`.

2. **Entrenar el Modelo con Datos Históricos**: Ejecuta `./train_pipeline.sh --season 26 --epochs 50 --batch_size 32`. El script descargará automáticamente los datasets de temporadas anteriores desde el repositorio público de BBB Archive, limpiará los datos (eliminando stopwords en portugués brasileño) y entrenará el modelo de predicción. Verifica los logs en `/var/log/bbb26/training.log`.

3. **Desplegar el Dashboard de Predicciones**: Usa el comando `docker-compose up -d --scale api=3` para levantar el servicio web con balanceo de carga. Accede a `http://localhost:8080/dashboard` donde verás las probabilidades actualizadas. Para recibir alertas en Telegram, configura el webhook en la sección de ajustes: `https://api.telegram.org/bot<TOKEN>/setWebhook?url=http://tu-dominio/webhook`.

### Mejores Prácticas

- **Calibración Semanal con Validación Cruzada**: Cada lunes, ejecuta `python validate_model.py --test_set semana_actual` para ajustar los pesos del modelo con los datos de la semana anterior. Esto evita el overfitting y mantiene la precisión por encima del 80% incluso cuando cambian las dinámicas del programa (ej: entrada de nuevos participantes).

- **Integración con Webhooks de Streaming**: No dependas solo del polling. Configura webhooks para recibir actualizaciones en tiempo real desde las plataformas de streaming (Globoplay, YouTube) usando sus APIs de eventos. Esto reduce la latencia de predicción de 15 minutos a menos de 30 segundos, crucial para decisiones de apuestas o contenido en vivo.
