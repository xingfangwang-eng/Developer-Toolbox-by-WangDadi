# bbb26dramapredictor - BBB_018

## Resumen Técnico

Herramienta de predicción de conflictos basada en machine learning para el programa BBB26. Utiliza procesamiento de lenguaje natural (NLP) y análisis de patrones de interacción para anticipar giros dramáticos en tiempo real. Ideal para productores, analistas de audiencia y fans hardcore que quieren adelantarse a la narrativa.

## Funcionalidades Clave (Hardcore)

1. **Motor de Predicción de Conflicto Markoviano**  
   Modelo probabilístico entrenado con datos históricos de 25 temporadas de BBB. Analiza microexpresiones faciales, tono de voz y frecuencia de interrupciones en conversaciones para generar un *índice de tensión* (0-100) con 87% de precisión en pruebas ciegas.

2. **Mapeo de Alianzas Dinámicas en Tiempo Real**  
   Grafo de redes sociales actualizado cada 5 minutos basado en menciones cruzadas y proximidad física detectada por cámaras. Detecta *clusters* de poder y posibles traiciones con alertas push configurables.

3. **Predictor de Estrategias de Votación**  
   Algoritmo de simulación Monte Carlo que ejecuta 10,000 escenarios de eliminación cada hora. Integra variables como popularidad en redes sociales, rendimiento en pruebas semanales y patrones de voto anteriores para proyectar resultados con margen de error <3%.

## Cómo Usar (Guía Técnica)

1. **Configurar el Pipeline de Datos**  
   Conecta el feed RTMP del programa a través del endpoint `/api/v1/ingest/bbb26`. Usa WebSocket para recibir actualizaciones en vivo. Ejemplo de autenticación:  
   ```bash
   curl -X POST https://api.bbb26dramapredictor.com/stream \
     -H "Authorization: Bearer $TOKEN" \
     -d '{"source": "globo_play", "resolution": "720p"}'
   ```

2. **Ejecutar Análisis de Sentimiento por Escena**  
   Activa el módulo `scene_analyzer` con parámetros de granularidad:  
   ```python
   from dramapredictor import SceneAnalyzer
   analyzer = SceneAnalyzer(model='bert-bbb26-es', threshold=0.75)
   result = analyzer.process_clip('clip_2025_03_15_22h45m.mp4')
   print(result['tension_index'])  # Output: 82.4
   ```

3. **Generar Reporte de Predicción Semanal**  
   Programa un cron job que ejecute el script `predictor.py --mode weekly --output json`. El reporte incluye:  
   - Matriz de probabilidades de eliminación por participante  
   - Ranking de jugadores con mayor *potencial de confrontación*  
   - Recomendaciones de edición para maximizar rating
