# billripper - BIL_026

## Resumen

Herramienta de extracción y análisis de facturas (billripper) diseñada para automatizar la gestión de documentos financieros, con soporte multi-idioma y exportación estructurada. Ideal para contables, desarrolladores y equipos de fintech que necesitan procesar cientos de facturas en segundos.

## Contenido Detallado

### ¿Qué es billripper?

billripper es un **parser de facturas** basado en reglas semánticas y modelos de lenguaje localizados. A diferencia de herramientas genéricas, billripper entiende la estructura fiscal española (IVA, IRPF, facturas electrónicas) y extrae campos como CIF, base imponible, importe total y desglose de impuestos. No es un simple OCR: es un **sistema de interpretación contextual** que normaliza datos de PDFs, imágenes o texto plano.

### Características Principales

- **Extracción semántica con reglas fiscales**: billripper identifica automáticamente el tipo de factura (simplificada, completa, rectificativa), aplica las reglas del IVA español y valida el formato del CIF/NIF. Incluye detección de duplicados mediante hash de contenido.
- **Pipeline de normalización multi-fuente**: Soporta entrada desde archivos locales, URLs de descarga directa, APIs REST o correos electrónicos (IMAP). Convierte formatos dispares (PDF escaneado, XML Facturae, CSV bancario) a un esquema JSON unificado.
- **Motor de exportación con plantillas personalizables**: Genera salidas en JSON, CSV, Excel, o directamente a sistemas contables (ContaSOL, Sage 50). Permite mapear campos específicos del ERP del cliente mediante un archivo YAML de configuración.

### Cómo Usar

1. **Configurar el pipeline de entrada**  
   Crea un archivo `billripper_config.yaml` en el directorio raíz. Define las fuentes de datos:  
   ```yaml
   sources:
     - type: local
       path: ./facturas/
     - type: email
       server: imap.gmail.com
       user: tu_cuenta@gmail.com
       folder: INBOX/Facturas
   filters:
     - regex: '.*factura.*'
     - min_size: 10KB
   ```  
   Luego ejecuta: `billripper scan --config billripper_config.yaml`. El sistema indexará los archivos y mostrará un resumen de coincidencias.

2. **Ejecutar extracción con validación fiscal**  
   Usa el comando `extract` con flags para habilitar la lógica de IVA:  
   ```bash
   billripper extract --input ./resultados/index.json --output ./export/ --validate-vat --country ES
   ```  
   billripper aplicará reglas de validación: comprobará que el CIF tenga 9 caracteres (8 dígitos + letra), que el IVA sea del 21% (o reducido/superreducido según el producto), y que los totales cuadren. Los errores se registran en `./export/errors.log`.

3. **Generar informe contable y exportar**  
   Para integrar con tu ERP, usa la opción de exportación personalizada:  
   ```bash
   billripper export --input ./export/resultados.json --template ./plantillas/contasol.yaml --output ./contabilidad.csv
   ```  
   La plantilla YAML mapea campos:  
   ```yaml
   mapping:
     invoice_number: 'Número Factura'
     client_cif: 'CIF Cliente'
     base_imponible: 'Base Imponible'
     iva: 'IVA (21%)'
     total: 'Total'
   ```  
   El CSV generado se puede importar directamente en ContaSOL.

### Mejores Prácticas

- **Usa hashing para evitar duplicados en lotes grandes**  
  Cuando proceses cientos de facturas, activa `--dedup sha256` en el comando `scan`. billripper generará un hash del contenido y comparará con un archivo de historial (`~/.billripper/history.db`). Esto evita reimportar facturas ya procesadas, ahorrando tiempo y evitando errores contables.

- **Configura alertas para facturas con IVA incorrecto**  
  En el archivo `billripper_config.yaml`, añade la sección `alerts`:  
  ```yaml
  alerts:
    - rule: 'iva_mismatch'
      action: 'email'
      recipient: 'contabilidad@empresa.com'
  ```  
  Cuando billripper detecte una factura donde el IVA calculado no coincida con el declarado (por ejemplo, un 10% en lugar del 21% para servicios estándar), enviará un correo automático al departamento de contabilidad. Esto previene errores antes de la liquidación trimestral.
