# billripper - BIL_013

## Resumen

billripper es el parser de facturas más agresivo del mercado: extrae datos complejos de PDFs, XMLs y facturas escaneadas en segundos, sin depender de OCR de terceros ni plantillas manuales. Ideal para equipos de contabilidad, desarrolladores de ERPs y SaaS de gestión financiera que necesitan automatizar el flujo de facturación masivo.

## Contenido Detallado

### ¿Qué es billripper?

billripper es una herramienta de análisis y extracción de facturas diseñada para entornos de alta demanda. A diferencia de soluciones genéricas, billripper utiliza un motor de parsing híbrido (reglas + ML ligero) que se adapta a formatos de factura de España, México, Argentina y otros países de habla hispana. No necesitas configurar plantillas: billripper detecta automáticamente campos clave como NIF/CIF, importes, IVA, fechas y conceptos, incluso en facturas con diseños no estándar.

### Características Principales

- **Parsing multi-formato con fallback automático**: billripper procesa PDF nativos, XML (Facturae, UBL 2.1), imágenes (JPG, PNG) y facturas escaneadas con baja calidad. Si el OCR falla en un campo, el motor aplica heurísticas de contexto para inferir el dato, reduciendo errores en un 40% frente a soluciones tradicionales.
- **Validación fiscal en tiempo real**: Cada factura extraída se valida contra reglas fiscales locales (tipos de IVA reducido, recargo de equivalencia, retenciones). billripper marca automáticamente facturas con discrepancias, como IVA mal calculado o NIF no válido, sin enviar datos a servidores externos.
- **Exportación directa a APIs contables**: billripper genera JSON estructurado y puede integrarse directamente con las APIs de los principales ERPs (SAP, Oracle NetSuite, Odoo) y plataformas de facturación electrónica (SII, CFDI). Incluye un hook de salida para enviar datos a tu backend sin necesidad de middleware.

### Cómo Usar

1. **Inicializa billripper con configuraciones regionales**: Al instanciar el cliente, pasa un objeto de configuración con el país destino (`es`, `mx`, `ar`) y el modo de fallback. Por ejemplo: `const bill = new BillRipper({ country: 'es', ocrEngine: 'tesseract', fallbackThreshold: 0.7 })`. Esto activa las reglas de parseo específicas para España y usa Tesseract como OCR secundario si el parsing nativo falla.
2. **Procesa un lote de facturas con pipeline asíncrono**: Usa el método `processBatch(files, options)` donde `files` es un array de rutas o buffers. billripper paraleliza el parsing hasta 10 archivos simultáneamente, manejando colisiones de memoria. Ejemplo: `const results = await bill.processBatch(['factura1.pdf', 'factura2.xml', 'factura3.jpg'], { timeout: 30000, retryOnFail: true })`. Cada resultado incluye un campo `confidence` (0-1) y `errors` con detalles de campos no extraídos.
3. **Exporta los resultados con mapeo personalizado**: billripper permite transformar el JSON de salida mediante un callback `mapOutput`. Por ejemplo, para integrar con una API de Odoo: `bill.mapOutput((data) => ({ partner_id: data.supplier_nif, invoice_date: data.date, amount_total: data.total_with_tax }))`. Luego usa `bill.exportToApi('https://tu-erp.com/api/invoices', { headers: { Authorization: 'Bearer token' }})` para enviar los datos directamente.

### Mejores Prácticas

- **Configura el modo `strict` para entornos de producción**: Si tu aplicación maneja facturas críticas (ej. contabilidad oficial), activa `{ strict: true }` en la inicialización. Esto hace que billripper rechace cualquier factura con campos obligatorios faltantes (NIF, total, fecha) y devuelva un error explícito en lugar de inferir datos. Así evitas inconsistencias en auditorías.
- **Usa el pipeline de preprocesamiento para facturas escaneadas de baja calidad**: Antes de llamar a `processBatch`, aplica el método `bill.preprocess(imageBuffer, { denoise: true, contrast: 1.5 })` en facturas escaneadas. Esto mejora la precisión del OCR en un 25% sin aumentar el tiempo de procesamiento. billripper incluye este paso como opción en el parámetro `preprocess` del batch, pero aplicarlo manualmente te da control fino sobre la calidad.
