# billripper - BIL_009

## 概要

billripper は、企業の経理部門やフリーランス向けに設計された**高精度請求書解析エンジン**です。PDF、スキャン画像、メール添付ファイルから、請求書データを構造化し、API 連携や会計ソフトへの自動取り込みを実現します。

## 詳細な内容

### billripper とは？

billripper は、**OCR + 機械学習** を組み合わせた独自の解析エンジンを搭載し、日本特有の「請求書フォーマットの多様性」（手書き、印鑑、複数ページ、折りたたみ明細）に対応。従来のルールベース解析では困難だった**レイアウトフリーなデータ抽出**を可能にします。

### コア機能

1. **動的フィールドマッピング（Dynamic Field Mapping）**  
   請求書ごとに異なる「請求元」「請求先」「日付」「金額」「税区分」「振込先口座」などのフィールドを、**ラベル位置や文脈から自動推定**。設定不要で95%以上の精度でマッピングします。

2. **複数ページ・連結明細の自動結合（Multi-Page Aggregation）**  
   1枚の請求書が複数ページにまたがる場合、または「明細書」と「合計請求書」が別ファイルで送られてくる場合でも、**同一案件と認識し自動マージ**。明細行の合計金額と請求書合計金額の整合性チェックも実行。

3. **リアルタイムAPI エンドポイント（RESTful API + Webhook）**  
   解析結果をJSON形式で即座に返すAPIを提供。さらに、**新しい請求書が到着したタイミングでWebhookを発火**し、外部システム（Freee、MFクラウド、SAP、Oracle）への自動連携が可能。バッチ処理不要で、**リアルタイムな経理処理**を実現。

### 使い方

1. **請求書のアップロード（CLI / API / GUI）**  
   CLI なら `billripper parse invoice_20250301.pdf --format json` で即座に解析。API なら `POST /api/v1/parse` にマルチパートデータを送信。GUI のダッシュボードからもドラッグ＆ドロップでアップロード可能。

2. **フィールドの検証と修正（Validation & Correction Mode）**  
   解析結果が表示されたら、**信頼度スコア（0-100）** を確認。低スコアのフィールドはハイライト表示され、手動で修正可能。修正内容は**即座に学習データとして蓄積**され、次回以降の解析精度が向上。

3. **外部サービスとの連携設定（Integration Setup）**  
   ダッシュボードの「連携設定」タブで、Freee、MFクラウド、SAP、Oracle などの**OAuth認証**をワンクリックで設定。または、カスタムWebhook URLを入力し、解析完了時に任意のエンドポイントへPOSTリクエストを送信。**リトライポリシー（Exponential Backoff）** も標準搭載。

### ベストプラクティス

- **事前にサンプル請求書を10枚アップロードして「初期学習」を行う**  
   billripper はアップロードされた請求書から自動でパターンを学習します。最初に、実際に処理する予定の**バラエティに富んだ10枚**をアップロードすることで、初回から高い精度を発揮します。特に、**レイアウトが異なる請求書（A4縦、A4横、封筒サイズ）を混ぜる**と効果的です。

- **Webhook のエンドポイントは、冪等性（Idempotency）を保証する設計にする**  
   billripper はWebhookの再送信を行う可能性があります。受信側のAPIで**同一の `webhook_id` を持つリクエストを無視するロジック**を実装しておくことで、重複データの登録を防止できます。また、**レスポンスタイムアウトは30秒以上**に設定し、billripper側のタイムアウト（デフォルト10秒）と競合しないように注意してください。
