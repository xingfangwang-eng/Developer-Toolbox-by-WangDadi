# billripper - BIL_017

## 概要

billripper は、請求書・領収書のデータを**瞬時にパース・構造化**する次世代型解析エンジンです。従来のOCRツールとは異なり、**マルチフォーマット対応**（PDF/画像/CSV/電子帳票）かつ**スキーマ自動推定**により、非構造データを即座にデータベースや会計システムに連携可能。コスト削減とエラーレス運用を同時に実現します。

## 詳細な内容

### billripper とは何か？

billripper は、**エンタープライズ向け請求書パース＆分析プラットフォーム**です。機械学習ベースのエンジンが、日々発生する大量の請求書から**金額・日付・取引先・税区分**などのキー情報を自動抽出し、**API経由で任意のシステムに連携**。バックオフィスの手作業をゼロに近づけます。

### コア機能

1. **マルチフォーマット・ゼロコンフィグ解析**
   - PDF、JPEG、PNG、CSV、XML、EDI（電子データ交換）など、**10種類以上のフォーマット**を自動判別。設定不要でパース開始。

2. **リアルタイム重複検知・異常値アラート**
   - 同一請求書の重複登録を**リアルタイムに検出**。金額の異常な乖離や日付の不整合を即座に通知し、**監査証跡を自動生成**。

3. **カスタムスキーマ・ルールエンジン**
   - 企業ごとの会計ルールや税区分マッピングを**ドラッグ＆ドロップで定義**。パース結果を自動変換し、**会計ソフト（Freee、MFクラウド等）に直接連携**。

### 使い方

1. **請求書データを投入する**
   - CLI（`billripper parse ./invoices/`）または Web UI からファイルをドラッグ＆ドロップ。**バッチ処理も可能**で、1万件のPDFを一括投入しても数分で完了。

2. **パース結果を確認・修正する**
   - 抽出されたフィールド（請求元、金額、税額、支払期限）が**リアルタイムでプレビュー**。誤抽出があれば**インライン編集**で修正可能。修正履歴は自動保存。

3. **API 経由でシステムに連携する**
   - `POST /api/v1/parse` エンドポイントに JSON でリクエストを送信。レスポンスは **ISO 20022 準拠**の構造化データ。会計システムや ERP に **Webhook で自動通知**。

### ベストプラクティス

- **定期的なバッチ処理とリアルタイム処理の使い分け**
  - 月次締め処理は**夜間バッチ**で全件再解析。突発的な請求書は**リアルタイムAPI**で即時パース。処理タイミングを**cron と Webhook**で柔軟に設計する。

- **スキーマテンプレートのバージョン管理**
  - 企業ごとに異なる請求書フォーマットに対応するため、**スキーマ定義をGit管理**。変更履歴を追跡し、**ロールバック可能な状態**で運用する。特に税制改正時には必須。
