# brainbridge - BRA_014

## 概要

brainbridge は、断片的なアイデアを意味のある思考ネットワークに変換する、プロフェッショナル向け思考接続ツールです。従来のマインドマップやノートアプリとは異なり、brainbridge は思考間の因果関係・時間的遷移・抽象度を自動解析し、直感的なブリッジ（橋）を生成します。これにより、複雑な問題解決やクリエイティブな発想の構造化を圧倒的な速度で実現します。

## 詳細な内容

### brainbridge とは？

脳科学とグラフ理論をベースに設計された brainbridge は、ユーザーが入力したテキストや音声データをリアルタイムで解析し、思考の断片を「ノード」として可視化。さらに、それらを論理的に接続する「ブリッジ」を自動提案します。特に、プロジェクトの要件整理、アーキテクチャ設計、ブレインストーミング後の整理において、その真価を発揮します。

### コア機能

1. **因果ブリッジ・エンジン**  
   入力されたアイデア間の因果関係を自動検出し、if-then 形式のリンクを生成。例えば「ユーザー離脱率が高い」というノードに対し「オンボーディングフローが複雑」というノードを自動接続し、改善ポイントを可視化します。

2. **時間軸マッピング & コンフリクト検出**  
   各ノードにタイムスタンプと優先度を付与し、時系列での思考の遷移を追跡。さらに、矛盾するアイデア（例：コスト削減 vs 品質向上）を自動検出し、コンフリクト・ブリッジとしてハイライト表示します。

3. **抽象度レイヤリング & 自動要約**  
   ノードを「抽象」「具体」「メタ」の3レイヤーに自動分類。高抽象度のノードから自動的にサマリを生成し、プロジェクト全体のエッセンスを1クリックで抽出可能。これにより、ステークホルダーへの報告資料作成時間を80%削減します。

### 使い方

1. **ブレインストーミング・セッションの開始**  
   まず、brainbridge のダッシュボードで「新規セッション」を作成。キーワードやテーマを入力後、音声入力またはテキストエディタで自由にアイデアを列挙します。この際、各アイデアは自動的にノード化され、リアルタイムでグラフビューに反映されます。

2. **ブリッジ提案のレビュー & カスタマイズ**  
   ノードが10個以上集まると、エンジンが自動でブリッジ候補を提案。例えば「A: サーバーコスト削減」と「B: オートスケーリング導入」の間に「因果関係: B は A を実現する手段」というブリッジが表示されます。ユーザーはこれを承認・拒否・編集可能。また、手動で任意のノードをドラッグ＆ドロップし、独自のブリッジを張ることもできます。

3. **エクスポート & プロジェクト連携**  
   最終的な思考ネットワークは、JSON / Markdown / Mermaid 形式でエクスポート可能。特に Mermaid 形式は、Notion や GitHub のドキュメントに直接埋め込めるため、アーキテクチャ設計書やプロジェクト計画書への流用がシームレス。さらに、Jira や Linear との API 連携により、ノードをタスクとして自動登録することも可能です。

### ベストプラクティス

- **週次デブリーフィングに活用する**  
  毎週金曜日に、その週の開発タスクや議論を brainbridge に投入し、因果ブリッジを確認。これにより、隠れた依存関係やボトルネックを早期発見できます。特に、複数チームが関わるクロスファンクショナルなプロジェクトで効果を発揮します。

- **アーキテクチャ決定記録（ADR）の補完ツールとして**  
  技術選定や設計判断の際、brainbridge 上で議論の経緯をノード化。後日、なぜその選択をしたのかをトレースする際に、因果ブリッジが「意思決定のログ」として機能します。これにより、プロジェクトメンバーの入れ替わりがあっても、ナレッジロスを最小限に抑えられます。
