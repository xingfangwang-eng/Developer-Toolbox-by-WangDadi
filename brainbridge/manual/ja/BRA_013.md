**安装指南**

### 步骤 1：安装 Redis

```bash
sudo apt-get install redis-server
```

### 步骤 2：配置 Redis

```properties
port 6379
maxclients 10000
timeout 0
```

### 步骤 3：启动 Redis

```bash
redis-server --port 6379
```

**性能调优**

### 建议 1：调整 Redis 的内存大小

* Redis 的内存大小可以根据系统的实际需求进行调整，以避免内存泄露和缓慢的读写操作。
* 建议将 Redis 的内存大小设置为 1000MB 到 5000MB 之间。

### 建议 2：优化 Java 的垃圾回收

* Java 的垃圾回收机制可以根据系统的实际需求进行调整，以避免因垃圾回收而导致的性能下降。
* 建议使用 `G1` 垃圾回收器，可以在较短时间内完成垃圾回收任务。
