# brainbridge - BRA_029

## 概要

brainbridge は、散在する思考フラグメントと外部知識ベースをリアルタイムで接続し、**思考の連続性と構造化**を実現するコグニティブ・ブリッジツールです。従来のマインドマップやノートツールとは一線を画し、**意味的関連性の自動検出**と**動的グラフ構築**により、アイデアの「橋渡し」を自動化します。

## 詳細な内容

### brainbridge とは？

brainbridge は、**思考の断片（Thought Node）** を入力とし、それらを**意味的類似性・因果関係・時間的文脈**に基づいて自動リンクするプロフェッショナルツールです。特に、**研究・開発・戦略立案**のフェーズにおいて、**暗黙知の形式化**と**思考の可視化**を支援します。単なるメモツールではなく、**コグニティブ・アーキテクチャ**の一部として機能します。

### コア機能

1. **セマンティック・ブリッジ・エンジン**
   - 入力されたテキスト断片（Thought Node）を**ベクトル埋め込み**し、**意味的類似度スコア**に基づいて自動リンクを生成。リンク強度は**コサイン類似度**で動的に調整され、ユーザーは**スコア閾値**を調整可能。
   - リンクされたノードは**グラフDB（Neo4j互換）** に保存され、**時系列の思考遷移**を追跡可能。

2. **コンテキスト・アウェア・リコメンデーション**
   - 現在の思考ノードに対して、**過去の関連ノード**や**外部知識ベース（Wikipedia、arXiv、社内Wiki）** から**関連性の高いリソース**を自動推薦。
   - 推薦ロジックは**トランスフォーマーモデル（BERTベース）** で実装され、**ドメイン適応型ファインチューニング**に対応。

3. **ダイナミック・ナレッジグラフ・ビジュアライゼーション**
   - 思考ノードとリンクを**3Dフォースダイレクトグラフ**で可視化。**ノードの色・サイズ**は**重要度**や**更新頻度**に基づいて動的に変化。
   - **フィルタリング機能**により、**特定のタグ**や**日付範囲**でグラフを絞り込み、**思考のパターン**を発見可能。

### 使い方

1. **Thought Node の作成とリンク生成**
   - 任意のテキスト（アイデア、メモ、コードスニペット）を **`brainbridge add`** コマンドで投入。
   - 自動リンク生成は **`brainbridge link --threshold 0.75`** で実行。スコア0.75以上のノードが自動リンクされる。
   - リンク結果は **`brainbridge graph --format json`** でエクスポート可能。

2. **コンテキスト・リコメンデーションの活用**
   - 現在のノードIDを指定し、**`brainbridge recommend --node-id <ID> --source arxiv`** を実行。
   - 結果は**関連性スコア順**に表示され、**`--limit 5`** で取得数を制限可能。
   - 外部ソースは **`brainbridge config --source add arxiv`** で動的に追加可能。

3. **ナレッジグラフのカスタマイズ**
   - 可視化は **`brainbridge visualize --theme dark --node-size importance`** で起動。
   - **`--filter tag:research`** で特定タグのみ表示。
   - グラフは **`brainbridge export --format png`** で画像出力可能。

### ベストプラクティス

1. **思考ノードの粒度を統一する**
   - ノードは**1アイデア＝1ノード**の原則で作成。複数のアイデアを1ノードに詰め込むと、リンク精度が低下する。**粒度は50〜200文字**が最適。長文は **`brainbridge split`** で自動分割可能。

2. **外部知識ベースとの連携を定期実行する**
   - **cronジョブ**で **`brainbridge recommend --source all`** を毎日実行し、新しい研究論文や記事とのリンクを自動更新。これにより、**思考の陳腐化**を防止し、**最新の知見**を常に反映。
