### 2. コード インストール ガイド

#### 2.1. nukeprivacy  Installation Guide

```bash
# Step 1: Install Java Development Kit (JDK)
sudo apt-get install openjdk-11-jdk

# Step 2: Download and extract the nukeprivacy source code
wget https://github.com/nukeprivacy/nukeprivacy/archive/master.zip
unzip master.zip

# Step 3: Compile the nukeprivacy source code
cd nukeprivacy-master/
ant compile

# Step 4: Run the nukeprivacy application
java -jar nukeprivacy.jar
```

#### 3.1.  Thread Pool Size Adjustment

To improve performance under high concurrency, we recommend adjusting the thread pool size to optimize the number of threads based on the system load.

* For low to medium load, set the thread pool size to 4-8.
* For high load, set the thread pool size to 16-32.
