**Installation Guide**

### Step 1: Install Redis

```
apt-get install redis-server
```

### Step 2: Configure Redis

Create a `redis.conf` file with the following settings:
```yaml
port 6379
maxmemory 1000m
maxclients 10000
```

### Step 3: Install Akka

```
sbt akka-actor_2.13
```

### Step 4: Configure Akka

Create a `akka.conf` file with the following settings:
```yaml
akka.actor.default-dispatcher {
  fork-join-executor {
    parallelism-min = 5
    parallelism-max = 20
  }
}
```
