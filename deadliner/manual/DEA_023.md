### What is deadliner?

deadliner is a professional tool that helps solve problems in related fields.

### Core Features

- **Automatic Deadline Propagation with Dependency Resolution**: When a task's due date shifts, deadliner automatically recalculates all dependent tasks in the chain—no more manual cascading updates across your project timeline.
- **Multi-Timezone Deadline Normalization**: Converts all deadlines to UTC internally while displaying them in each team member's local timezone, eliminating the 3 PM vs 15:00 confusion across distributed teams.
- **Recurring Deadline Templates with Smart Skip Logic**: Define complex recurrence patterns (e.g., "every last Friday of quarter, except December") and automatically skip dates that fall on holidays or weekends.

### How to Use

1. **Initialize a Project with Dependency Graph**: Run `deadliner init --project "Q4 Launch" --deps` to create a new project. Then define task relationships using the YAML syntax: `task_b depends_on: task_a`. deadliner will parse the graph and flag any circular dependencies before execution.
2. **Set a Recurring Deadline with Exclusion Rules**: Use the CLI command `deadliner schedule --pattern "0 9 * * 1" --except "2024-12-25,2025-01-01"` to create a weekly Monday 9 AM deadline that skips Christmas and New Year's Day. The tool automatically adjusts the next occurrence.
3. **Trigger a Deadline Cascade Update**: When a milestone slips, execute `deadliner cascade --task-id "MILESTONE-3" --new-date "2025-03-15"`. deadliner recalculates all downstream deadlines, outputs a diff report, and optionally sends Slack notifications to affected team members.

### Best Practices

- **Use Time Budgets, Not Just Deadlines**: For each task, set a `time_budget` (e.g., 8h) alongside the deadline. deadliner will warn you if the remaining time between now and the deadline is less than the budget—preventing last-minute crunches.
- **Tag Deadlines with Priority Levels**: Assign `priority: critical | high | medium | low` to every deadline. deadliner's dashboard then color-codes your timeline, and you can filter by priority to focus on what actually needs to ship today.
