## 2. Installation Guide

### 2.1. Step 1: Install Required Dependencies

```bash
sudo apt-get install -y redis-server memcached gcc make
```

### 2.2. Step 2: Configure Redis and Memcached

Create a `redis.conf` file with the following contents:
```properties
port 6379
maxmemory 10000mb
```
Create a `memcached.conf` file with the following contents:
```properties
port 11211
max_memory 10000mb
```

### 2.3. Step 3: Install and Configure HAProxy

Install HAProxy using the following command:
```bash
sudo apt-get install -y haproxy
```
Create a `haproxy.conf` file with the following contents:
```properties
global
    daemon
    maxconn 256

defaults
    mode http
    timeout connect 5000
    timeout client  10000
    timeout server  10000

listen http_proxy 0.0.0.0:80
    balance roundrobin
    server node1 127.0.0.1:8080 weight 1 maxconn 128
    server node2 127.0.0.1:8081 weight 1 maxconn 128
```
