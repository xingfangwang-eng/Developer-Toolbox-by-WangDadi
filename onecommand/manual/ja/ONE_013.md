### Installation Guide

To install onecommand, follow these steps:

**Step 1: Install Java 8 (or later)**

```
sudo apt-get update
sudo apt-get install openjdk-8-jdk -y
```

**Step 2: Set up Apache Ignite**

```
wget https://ignite.apache.org/download.cgi#apache-ignite-3.10.0-bin-cp.zip
unzip apache-ignite-3.10.0-bin-cp.zip
sudo mv apache-ignite-3.10.0-bin /usr/local/ignite

sudo chown -R $USER:$GROUP /usr/local/ignite
```

**Step 3: Configure onecommand**

Create a `config.properties` file with the following contents:

```properties
# Distributed cache configuration
distributed.cache.enabled=true
distributed.cache.nodes=2-4

# Multi-threaded optimization configuration
multithreaded.optimization.threads=8

# Atomic operations configuration
atomic.operations.lock.timeout=1000
```
