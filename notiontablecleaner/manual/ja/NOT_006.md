在下载的 NotionTableCleaner 文件夹中，执行以下命令来安装依赖项：
```bash

```

在 NotionTableCleaner 文件夹中，执行以下命令来运行 NotionTableCleaner：
```bash
java -jar notiontablecleaner.jar
```
