### How To Use

#### Step 1: Create a New Script
- Open your preferred text editor.
- Paste your code into the editor.
- Save it as a `.txt` file or use the `scriptkill -f [file-name]` command to automatically generate an optimized script file. 

#### Step 2: Analyze Your Scripts
- After saving, open `scriptkill`.
- Select the directory where your scripts are saved using the drop-down menu.
- Scriptkill will analyze all .txt files within this directory.

#### Step 3: Improve Your Scripts
