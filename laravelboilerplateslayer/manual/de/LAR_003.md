## 快速开始
### 步骤一：安装配置

1. Clone the Laravel Boilerplate Slayer repository: `git clone https://github.com/wangdadi/laravel-boilerplate-slayer.git`
2. Run the installation script: `php artisan slayer:install`
3. Configure your environment variables according to our documentation.

### 步骤二：初始化

1. Initialize your project by running the following command: `php artisan slayer:init`
2. Set up your database schema using our intuitive migration tool.
3. Create a new route and start building your application!

### 步骤三：运行验证

1. Run your application in development mode: `php artisan serve`
2. Test your API endpoints using Postman or cURL.
3. Verify that your app responds quickly and efficiently - it should!
