## Schnellstart
### Schritt 1: Installation und Konfiguration

Um Laravelboilerplateslayer zu installieren, benötigen Sie lediglich ein paar Minuten. Fügen Sie den folgenden Code in Ihrem Terminal ein:
```bash
composer create-project --prefer-dist laravel/laravelboilerplateslayer example
```
### Schritt 2: Initialisierung

Nach der Installation müssen Sie die Boilerplatten initialisieren. Öffnen Sie das Projektverzeichnis und führen Sie den folgenden Befehl aus:
