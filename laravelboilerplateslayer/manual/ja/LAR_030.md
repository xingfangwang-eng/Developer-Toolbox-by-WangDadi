## Quick Start
### 步骤一：インストール設定
1. プロジェクトディレクトリーに移動する。
2. `composer install` を実行する。
3. `.env` ファイルを更新し、DB 設定を行う。
4. `npm install` を実行する。

### 步骤二：初期化
1. プロジェクトディレクトリーに移動する。
2. `php artisan key:generate` を実行する。
3. `.env` ファイルを更新し、DB 設定を行う。
4. `npm run dev` を実行する。

### 步骤三：ランニング検証
1. プロジェクトディレクトリーに移動する。
2. `php artisan serve` を実行する。
3. ブラウザーで `http://localhost:8000` にアクセスする。
