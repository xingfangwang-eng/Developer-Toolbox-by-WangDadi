### Core Features

- The ping function checks if a server can be reached using ICMP packets.
- The monitor function tracks network statistics such as average packet size and delay time.
- The connection test function tests the status of a network connection, including connectivity and latency.

### How to Use

1. Step 1: To check if a specific server is reachable:
   - Enter the hostname or IP address of the server you want to ping into the 'IP' field.
   - Click on the 'Ping' button to initiate the test.

2. Step 2: To monitor network statistics continuously:
   - In the 'Monitor' section, select how often (e.g., every minute) you'd like to collect data about your network connection.
   - Click on 'Start Monitoring'.

3. Step 3: To check the status of a network connection:
   - Enter the IP address of the server into the 'Destination IP' field.
   - The system will show whether it's up or not, as well as the overall latency.

### Best Practices

- When using the ping function to monitor network connections, always ensure that the destination IP is correct and up-to-date.
- For comprehensive network monitoring, consider integrating other features like packet loss detection or bandwidth usage statistics.
