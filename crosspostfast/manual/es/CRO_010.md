## Resumen Técnico

Herramienta de publicación cruzada optimizada para SEO y gestión de contenido multi-plataforma. Elimina fricciones en la distribución de contenido entre redes sociales, con soporte nativo para APIs REST, webhooks y programación en lote. Ideal para equipos de marketing y desarrolladores que necesitan sincronización en tiempo real sin depender de intermediarios.

## Contenido Detallado

### ¿Qué es crosspostfast?

**crosspostfast** es un motor de publicación multi-plataforma diseñado para desarrolladores y estrategas de contenido. No es un simple programador de posts: es un agente autónomo que gestiona la adaptación semántica, la optimización de metadatos y la validación de formatos específicos de cada red social (Twitter, LinkedIn, Instagram, Facebook, TikTok, etc.). Su núcleo está construido sobre un sistema de colas asíncronas y cacheo inteligente, lo que permite manejar miles de publicaciones simultáneas sin degradación de rendimiento.

### Características Principales (Hardcore)

1. **Adaptación Semántica Automática por Plataforma**  
   crosspostfast analiza el contenido original y lo reestructura según las reglas de cada red: límites de caracteres, formato de hashtags, tipos de medios (imagen, video, carrusel), y restricciones de enlaces. Usa un modelo de reglas basado en heurísticas predefinidas y permite sobrescritura por API.

2. **Pipeline de Validación y Fallback con Retry Inteligente**  
   Cada publicación pasa por un pipeline de validación: verificación de formato, detección de contenido duplicado, comprobación de enlaces rotos y validación de tokens OAuth. Si una publicación falla en una plataforma, el sistema ejecuta un retry exponencial con backoff y notifica por webhook. No se pierde ni un post.

3. **Sincronización Bidireccional con Estados y Logs**  
   No solo publicas: crosspostfast mantiene un registro de estado completo (pendiente, publicado, fallido, en cola) y permite consultar logs en tiempo real vía API REST. Además, ofrece un endpoint de callback para que tu sistema sepa exactamente cuándo se completó cada acción. Ideal para integraciones CI/CD o automatizaciones con Zapier/n8n.

### Cómo Usar (Guía Técnica)

1. **Configuración de Tokens y Endpoints por Plataforma**  
   - Accede a la sección `Configuración > Proveedores` y añade tus credenciales OAuth para cada red social.  
   - crosspostfast soporta autenticación directa (API Key) y flujos OAuth 2.0 con refresh automático.  
   - Ejemplo para Twitter: `POST /api/v1/providers/twitter` con `{ "api_key": "...", "api_secret": "...", "access_token": "...", "access_token_secret": "..." }`.  
   - Verifica la conexión con `GET /api/v1/providers/status`. Si hay error, el sistema te devuelve el mensaje exacto del proveedor.

2. **Creación de una Campaña de Publicación Multi-Plataforma**  
   - Define tu contenido en JSON: `{ "content": "Texto del post", "media": ["url1.jpg", "url2.mp4"], "platforms": ["twitter", "linkedin", "instagram"], "schedule": "2025-04-15T10:00:00Z" }`.  
   - crosspostfast adapta automáticamente el texto y los medios para cada red. Si Instagram no acepta enlaces, los convierte en texto plano.  
   - Envía la solicitud a `POST /api/v1/campaigns`. Recibirás un `campaign_id` y un array de `job_ids` (uno por plataforma).  
   - Monitorea el progreso con `GET /api/v1/campaigns/{id}/status`. Cada job tiene su propio estado y log.

3. **Implementación de Webhooks para Notificaciones en Tiempo Real**  
   - Ve a `Configuración > Webhooks` y añade un endpoint HTTPS donde crosspostfast enviará eventos.  
   - Eventos disponibles: `job.completed`, `job.failed`, `campaign.completed`, `campaign.failed`.  
   - Ejemplo de payload: `{ "event": "job.completed", "job_id": "abc123", "platform": "twitter", "status": "published", "timestamp": "2025-04-15T10:00:05Z" }`.  
   - Usa esto para activar acciones posteriores (ej: enviar un mensaje a Slack, actualizar tu CRM, o lanzar una nueva campaña). crosspostfast reintenta hasta 3 veces en caso de fallo del webhook.

### Mejores Prácticas para Desarrolladores y Estrategas

1. **Usa la API de Colas para Cargas Masivas sin Bloqueo**  
   Si necesitas publicar cientos de posts en minutos, no envíes solicitudes síncronas. crosspostfast expone un endpoint de cola (`POST /api/v1/queue`) que acepta un array de campañas. El sistema las procesa en lotes de 50, con un límite de 10 solicitudes concurrentes por plataforma. Esto evita rate limiting y mejora la fiabilidad. Ideal para migraciones de contenido o lanzamientos de producto.

2. **Implementa un Sistema de Logs y Auditoría con Exportación a CSV**  
   crosspostfast guarda cada acción con timestamp, IP de origen, usuario y resultado. Habilita la exportación de logs desde `Configuración > Logs` o vía API (`GET /api/v1/logs?format=csv`). Esto es crítico para auditorías de cumplimiento (GDPR, CCPA) y para depurar fallos en producción. Además, puedes configurar la retención de logs (7, 30 o 90 días) para optimizar almacenamiento.
