### What is crosspostfast?

crosspostfast is a professional tool designed to streamline and automate the process of publishing content across multiple social media platforms. It eliminates the need for manual posting, reduces errors, and ensures consistency in branding and messaging. Whether you're a solo creator, a marketing agency, or a large enterprise, crosspostfast saves time and boosts efficiency by handling cross-platform scheduling, formatting, and analytics in one unified dashboard.

### Core Features

- **Automated Cross-Platform Formatting Engine**: Automatically adapts your content (text, images, videos) to the optimal format for each platform—Twitter's character limit, Instagram's aspect ratio, LinkedIn's rich media, and more—without manual tweaking. This engine uses a rules-based system that respects each API's constraints, ensuring your post looks native and professional everywhere.

- **Intelligent Queue Management with Smart Scheduling**: Prioritize and schedule posts based on platform-specific peak engagement times, audience time zones, and content type. The queue uses a machine learning model to predict optimal posting windows, reducing manual guesswork. You can also set auto-retry for failed posts with exponential backoff, ensuring delivery even during API outages.

- **Unified Analytics Dashboard with Cross-Platform Comparison**: Track performance metrics (likes, shares, comments, clicks) across all connected platforms in real-time. The dashboard provides a normalized comparison view, allowing you to see which content resonates best on which channel. Export raw data in CSV or JSON for further analysis in your own BI tools.

### How to Use

1. **Connect Your Social Profiles via OAuth**: Navigate to the "Accounts" tab and click "Add Platform." Select from Twitter, LinkedIn, Instagram, Facebook, or custom APIs. Authenticate using OAuth 2.0—no passwords stored. You can connect multiple accounts per platform (e.g., 3 Twitter handles). The system validates permissions (read/write) and stores tokens securely with encryption at rest.

2. **Create a Cross-Platform Post Using the Composer**: In the "New Post" composer, write your main copy. The system automatically suggests platform-specific variants—truncate for Twitter, add hashtags for Instagram, format for LinkedIn. You can manually override each variant. Attach media (images or videos) and the engine will resize/crop them per platform specs. Preview each platform's final look before scheduling.

3. **Schedule or Publish with Queue Rules**: Choose "Publish Now" for immediate cross-posting, or "Add to Queue" to set a specific time. Use the "Smart Schedule" toggle to let the system auto-select the best time per platform based on historical engagement data. You can also set priority (high/medium/low) to control queue order. The system will automatically retry failed posts up to 3 times with a 5-minute delay between attempts.

### Best Practices

- **Use Platform-Specific Hashtags and Mentions**: Avoid using the same hashtags across all platforms. For Instagram, use up to 30 relevant hashtags; for LinkedIn, limit to 3-5 professional tags. crosspostfast lets you set per-platform hashtag sets in the composer. This increases visibility and avoids spam flags.

- **Leverage the Smart Scheduling for Time-Sensitive Content**: If you're posting breaking news, event announcements, or time-limited offers, use the "Smart Schedule" feature to ensure each platform gets the post at its optimal local time. For example, post at 9 AM EST for LinkedIn (business audience) and 7 PM PST for Instagram (evening scroll). This maximizes engagement without manual timezone calculations.
