# crosspostfast - CRO_015

## 概要

クロスプラットフォーム投稿の自動化ツール。Twitter、LinkedIn、Threads、Bluesky への同時配信をワンクリックで実現。SEO 最適化済みメタデータ管理機能を内蔵し、マルチプラットフォームパブリッシングを効率化する。

## 詳細な内容

### crosspostfast とは

crosspostfast は、ソーシャルメディア運用者が複数プラットフォームへ同時にコンテンツを配信するための SaaS ツールです。各プラットフォームの API 制限や文字数制限を自動検出し、最適なフォーマットで投稿を生成します。バックグラウンドでジョブキューを管理し、失敗時のリトライ処理も自動実行します。

### コア機能

- **プラットフォーム間リンク解決エンジン**: 各プラットフォームの投稿 ID を自動紐付け。同一コンテンツが複数プラットフォームに存在する場合、相互リンクを動的に生成し、Google の重複コンテンツペナルティを回避。さらに、Open Graph タグを動的生成し、SNS シェア時のプレビューを最適化。
- **マルチフォーマットアダプティブテンプレート**: 各プラットフォームの文字数制限（Twitter: 280字、LinkedIn: 3000字、Threads: 500字）に応じて、コンテンツを自動分割・再構成。画像は WebP に自動変換し、CDN 経由で配信。動画は HLS ストリーミングに変換し、再生パフォーマンスを最大化。
- **スケジュール最適化アルゴリズム**: 投稿時間をプラットフォームごとのアクティブユーザー分布に基づいて自動調整。過去のエンゲージメントデータを基に、最適な投稿時間帯を機械学習で予測。キャッシュ無効化戦略を組み込み、CDN パージを自動実行。

### 使い方

1. **API キー登録とプラットフォーム紐付け**  
   プラットフォームごとに OAuth 2.0 認証を実施。各プラットフォームの API スコープを最小権限で設定し、リフレッシュトークンを安全に保存。開発者コンソールから Client ID と Client Secret を取得し、crosspostfast のダッシュボードで登録。登録後、自動で Webhook エンドポイントが生成され、投稿完了通知を受け取れる。

2. **テンプレート作成と変数マッピング**  
   Jinja2 テンプレートエンジンを使用し、プラットフォームごとに異なるテンプレートを作成。変数として `{title}`, `{description}`, `{url}`, `{image_url}` を定義し、各プラットフォームの文字数制限に応じて自動展開。例えば、Twitter 用テンプレートでは `{title}` を 80 字にトリミングし、`{url}` を短縮 URL に変換。LinkedIn 用では `{description}` を 150 字に要約。

3. **スケジュール設定とバッチ投稿**  
   cron 式または ISO 8601 形式で投稿スケジュールを指定。バッチ投稿機能を使用すると、複数のプラットフォームに同時に投稿可能。投稿前にプレビューモードで各プラットフォームの表示を確認。投稿後は、各プラットフォームの API レスポンスを検証し、エラーが発生した場合は自動リトライ（最大 3 回、指数バックオフ）。

### ベストプラクティス

- **プラットフォームごとに個別のテンプレートを持つ**  
  全プラットフォームに同じテキストを投稿するのではなく、各プラットフォームの文化や文字数制限に合わせてテンプレートを最適化。例えば、Twitter では短く簡潔に、LinkedIn では専門性を強調した文章に。テンプレートは Git で管理し、変更履歴を追跡可能にする。

- **定期的な API エンドポイントのヘルスチェック**  
  各プラットフォームの API は頻繁に仕様変更が発生する。crosspostfast のダッシュボードで週次でヘルスチェックを実行し、エンドポイントの応答時間やエラー率を監視。異常が検出された場合は、Slack や Discord への Webhook 通知を設定し、即座に対応できる体制を整える。
