**Installation Guide**

Here's a step-by-step installation guide for Office-Sync:

### Step 1: Install Redis

```bash
sudo apt-get install redis-server
```

### Step 2: Configure Memcached

```bash
sudo apt-get install memcached
sudo nano /etc/memcached.conf
```

Add the following lines to the configuration file:
```properties
-b 127.0.0.1
-c 1024
-m 64M
-p 11211
```

### Step 3: Start Office-Sync Services

```bash
sudo systemctl start office-sync
sudo systemctl enable office-sync
```
