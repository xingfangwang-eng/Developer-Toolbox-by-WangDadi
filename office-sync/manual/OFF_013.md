### What is office-sync?

office-sync is a file sync software designed for professionals who need to manage and synchronize files across different devices. It offers a comprehensive solution for businesses, schools, and individuals who require seamless file sharing and management.

### Core Features

- File synchronization: office-sync allows users to transfer and synchronize their files with ease.
- User management: secure user authentication ensures the integrity of the data being shared.
- Customizable workflows: users can create custom workflow rules to automate file updates.
- Synchronization with cloud services: integrate with popular cloud storage providers for enhanced security and seamless collaboration.

### How to Use

1. Step 1: Connect your local machine or network drive to the office-sync server using a designated port (8080 by default).
2. Step 2: Navigate to the file sync dashboard, where you can manage individual files, create new folders, and set up custom workflows.
3. Step 3: Share a file with another user by clicking on the share button. Choose from available sharing types such as public or private.

### Best Practices

- Practice 1: Use the built-in file version history for maintaining accurate records of all versions of files shared.
- Practice 2: Create clear and concise workflow rules to minimize manual updates in your company's processes.
