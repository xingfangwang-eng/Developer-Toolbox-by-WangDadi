# capsule-factory-saas - CAP_023

## Resumen

Plataforma SaaS de gestión integral para fábricas de cápsulas, diseñada para optimizar la producción, trazabilidad y calidad en entornos farmacéuticos y nutracéuticos. Automatiza el flujo desde la materia prima hasta el empaquetado, con dashboards en tiempo real y cumplimiento normativo GMP.

## Contenido Detallado

### ¿Qué es capsule-factory-saas?

**capsule-factory-saas** es una solución cloud-native que unifica la planificación de producción, control de calidad y logística en fábricas de cápsulas duras y blandas. Integra sensores IoT, gestión de lotes y alertas predictivas para reducir mermas y asegurar la trazabilidad farmacéutica. Ideal para plantas que buscan escalar sin perder cumplimiento regulatorio (FDA, EMA, COFEPRIS).

### Características Principales

- **Trazabilidad Lote-2-Lote con Blockchain Inmutable**  
  Cada cápsula se asocia a un hash único desde la granulación hasta el blister. El sistema registra automáticamente temperatura, humedad y tiempos de encapsulado, generando certificados digitales auditables sin intervención manual.

- **Optimización Predictiva de Velocidad de Encapsulado**  
  Algoritmos de machine learning ajustan en tiempo real la velocidad del cabezal de encapsulado (rpm) según la viscosidad del relleno y la humedad ambiental. Reduce roturas de cápsula hasta un 40% y elimina paradas por atascos recurrentes.

- **Dashboard de OEE y Rendimiento por Máquina**  
  Visualiza en vivo la Efectividad General del Equipo (OEE) desglosada por disponibilidad, rendimiento y calidad. Incluye alertas configurables cuando el rendimiento cae por debajo del 85% y recomendaciones de mantenimiento predictivo.

### Cómo Usar

1. **Configurar el Modelo de Producción y Parámetros de Calidad**  
   Accede al módulo `Configuración > Líneas de Producción`. Define el tipo de cápsula (gelatina dura, HPMC, blanda), los límites de peso promedio (ej. 500 mg ± 5%) y los puntos de control de calidad (disolución, desintegración). Asigna sensores IoT a cada máquina mediante la API REST (`POST /api/v1/machines`). Guarda y despliega la configuración; el sistema comenzará a capturar datos en tiempo real.

2. **Ejecutar una Orden de Producción con Monitoreo en Vivo**  
   En el panel `Órdenes`, crea una nueva orden ingresando el SKU del producto, cantidad objetivo (ej. 200,000 cápsulas) y fecha límite. Activa la opción `Monitoreo Predictivo` para que el algoritmo ajuste automáticamente la velocidad de encapsulado. Durante la ejecución, revisa el gráfico de OEE y los indicadores de calidad; si el peso promedio se desvía, el sistema pausa la línea y envía una notificación al supervisor vía webhook.

3. **Auditar un Lote y Generar Certificado de Cumplimiento**  
   Finalizada la orden, ve al módulo `Trazabilidad`. Selecciona el lote y haz clic en `Generar Informe`. El sistema compila automáticamente: registros de temperatura/humedad, resultados de pruebas de calidad, y el hash blockchain. Exporta el PDF firmado digitalmente o compártelo mediante enlace seguro. Para integraciones ERP, usa `GET /api/v1/lotes/{id}/certificado`.

### Mejores Prácticas

- **Calibración Semanal de Sensores con Datos Sintéticos**  
  Para mantener la precisión del modelo predictivo, programa una calibración semanal usando el endpoint `/api/v1/calibration/simulate`. Envía datos sintéticos de temperatura y viscosidad (ej. 22°C, 350 cP) y verifica que el algoritmo ajuste la velocidad esperada. Esto evita deriva del modelo en entornos con alta variabilidad estacional.

- **Segmentación de Alertas por Rol y Canal**  
  Configura reglas de notificación granulares: alertas críticas (rotura de cápsula, desviación de peso >10%) deben ir a Slack y SMS del jefe de turno; alertas informativas (bajo rendimiento de máquina) solo al dashboard y correo diario. Usa el módulo `Alertas > Reglas` con condiciones tipo `IF (oee < 70%) THEN (notificar a rol:supervisor via:slack)`. Esto reduce ruido y acelera la respuesta ante fallos.
