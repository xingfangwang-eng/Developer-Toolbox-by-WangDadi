# capsule-factory-saas - CAP_011

## Resumen Ejecutivo

**capsule-factory-saas** es una plataforma SaaS modular diseñada para la gestión integral de producción de cápsulas (farmacéuticas, nutracéuticas o de café). Optimiza el ciclo de vida del lote, desde la formulación hasta el envasado, con trazabilidad blockchain-ready y dashboards en tiempo real para operaciones multi-sede. Ideal para fabricantes que buscan escalar sin perder control de calidad ni compliance regulatorio (GMP, FDA, ISO 22000).

---

## Funcionalidades Clave (Hardcore)

### 1. **Motor de Formulación Dinámica con Pesaje Inversivo**
El sistema permite crear recetas maestras con variables paramétricas (humedad, granulometría, tiempo de mezcla). Cada lote ajusta automáticamente las cantidades de excipientes y principios activos según la densidad aparente medida *in situ*. Soporta cálculos de desviación estándar en tiempo real y alerta si el coeficiente de variación supera el 2%.

### 2. **Trazabilidad Lote-a-Lote con Hash Inmutable**
Cada lote genera un identificador único (UUID v4 + timestamp nanosegundo) que se encadena al lote anterior mediante un hash SHA-256. Esto permite auditorías ciegas y certificación de origen sin necesidad de un tercero. Ideal para exportación a mercados con exigencias de *track & trace* (UE FMD, FDA DSCSA).

### 3. **Planificador de Producción con Algoritmo de Cuello de Botella Predictivo**
El scheduler integra variables de mantenimiento preventivo, disponibilidad de moldes y velocidad de encapsuladoras (modelos específicos: Bosch GKF, IMA Zanasi, MG2). Predice con un 94% de precisión la fecha de finalización de un lote, incluso con paradas no programadas. Se integra con SCADA vía OPC-UA.

## Cómo Usar (Pasos Detallados)

### 1. **Configurar una Receta Maestra con Parámetros Variables**
- Accede al módulo **Formulación** → selecciona “Nueva Receta”.
- Define los componentes activos y excipientes, asignando rangos de tolerancia (ej: cafeína 99.5%–100.2%).
- Activa el flag **Pesaje Dinámico** y vincula los sensores de balanza vía API REST.
- Guarda la receta como versión 1.0. El sistema generará un *Bill of Materials* (BOM) en formato JSON.

### 2. **Crear una Orden de Producción con Trazabilidad Activa**
- Ve a **Órdenes** → “Nueva Orden”. Selecciona la receta maestra.
- Define la cantidad objetivo (ej: 500.000 cápsulas) y la línea de producción.
- El sistema asigna automáticamente el *lote padre* y genera un *QR de lote hijo* por cada sub-lote de 10.000 unidades.
- Durante la producción, cada evento (llenado, pesaje, encapsulado) se registra con timestamp y firma digital.

### 3. **Ejecutar una Auditoría de Trazabilidad Inversa**
- En el dashboard, ingresa el número de lote de un producto terminado.
- El sistema despliega el árbol completo: desde el lote de materia prima hasta la máquina encapsuladora y el operador.
- Exporta el reporte en PDF con sello de tiempo y hash de verificación. Este documento es aceptado por entes reguladores como evidencia de trazabilidad.

## Mejores Prácticas Recomendadas

### 1. **Estandariza la Nomenclatura de Lotes**
Usa el formato `YYYYMMDD-LINEA-SEQUENCIAL` (ej: `20250415-L2-00123`). Esto facilita búsquedas en logs y evita colisiones. Configura el sistema para rechazar órdenes con IDs duplicados automáticamente.

### 2. **Implementa Alertas de Desviación en Tiempo Real**
Configura reglas de negocio en el módulo de monitoreo: si la temperatura de la cápsula supera los 35°C durante más de 10 segundos, el sistema debe pausar la línea y notificar al supervisor vía WebSocket. Esto reduce mermas por degradación térmica.
