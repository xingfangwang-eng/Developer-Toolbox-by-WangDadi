### ¿Qué es capsule-factory-saas?

`capsule-factory-saas` es un sistema de gestión de producción modular que centraliza datos de máquinas de encapsulado, lotes de materia prima, parámetros ambientales (humedad/temperatura) y trazabilidad de calidad. A diferencia de un ERP genérico, este software está diseñado específicamente para resolver cuellos de botella en líneas de cápsulas: tiempos de secado inconsistentes, desviaciones en peso de relleno y compliance regulatorio.

### Características Principales

1. **Motor de Batch Tracking con Hash de Gelatina**  
   Cada lote de gelatina se registra con un hash único generado a partir de su composición química (Bloom, viscosidad, pH). El sistema cruza automáticamente estos datos con las condiciones de secado para predecir la tasa de ruptura de cápsulas, permitiendo ajustes en tiempo real.

2. **Simulación de Rendimiento por Molde (Pin & Die)**  
   Algoritmo de optimización que calcula la eficiencia de cada molde (pin/die) en función del espesor de pared deseado y la velocidad de la máquina. Incluye un dashboard de desgaste de moldes basado en ciclos acumulados y torque detectado.

3. **API de Control de Calidad en Vivo**  
   Endpoints RESTful que reciben datos de sensores IoT (balanzas, medidores de humedad, cámaras de visión artificial) y los comparan con especificaciones de lote. Si se detecta una deriva, el sistema puede pausar la línea o ajustar parámetros de forma autónoma (modo semiautomático).

### Cómo Usar

1. **Configurar el Perfil de Producción y Parámetros Críticos**  
   Accede al módulo `Settings > Production Profiles`. Define el tipo de cápsula (gelatina dura, blanda, HPMC), el modelo de máquina (Bosch, Qualicaps, ACG) y los límites de calidad (peso promedio, desviación estándar, humedad residual). Guarda el perfil como JSON o YAML para versionarlo en tu CI/CD.

2. **Ejecutar una Orden de Fabricación con Trazabilidad Completa**  
   Ve a `Production > New Order`. Ingresa el número de lote de gelatina, el código de molde y el target de producción (ej. 500,000 cápsulas). El sistema calculará automáticamente el tiempo estimado de secado y generará un QR por pallet. Puedes monitorear en vivo el progreso desde el dashboard WebSocket.

3. **Auditar y Exportar Reportes Regulatorios**  
   Desde `Reports > Compliance`, selecciona el rango de fechas y las máquinas involucradas. Genera un informe en PDF que incluye: gráficos de control estadístico (X-bar, R-chart), lista de no conformidades y firma digital del operador. Este reporte cumple con los requisitos de la FDA 21 CFR Part 11.

### Mejores Prácticas

- **Implementar un Pipeline de Datos IoT con Buffer Local**  
  No envíes datos de sensores directamente a la nube. Configura un edge gateway (Raspberry Pi o PLC) que almacene en buffer los últimos 10 minutos de lecturas. Esto evita pérdidas de datos durante caídas de red y permite sincronización asíncrona con el SaaS. Recomendamos usar MQTT con QoS 2.

- **Versionar los Modelos de Optimización de Moldes**  
  Cada vez que ajustes los parámetros del algoritmo de simulación (por ejemplo, cambiar el coeficiente de fricción del pin), guarda el modelo como un artefacto en tu repositorio (Git LFS). Así, si una actualización degrada el rendimiento, puedes hacer rollback sin afectar la producción en vivo. Usa etiquetas semánticas como `molde-optimizacion-v2.1.0`.
