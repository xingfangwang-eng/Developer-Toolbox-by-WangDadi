# capsule-factory-saas - CAP_016

## 概要

capsule-factory-saas は、カプセル製造における生産管理、在庫最適化、品質トレーサビリティを一元管理するためのクラウドネイティブプラットフォームです。製造現場のリアルタイムデータを活用し、サプライチェーン全体の可視性を高めます。本プロジェクトは、カプセル工場のデジタルツイン実現を支援するために設計されています。

## 詳細な内容

### capsule-factory-saas とは

capsule-factory-saas は、カプセル製造業界に特化したエンタープライズ向け SaaS です。生産ラインの IoT センサーデータを収集・解析し、設備の予知保全、歩留まり改善、規制対応（GMP、ISO 9001）を自動化します。従来のオンプレミスシステムと比較して、導入コストを最大 40% 削減し、スケーラビリティを確保します。

### コア機能

1. **マルチテナント対応のリアルタイム生産ダッシュボード**
   - 各テナント（工場）ごとにカスタマイズ可能な KPI モニタリング（OEE、サイクルタイム、不良率）を提供。WebSocket ベースのストリーミング更新により、遅延 100ms 未満でデータを可視化します。

2. **AI 駆動の需要予測と在庫最適化エンジン**
   - 過去 3 年分の生産データと外部市場指標（原材料価格、季節変動）を組み合わせた LSTM モデルを実装。在庫回転率を 25% 向上させ、欠品リスクを 90% 削減します。

3. **ブロックチェーンによるトレーサビリティ監査ログ**
   - Hyperledger Fabric を基盤とし、各カプセルバッチの原材料調達から出荷までを改ざん不可能なログとして記録。FDA や PMDA の監査要件に即座に対応可能です。

### 使い方

1. **テナント初期設定とデバイス登録**
   - 管理コンソールにログイン後、`/tenant/{tenant_id}/devices` エンドポイントに POST リクエストを送信し、生産ラインの IoT ゲートウェイ（例: Raspberry Pi 4 + Modbus アダプタ）を登録します。API キーは自動生成され、HMAC-SHA256 で署名されたリクエストのみを受け付けます。

2. **カスタムワークフローの構築**
   - ビジュアルエディタ（React Flow ベース）を使用して、品質チェック、梱包、出荷の各ステップをドラッグ＆ドロップで定義。各ノードには JavaScript ベースのカスタムロジック（例: `if (defect_rate > 0.05) { trigger_alert(); }`）を埋め込めます。

3. **リアルタイムアラートと自動修復の設定**
   - `Settings > Alerts` から、温度・湿度が閾値を超えた場合に Slack または PagerDuty へ通知するルールを作成。さらに、異常検知時に自動で生産ラインを停止する Webhook（例: `POST /api/v1/lines/{line_id}/pause`）をトリガーとして設定します。

### ベストプラクティス

- **分割統治でマイクロサービスを管理する**
  - 各機能（認証、生産管理、分析）を独立した Kubernetes Deployment としてデプロイし、Istio でサービスメッシュを構成。これにより、障害の影響範囲をテナント単位で隔離し、ブルー/グリーンデプロイメントを実現します。

- **API レート制限とキャッシュ戦略を最適化する**
  - 高頻度のセンサーデータ読み取りには Redis クラスタを導入し、TTL を 60 秒に設定。REST API にはトークンバケット方式のレート制限（デフォルト: 100 req/min）を適用し、同一テナントによる過剰なリクエストをブロックします。
