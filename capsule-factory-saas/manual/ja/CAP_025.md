# capsule-factory-saas - CAP_025

## 概要

このプロジェクトは、カプセル製造工場向けに特化したSaaS型生産管理プラットフォームです。GMP準拠の製造ラインからリアルタイム在庫管理、トレーサビリティまでをワンストップで実現。カプセル製造業界における歩留まり向上と品質管理の自動化を主目的としています。

## 詳細な内容

### capsule-factory-saas とは

capsule-factory-saas は、カプセル製造工場の生産管理プロセスをデジタルトランスフォーメーションするためのクラウドネイティブなSaaSツールです。従来のExcelやオンプレミスシステムでは追跡が困難だった、原料ロットから充填工程、最終出荷までの全工程をAPIベースで連携。特に、カプセルサイズごとの生産計画最適化とリアルタイム品質データ分析に強みを持ちます。

### コア機能

1. **マルチライン生産スケジューリングエンジン**
   複数のカプセル充填ライン（例：サイズ0、サイズ1、ソフトジェル）を同時管理。機械の稼働率、メンテナンス予定、原料供給状況を考慮した自動スケジューリングを実行。GMPのバリデーション要件を満たすロット追跡も内蔵。

2. **リアルタイム品質異常検知ダッシュボード**
   各充填機から吐出されるカプセル重量、シーム強度、溶解試験データをIoTセンサー経由で収集。統計的プロセス制御（SPC）に基づき、異常値を検出した瞬間にアラートを発報。ヒューマンエラーを排除した品質管理を実現。

3. **原料ロットトレーサビリティモジュール**
   QRコードまたはRFIDタグで管理された原料（ゼラチン、グリセリン、有効成分）の受け入れから使用完了までをブロックチェーンライクな改ざん防止ログで記録。万が一のリコール時には、該当ロットを使用した全製品を30秒以内に特定可能。

### 使い方

1. **生産ラインの初期セットアップとマスタデータ登録**
   まず、管理対象の充填ラインごとに「ラインID」「最大生産速度（capsules/min）」「対応カプセルサイズ」をJSON形式でAPI経由または管理画面から登録します。次に、各原料の「ロット番号」「有効期限」「保管場所（冷蔵/常温）」を入力。この段階でGMP準拠のバリデーション設定も同時に行います。

   ```json
   {
     "line_id": "LINE-02",
     "capsule_size": "size_1",
     "max_speed": 1200,
     "assigned_material_lots": ["GEL-20250301", "API-20250410"]
   }
   ```

2. **生産計画の作成と自動最適化**
   ダッシュボードの「生産計画」タブから、対象期間（例：2025年4月1日～4月7日）と目標生産量（例：500,000カプセル）を設定します。システムは登録済みラインの稼働カレンダー、メンテナンス予定、原料在庫を自動でクロスチェック。最適なライン割り当てと開始時刻を提案します。提案内容を確認し、必要に応じて手動で調整後、「実行」ボタンで全ラインに一斉展開。

3. **リアルタイムモニタリングと異常対応**
   生産開始後は、リアルタイムダッシュボードで各ラインの「現在の生産速度」「累積生産数」「不良率（%）」を監視。例えば、充填機のシーム強度が閾値（例：2.5N）を下回った場合、画面上に赤色アラートと共に「推奨アクション：ノズル交換」がポップアップ表示されます。オペレーターはその場で交換作業を記録し、システムは自動でその時間帯の生産ロットに注釈を追加。全工程が監査証跡として保存されます。

### ベストプラクティス

- **原料ロットの事前分割登録**
   大量の原料ロットを一括で登録するのではなく、使用予定の製造バッチごとに分割して登録することを推奨します。例えば、1トンのゼラチンロットを10バッチ分に分割登録することで、万が一の品質問題発生時に影響範囲を最小限に抑えられます。システム上では「親ロット」「子ロット」の階層管理が可能です。

- **週次スケジュールレビューの自動化**
   毎週月曜日の午前8時に、次週の生産計画を自動生成し、チーム全体にSlack/Teams通知する設定を組み込みましょう。これにより、原料調達部門と生産管理部門のコミュニケーションロスを防止。特にカプセル製造では、ゼラチンの溶解準備にリードタイムがかかるため、48時間前の原料確保が重要です。
