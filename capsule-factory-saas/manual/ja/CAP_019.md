# capsule-factory-saas - CAP_019

## 概要

capsule-factory-saas は、カプセル製造工場における生産管理、品質トレーサビリティ、設備連携を一元化するクラウドネイティブ SaaS です。多言語対応の API ファースト設計により、グローバルなサプライチェーンとシームレスに統合可能で、リアルタイムな生産最適化を実現します。

## 詳細な内容

### capsule-factory-saas とは

capsule-factory-saas は、カプセル充填・包装ラインの IoT データを収集し、機械学習ベースの異常検知と生産スケジューリングを提供するプロフェッショナルツールです。GMP や FDA 21 CFR Part 11 に準拠した監査証跡機能を内蔵し、製薬・サプリメント業界の厳格なコンプライアンス要件を満たします。

### コア機能

- **マルチテナント対応のリアルタイム生産ダッシュボード**: テナントごとに異なる生産ラインの OEE（設備総合効率）を秒単位で可視化。カスタム KPI アラートと異常値検出アルゴリズムにより、ダウンタイムを最小化します。
- **充填ノズル単位のトレーサビリティエンジン**: 各カプセルに紐づくバッチ、原料ロット、充填圧力、温度プロファイルをブロックチェーン方式で記録。不適合発生時には、影響範囲を秒単位で特定し、リコールコストを劇的に削減します。
- **動的スケジューリング最適化エンジン**: 受注変動、設備メンテナンス予定、原料在庫を考慮した制約充足問題（CSP）を、GPU アクセラレーションで 5 秒以内に解きます。納期遵守率 99% 以上を実現。

### 使い方

1. **IoT デバイス登録とデータパイプライン設定**  
   MQTT または OPC UA 経由で充填機、乾燥機、包装機のセンサーを接続します。`/api/v1/devices/register` エンドポイントにデバイス証明書とメタデータ（最大 50 フィールド）を POST し、自動的に Kafka トピックが生成されます。データスキーマは Avro 形式で管理され、スキーマレジストリによるバージョン管理が可能です。

2. **カスタム品質ルールの定義とバリデーション**  
   Web UI のルールエディタで、充填重量 ±0.5% 以内、乾燥温度 45°C ±2°C などの条件をドラッグ＆ドロップで設定します。ルールは JSON 形式でエクスポートでき、CI/CD パイプラインに組み込んで自動テストが実行可能。不適合データは即座に `reject_topic` にパブリッシュされ、後続のプロセスがブロックされます。

3. **生産計画の自動生成と調整**  
   「週次生産計画」画面で、顧客オーダー CSV をアップロードするか、ERP 連携 API から取得。最適化エンジンが制約条件（機械稼働時間、金型交換時間、原料リードタイム）を解析し、最適な充填順序と設備割り当てを Gantt チャートで表示します。調整が必要な場合は、ドラッグ操作で手動介入可能で、変更履歴はすべて監査ログに記録されます。

### ベストプラクティス

- **データパイプラインの冗長化とレイテンシ管理**  
  IoT データは少なくとも 2 つの Kafka ブローカにレプリケーションし、コンシューマグループを分離してダッシュボード用とバッチ分析用のトピックを分けます。レイテンシが 500ms を超える場合は、gRPC ストリーミングに切り替えることを推奨。これにより、リアルタイム性と分析の負荷分散を両立できます。

- **マルチテナント環境でのリソース分離とコスト最適化**  
  各テナントに専用の Kubernetes namespace とリソースクォータを割り当て、CPU/メモリのバースト制限を設定します。月次でテナントごとの API 呼び出し数とデータ保存量を分析し、アイドルテナントのインスタンスを自動スケールダウン。余剰リソースはスポットインスタンスで賄い、クラウドコストを最大 40% 削減できます。
