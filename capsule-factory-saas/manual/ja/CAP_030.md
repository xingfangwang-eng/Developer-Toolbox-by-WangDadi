# capsule-factory-saas - CAP_030

## 概要

capsule-factory-saas は、カプセル製造業向けに特化したクラウドネイティブな生産管理プラットフォームです。GMP 準拠のバッチトレーサビリティ、リアルタイム生産ダッシュボード、サプライチェーン連携を一貫して提供し、小ロット多品種生産から大規模連続生産までをシームレスにカバーします。本プロジェクトは、カプセル工場のデジタルトランスフォーメーションを加速するためのオープンソース SaaS 基盤です。

## 詳細な内容

### capsule-factory-saas とは？

capsule-factory-saas は、カプセル製造プロセスにおける原料受入から充填、乾燥、検査、出荷に至る全工程を一元管理するためのプロフェッショナルツールです。特に、製薬・サプリメント業界で必要とされる厳格な品質管理とトレーサビリティ要件を満たす設計が特徴で、従来のオンプレミスシステムに代わるモダンな SaaS ソリューションとして開発されています。

### コア機能

- **バッチトレーサビリティエンジン**: 各カプセルロットに対して、原料ロット、製造条件（温度・湿度・圧力）、検査結果、包装日時をブロックチェーンライクな不変台帳に記録。FDA や PMDA の監査要件を満たす完全なデジタル監査証跡を自動生成します。
- **リアルタイム生産ダッシュボード**: WebSocket ベースで工場内の各ラインの稼働率、不良率、生産速度を 1 秒単位で可視化。異常値が発生した場合、Slack やメールに加えて、LINE Notify や Microsoft Teams へのアラート配信も可能です。
- **サプライチェーン連携 API**: 原料サプライヤーや受託製造業者と直接データ連携できる RESTful API と GraphQL エンドポイントを標準装備。EDI や CSV バッチ処理に依存せず、リアルタイム在庫調整と需要予測を実現します。

### 使い方

1. **バッチレコードの作成と設定**: まず、管理画面から新規バッチを作成し、原料ロット番号、目標生産数、機械パラメータ（充填速度、乾燥時間）を入力します。システムは自動でバッチ ID を発行し、QR コードを生成します。この QR コードを各工程の端末でスキャンすることで、すべての工程データが紐付けられます。
2. **リアルタイムモニタリングとアラート設定**: ダッシュボード画面で、各ラインの稼働状況を確認しながら、不良率が 2% を超えた場合に自動停止するトリガーを設定します。設定は YAML 形式でエクスポート可能で、CI/CD パイプラインに組み込んでバージョン管理できます。
3. **監査ログのエクスポートと分析**: 任意の期間を指定して、全バッチのトレーサビリティデータを JSON または CSV でエクスポートします。エクスポートされたデータは、内蔵の差分分析ツールを使って、同じ原料ロットで発生した不良率の比較や、機械別の性能推移をグラフ化できます。

### ベストプラクティス

- **マルチテナント分離の徹底**: 各工場や事業部ごとにデータベーススキーマを分離するのではなく、PostgreSQL の Row-Level Security を活用してテナント間のデータ隔離を実現してください。これにより、運用コストを抑えながらも、監査要件を満たす厳格なアクセス制御が可能になります。
- **CI/CD パイプラインへの組み込み**: capsule-factory-saas の API を活用して、製造条件の変更（例：充填速度の調整）を Git リポジトリで管理し、プルリクエストベースで本番環境にデプロイするワークフローを構築してください。これにより、変更履歴の完全な追跡と、承認プロセスの自動化が実現します。
