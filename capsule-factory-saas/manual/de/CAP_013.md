# capsule-factory-saas - CAP_013

## Übersicht

Die **capsule-factory-saas** ist eine hochperformante, cloud-native Plattform zur digitalen Steuerung und Optimierung von Kapselproduktionslinien. Sie kombiniert Echtzeit-MES-Funktionalität mit KI-gestützter Qualitätssicherung und ermöglicht eine nahtlose Integration in bestehende ERP- und IoT-Infrastrukturen. Entwickelt für mittelständische und industrielle Fertigungsbetriebe, die ihre Kapselproduktion auf das nächste Automatisierungsniveau heben wollen.

## Hardcore-Features

1. **Multi-Tenant-Rezeptur-Engine mit dynamischer Batch-Optimierung**  
   Verwaltung und Versionierung hunderter Rezepturen (Gelatine, HPMC, Pullulan) inkl. automatischer Anpassung von Temperatur-, Feuchte- und Druckprofilen basierend auf Echtzeit-Sensordaten. Die Engine nutzt einen genetischen Algorithmus zur Minimierung von Ausschuss und Zykluszeit.

2. **Inline-Qualitätskontrolle via Computer-Vision & Spektroskopie**  
   Direkte Anbindung an Hyperspektralkameras und Mikroskop-Module zur zerstörungsfreien Prüfung von Wandstärke, Füllvolumen und Siegelintegrität. Jede Charge erhält einen digitalen Zwilling mit lückenloser Rückverfolgbarkeit (DIN EN ISO 15378-konform).

3. **Edge-basierte Predictive Maintenance für Füll- & Siegelstationen**  
   Automatische Erfassung von Vibration, Drehmoment und Temperatur an kritischen Maschinenkomponenten. Ein trainiertes LSTM-Modell prognostiziert Ausfälle bis zu 72 Stunden im Voraus und generiert Wartungsaufträge direkt im angeschlossenen CMMS.

## Detaillierte Anwendungsschritte

### Schritt 1: Rezeptur-Import und Parameter-Kalibrierung
Öffnen Sie das Dashboard unter `Rezepturen > Import`. Laden Sie eine bestehende Rezeptur als CSV/XML oder nutzen Sie die REST-API zur Anbindung an Ihr PLM-System. Wählen Sie anschließend im Reiter „Prozessparameter“ die Zielmaschine aus. Das System gleicht automatisch die historischen Soll-Ist-Abweichungen ab und schlägt optimierte Startwerte für Fülltemperatur und Siegelkraft vor. Bestätigen Sie die Kalibrierung mit einem Klick auf „Prozess freigeben“.

### Schritt 2: Live-Überwachung und Qualitäts-Dashboard
Navigieren Sie zu `Produktion > Live-Monitor`. Hier sehen Sie eine Echtzeit-Tabelle mit Maschinen-ID, aktueller Zykluszeit, Ausschussrate und dem KI-Konfidenzwert der Inline-Prüfung. Klicken Sie auf eine beliebige Charge, um den digitalen Zwilling mit allen Messwerten (z. B. Wandstärke in µm, Siegelnahtbreite) aufzurufen. Bei Überschreitung eines Toleranzgrenzwerts löst das System automatisch eine Alarmmeldung und eine Pausen-Anforderung aus.

### Schritt 3: Predictive Maintenance planen und auslösen
Gehen Sie zu `Wartung > Predictive Insights`. Das Dashboard zeigt eine priorisierte Liste von Komponenten mit prognostizierter Restlebensdauer (z. B. „Fülldüse #3: 48h Restlaufzeit“). Wählen Sie eine Komponente aus und klicken Sie auf „Wartungsauftrag generieren“. Das System erstellt automatisch einen Ticket-Eintrag in Ihrem CMMS, inkl. Ersatzteilnummer und empfohlenem Zeitfenster. Bestätigen Sie die Ausführung, um die Maschine für den nächsten Zyklus freizugeben.

## Best Practices für die Industrie

1. **Führen Sie eine initiale „Cold-Start“-Kalibrierung für jede neue Maschine durch**  
   Auch wenn die Plattform mit Standardwerten arbeitet, sollten Sie für jede neue Füll- oder Siegelstation mindestens 50 Zyklen mit einem Referenzmaterial durchlaufen lassen. Das KI-Modell lernt so die individuellen Maschinencharakteristiken und reduziert die Fehlerquote in der ersten Produktionswoche um bis zu 40 %.

2. **Nutzen Sie die API-first-Architektur für eine durchgängige Digitalisierung**  
   Verzichten Sie auf manuelle Dateneingaben. Binden Sie die capsule-factory-saas über die bereitgestellten REST- und MQTT-Endpunkte direkt an Ihr ERP (z. B. SAP S/4HANA) und Ihr MES an. Nur so können Sie eine echte Closed-Loop-Qualitätsregelung realisieren – von der Rohstoffanlieferung bis zur Versandfreigabe.
