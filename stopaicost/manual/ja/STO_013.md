### 3 步安装指南

**Step 1: 依存関係の設定**

```
npm install --save express redis
```

**Step 2: Stopaicost の設定**

```
const express = require('express');
const Redis = require('ioredis');

const app = express();
app.use(express.json());

// Redis との接続
const redis = new Redis({
  host: 'localhost',
  port: 6379,
});

app.get('/api/data', (req, res) => {
  //Redis からデータを取得
  const data = redis.get('data');
  res.send(data);
});
```

**Step 3: Stopaicost の起動**

```
node index.js
```
