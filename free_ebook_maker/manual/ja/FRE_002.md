**Installation Guide**
-------------------------

### Step 1: Install Redis

To install Redis, follow these steps:

1. Download the Redis binary from the official website: <https://redis.io/download>
2. Unzip the downloaded file and move it to a directory of your choice.
3. Create a `redis.conf` file with the following content:
```bash
port 6379
```
4. Start Redis by running the command: `redis-server redis.conf`

### Step 2: Install Python Libraries

To install the required Python libraries, follow these steps:

### Step 3: Configure PostgreSQL

To configure PostgreSQL, follow these steps:

1. Install PostgreSQL on your system.
2. Create a new database and set up the required credentials.
3. Update the `pg_config` file with the following content:
```bash
host = 'localhost'
port = 5432
username = 'your_username'
password = 'your_password'
```
4. Start the PostgreSQL server by running the command: `pg_ctl start`
