## Installation Guide

### Step 1: Install Redis

```bash
sudo apt-get install redis-server
```

### Step 2: Configure Java 8 and Hazelcast

```java
// Create a Hazelcast configuration instance
Config config = new Config();
config.setNetworkConfig(new NetworkConfig().setPort(5701));

// Start the Hazelcast node
HazelcastInstance instance = Hazelcast.newHazelcastInstance(config);
```

### Step 3: Set up Nginx Load Balancing

```nginx
http {
    upstream boothell_nodes {
        server localhost:8080;
        server localhost:8081;
    }

    server {
        listen 80;
        location / {
            proxy_pass http://boothell_nodes;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
        }
    }
}
```
