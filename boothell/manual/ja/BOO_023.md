# boothell - BOO_023

## 概要

boothellは、ブートデザインとカスタマイズに特化したプロフェッショナルツールです。SEO最適化されたコンテンツ生成、ブランド管理、デプロイ自動化を一元化し、開発者のワークフローを極限まで効率化します。

## 詳細な内容

### boothellとは？

boothellは、ブートストラップベースのプロジェクトにおける**デザイン・カスタマイズ・ブランド管理**を、コードベースで完全に制御できるSaaS型ツールです。テーマの動的生成、アセットの最適化、SEOメタデータの自動挿入をリアルタイムで実行し、開発者が手動で行っていた煩雑な作業を排除します。特に、複数ブランドやマルチテナント環境での一貫性維持に強みを持ちます。

### コア機能

1. **動的テーマエンジン**  
   ブートストラップ変数（`$primary`, `$font-size-base` など）をYAML設定ファイルからリアルタイムに上書きし、ビルド時に最適化されたCSSを生成。テーマの差分適用やA/Bテストも可能。

2. **SEOメタデータ自動挿入**  
   ページごとに指定したキーワード（例: `boot design tool`, `boot customization`）を基に、`<title>`、`<meta name="description">`、Open Graphタグを自動生成。hreflangタグや構造化データ（JSON-LD）もサポート。

3. **ブランドアセット管理API**  
   ロゴ、カラーパレット、フォントセットを一元管理し、REST API経由で外部サービス（CMS、マーケティングツール）と連携。変更は即座に全チャネルに反映され、ブランドの一貫性を担保します。

### 使い方

1. **テーマ設定ファイルの作成**  
   プロジェクトルートに `boothell.yml` を作成し、ブートストラップ変数を上書きします。例:
   ```yaml
   theme:
     primary: '#1a73e8'
     font-family: 'Inter, sans-serif'
     border-radius: '0.5rem'
   seo:
     og-image: './assets/og-default.png'
   ```
   ファイルを保存後、`boothell build` を実行すると、最適化されたCSSとメタタグが自動生成されます。

2. **ブランドアセットの登録**  
   `boothell brand:add` コマンドでロゴやカラーパレットを登録します。
   ```bash
   boothell brand:add --name "Primary Logo" --file ./assets/logo.svg --type logo
   boothell brand:add --name "Accent Color" --color '#ff5722' --type color
   ```
   登録後、`boothell brand:list` で確認可能。APIエンドポイントは `https://api.boothell.dev/v1/brands/{project_id}` で公開されます。

3. **SEOメタデータの検証とデプロイ**  
   生成されたメタデータを確認するには、`boothell seo:check` を実行。検証が通ったら、`boothell deploy` で本番環境にプッシュします。
   ```bash
   boothell seo:check --page /products/boots
   boothell deploy --env production
   ```
   デプロイ後は、Google Search Consoleでインデックス状況を確認することを推奨します。

### ベストプラクティス

- **テーマのバージョン管理**  
  boothellの設定ファイル（`boothell.yml`）はGitで管理し、ブランチごとに異なるテーマを適用することで、ステージング環境でのA/Bテストが容易になります。マージ時に自動でCSSを再生成するCI/CDパイプラインを組むと効果的です。

- **キーワードの階層化**  
  SEOメタデータのキーワードは、ページごとにプライマリ（例: `boot customization`）とセカンダリ（例: `boot brand management`）を設定し、`boothell.yml` の `seo.keywords` にリスト形式で記述します。これにより、検索エンジンにページの主題が明確に伝わり、ランキング向上が期待できます。
