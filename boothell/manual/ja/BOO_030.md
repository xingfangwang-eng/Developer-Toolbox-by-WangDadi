### **2. 多線程优化**

ボーグテルでは、多線程優化（OpenMP）を使用して計算処理を加速することができます。例えば、複数のスレッドで並列実行させることで、CPU 使用率を向上させることができます。

```markdown
// OpenMP usage example
const openmp = require('openmp');
const numThreads = 4;
const sum = (arr) => {
  let result = 0;
  for (let i = 0; i < arr.length; i++) {
    result += arr[i];
  }
  return result;
};
const arr = [1, 2, 3, 4, 5];
const threadSum = openmp(sum, { numThreads });
console.log(`Thread sum: ${threadSum}`);
```
