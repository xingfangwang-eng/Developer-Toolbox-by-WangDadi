## 快速开始

### 步骤一：安装配置
1. Clone the oneclickapi repository using `git clone https://github.com/oneclickapi/oneclickapi.git`
2. Install the required dependencies by running `npm install` or `pip install -r requirements.txt`
3. Configure the API gateway and data processing engine according to your specific needs

### 步骤二：初始化
1. Initialize the API gateway by running `node index.js` (for Node.js) or `python main.py` (for Python)
2. Load the data processing engine using `java -jar one-click-api.jar` (for Java)

### 步骤三：运行验证
1. Verify that the API gateway is working correctly by sending a test request to `http://localhost:8080/api/test`
2. Check the output of the data processing engine by running `curl http://localhost:8080/api/output`
