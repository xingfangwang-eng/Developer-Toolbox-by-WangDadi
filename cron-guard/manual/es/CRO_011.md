**Instalación paso a paso**

1. Primero, instala el paquete `cron-guard` utilizando npm o yarn:
```bash
npm install cron-guard
```
2. Luego, crea un archivo de configuración denominado `cron-guard.config.js` con el siguiente contenido:
```javascript
module.exports = {
  apiEndpoint: 'https://api.cron-guard.com',
  token: 'YOUR_API_TOKEN'
};
```
3. Finalmente, carga el módulo y inicializa Cron-Guard utilizando el siguiente código:
```javascript
const cronGuard = require('cron-guard');
cronGuard.init();
```
**Recomendaciones de optimización para escenarios de alta carga**

1. **Ajustar la configuración de cache**: Asegúrate de ajustar la configuración de caché según sea necesario para tu escenario, ya que una cantidad excesiva de datos en memoria puede afectar el rendimiento.
* Valor recomendado: `cache.size = 1000000` (10 millones de elementos)

2. **Ajustar la configuración del circuit breaker**: Asegúrate de ajustar la configuración del circuit breaker según sea necesario para tu escenario, ya que un tiempo de espera demasiado bajo puede provocar una sobrecarga.
* Valor recomendado: `circuitBreaker.waitTime = 5000` (5 segundos)
