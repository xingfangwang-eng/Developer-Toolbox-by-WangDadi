### 2. 3 步代码级安装指南

#### Step 1: 安装 Redis

```bash
sudo apt-get install redis-server
```

#### Step 2: 配置 Redis

```properties
redis.conf:
port 6379
bind 127.0.0.1
```

#### Step 3: 安装 smesurvivalai

```bash
git clone https://github.com/smesurvivalai/smesurvivalai.git
cd smesurvivalai
mvn clean package
```

### 3. 高并发场景的性能调优建议

1. 使用熔断器设计来避免系统 overload。

2. 配置 Redis 的最大连接数和超时时间，以提高性能。

```properties
redis.conf:
maxclients 1000
timeout 30000
```
