### 性能调优建议

1. **调整 Redis 缓存大小**

根据您的需求调整 Redis 缓存大小。默认情况下，Redis 缓存大小为 100MB。如果您需要更大的缓存，可以使用以下命令调整：
```bash
redis-cli config set maxmemory 1024mb
```
2. **调整 MySQL 连接池大小**

根据您的需求调整 MySQL 连接池大小。默认情况下，MySQL 连接池大小为 10。如果您需要更多的连接，可以使用以下命令调整：
```sql
SET GLOBAL max_connections = 100;
```
