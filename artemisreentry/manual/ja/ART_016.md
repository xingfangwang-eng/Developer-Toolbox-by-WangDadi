# artemisreentry - ART_016

## 概要

本プロジェクトは、Artemis 再突入ミッションに特化した解析ツールキットです。キーワード「Artemis reentry」「Space reentry tool」「Space return analysis」に最適化されたSEOコンテンツを提供し、エンジニアや研究者のワークフローを効率化します。

## 詳細な内容

### artemisreentry とは？

artemisreentry は、宇宙機の再突入軌道計算、熱負荷解析、および帰還タイムライン最適化を実現するプロフェッショナルツールです。NASA の Artemis プログラムに準拠したパラメータセットを標準搭載し、カスタムミッションにも対応可能です。

### コア機能

1. **動的再突入軌道シミュレーター**  
   大気圏再突入時の抗力・重力・熱流束をリアルタイムで計算。3自由度（3-DOF）モデルと6自由度（6-DOF）モデルを切り替え可能で、パラシュート展開タイミングやスキップ軌道の最適化を自動化します。

2. **熱防護システム（TPS）健全性診断**  
   TPS素材（PICA-X、SLA-561Vなど）の熱劣化モデルを内蔵。再突入中の表面温度分布をFEMベースで推定し、アブレーション率や構造的リスクをスコアリングします。

3. **帰還ウィンドウ最適化エンジン**  
   月軌道からの帰還タイミングを、地球の大気状態・太陽活動・通信遅延を考慮して自動調整。複数の再突入コリドー（L/D比、進入角）を比較し、最適な軌道を提示します。

### 使い方

1. **ミッションパラメータの設定**  
   `config.json` に初期軌道要素（近地点高度、離心率、軌道傾斜角）と機体特性（質量、断面積、抗力係数）を記述。例：
   ```json
   {
     "altitude": 120000,
     "velocity": 7800,
     "mass": 25000,
     "cd": 1.2,
     "tps_material": "PICA-X"
   }
   ```
   その後、`artemisreentry --config config.json` で解析を開始。

2. **再突入シミュレーションの実行**  
   `--mode 6dof` フラグを追加し、`--output heatmap` で熱分布データをCSV出力。実行中は `ctrl+c` で一時停止、`--resume` で再開可能。結果は `reentry_log_*.json` に保存。

3. **結果の可視化とレポート生成**  
   `artemisreentry --plot trajectory --output report.pdf` で軌道プロファイルと温度曲線を重ねたPDFレポートを生成。オプションで `--compare` を指定すると、過去のシミュレーション結果との差分を強調表示。

### ベストプラクティス

- **大気モデルの選択**  
  日本国内のユーザーは、JAXA の標準大気モデル（JSM）を `--atmosphere jsm` で指定すると、局所的な気象予測に基づく高精度計算が可能。米国ユーザーには `--atmosphere us76` を推奨。

- **バッチ処理によるパラメータスイープ**  
  `artemisreentry --sweep velocity:7600-8000:50,angle:-6:-4:0.5` で速度と進入角を自動スイープ。結果は `sweep_results.csv` に集約され、最適パラメータがハイライト表示されます。
