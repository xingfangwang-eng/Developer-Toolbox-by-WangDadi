# artemisreentry - ART_011

## Resumen

Herramienta de análisis de reentrada atmosférica para misiones Artemis, diseñada para simular trayectorias de retorno con precisión suborbital y optimizar la ventana de aterrizaje. Integra modelos de resistencia aerodinámica y propagación de perturbaciones gravitacionales en tiempo real.

## Funcionalidades Clave

### 1. Simulación de Corredor de Reentrada Dinámica
Calcula el corredor de reentrada seguro en función de la velocidad de entrada, el ángulo de inclinación y la densidad atmosférica variable. Ajusta automáticamente trayectorias de emergencia si se detectan desviaciones superiores al 0.5% respecto al perfil nominal.

### 2. Análisis de Térmica de Escudo Térmico
Modela la transferencia de calor en la capa límite del escudo térmico utilizando ecuaciones de Navier-Stokes reducidas. Genera mapas de temperatura superficial con resolución de 1 mm para identificar puntos críticos de ablación.

### 3. Optimización de Ventana de Aterrizaje con Algoritmos Genéticos
Utiliza un algoritmo genético multiobjetivo para encontrar la combinación óptima de fecha/hora de reentrada y punto de impacto, minimizando el consumo de combustible y maximizando la precisión de aterrizaje (error < 50 m).

## Cómo Usar

### 1. Configurar Perfil de Misión
Ejecuta el módulo de inicialización con parámetros de la cápsula Orion:
```bash
python artemisreentry.py --capsule orion --mass 10500 --drag-coeff 1.2 --entry-speed 11.2
```
Esto carga los coeficientes aerodinámicos precalculados y la base de datos de densidad atmosférica (modelo NRLMSISE-00).

### 2. Ejecutar Simulación Monte Carlo
Para evaluar la robustez del perfil de reentrada ante incertidumbres:
```python
from artemisreentry import MonteCarloSim
sim = MonteCarloSim(iterations=10000, variables=['entry_angle', 'wind_speed', 'atmospheric_density'])
results = sim.run()
print(f"Probabilidad de éxito: {results.success_rate:.2%}")
```
El resultado incluye histogramas de desviación y mapas de calor de sensibilidad.

### 3. Generar Informe de Misión
Exporta los resultados en formato PDF o JSON con el comando:
```bash
artemisreentry report --format pdf --output artemis_reentry_report.pdf --include-thermal-maps
```
El informe incluye gráficos de evolución temporal de altitud, velocidad y temperatura del escudo.

## Mejores Prácticas

### 1. Validar con Datos de Misiones Anteriores
Antes de usar los resultados para planificación real, realiza una validación cruzada con telemetría de misiones Apollo o SpaceX Dragon. Ajusta los parámetros de fricción atmosférica si la desviación supera el 2%.

### 2. Usar Modo de Alta Precisión para Ventanas Críticas
Cuando la ventana de aterrizaje sea estrecha (menos de 30 minutos), activa el modo `--high-precision` que reduce el paso de integración temporal a 0.1 segundos y utiliza un modelo de atmósfera de capas múltiples con datos en tiempo real del NOAA.
