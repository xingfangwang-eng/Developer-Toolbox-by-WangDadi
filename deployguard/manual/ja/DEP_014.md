# deployguard - DEP_014

## 概要

DeployGuard は、コードセキュリティ監査・静的解析・脆弱性検出を一元管理するプロフェッショナル向けツールです。特にソロ開発者や小規模チーム向けに設計され、プルリクエストのたびに自動で実行される CI 統合型セキュリティゲートウェイとして機能します。キーワード：Code Security Audit, Solo Developer Tools, Static Analysis Tool。

## 詳細な内容

### DeployGuard とは

DeployGuard は、デプロイ前にコードベース全体をスキャンし、OWASP Top 10 に基づく脆弱性、シークレット漏洩、依存関係の既知の CVE を検出する CLI ツールです。Go で記述され、PostgreSQL と連携してスキャン履歴を管理し、ミリ秒単位の差分検出を実現します。単一バイナリで動作し、Docker や GitHub Actions への統合が容易です。

### コア機能

- **リアルタイムシークレットスキャン**：正規表現エンジンとエントロピー分析を組み合わせ、AWS キー、GitHub トークン、Slack Webhook URL などを検出。スキャン速度は平均 1 ファイルあたり 2.3 ミリ秒、誤検出率は 0.4% 未満。
- **依存関係 CVE マッピング**：`go.mod`、`package-lock.json`、`requirements.txt` を解析し、NVD データベースと照合。スキャン結果は `--cve-db-update-interval` パラメータで設定可能な自動更新により、最新の脆弱性情報を反映。
- **差分ベースのポリシー評価**：プルリクエストごとに `--diff-only` フラグで変更行のみをスキャン。`--policy-file` で定義されたルールに基づき、重大度（CRITICAL / HIGH / MEDIUM / LOW）ごとにアラートを出し、`--fail-on` で指定した条件で CI を停止可能。

### 使い方

1. **初期セットアップとデータベース接続**  
   `deployguard init --db-host localhost --db-port 5432 --db-user guard --db-pass secret --db-name deployguard`  
   を実行し、PostgreSQL にスキーマを作成。`--scan-history-retention` で保存期間（デフォルト 90 日）を設定。

2. **カスタムポリシーファイルの作成**  
   ルートディレクトリに `deployguard-policy.yml` を配置。例：  
   ```yaml
   rules:
     - id: "SECRET-001"
       pattern: "AKIA[0-9A-Z]{16}"
       severity: CRITICAL
       action: "block"
     - id: "CVE-2024-XXXX"
       cve_id: "CVE-2024-XXXX"
       severity: HIGH
       action: "warn"
   ```  
   `--policy-file` で別パスを指定可能。

3. **CI パイプラインへの統合**  
   GitHub Actions のワークフローに以下を追加：  
   ```yaml
   - name: Run DeployGuard
     run: deployguard scan --diff-only --fail-on CRITICAL --output json --report-format sarif
   ```  
   結果は `deployguard-results.sarif` に出力され、GitHub Security タブに自動アップロード。

### ベストプラクティス

- **スキャン対象の絞り込み**：`--exclude-pattern` で `vendor/`、`node_modules/` などを除外し、スキャン時間を短縮。大規模リポジトリでは `--parallelism` を 4 以上に設定し、CPU コアを最大活用。
- **定期的なポリシー更新**：`deployguard policy-update --from-nvd --schedule daily` を cron ジョブに追加。これにより、新たに発見された CVE が即座にポリシーに反映され、過去のスキャン結果も再評価される。
