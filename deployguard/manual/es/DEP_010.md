# deployguard - DEP_010

## Resumen

DeployGuard es un sistema de **análisis estático y auditoría de seguridad de código** diseñado específicamente para desarrolladores solitarios que trabajan con múltiples repositorios en entornos de CI/CD. Integra motores de detección de vulnerabilidades en tiempo real, sin depender de servicios externos. Keywords: `Code Security Audit`, `Solo Developer Tools`, `Static Analysis Tool`.

## Contenido Detallado

### ¿Qué es DeployGuard?

DeployGuard es una herramienta de **línea de comandos y API REST** que escanea el código fuente en busca de **fugas de secretos, configuraciones inseguras y dependencias obsoletas**. A diferencia de soluciones cloud, opera 100% offline, con una base de reglas actualizable vía Git. Soporta Python, JavaScript, Go y Rust, y se integra directamente con hooks de pre-commit y pipelines de GitHub Actions.

### Características Principales

- **Escaneo de Secretos en Tiempo Real con Motor Heurístico**  
  Detecta claves API, tokens OAuth, contraseñas hardcodeadas y certificados privados en archivos de texto, binarios y logs. Utiliza un modelo de **análisis de entropía (umbral > 4.5 bits/byte)** y **reglas contextuales** (por ejemplo, `aws_secret_access_key` seguido de 40 caracteres alfanuméricos). Reporta la línea exacta, el tipo de secreto y su nivel de riesgo (crítico, alto, medio). Soporta **expresiones regulares personalizadas** mediante archivos `.dg-rules.yaml`.

- **Auditoría de Dependencias con Base de Datos Local (PostgreSQL)**
