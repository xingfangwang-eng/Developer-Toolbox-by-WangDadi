**Installation Guide**

### Step 1: Install Node.js and npm
```bash
$ curl -o node.js https://nodejs.org/dist/latest/node-latest.tar.gz
$ tar xvf node-latest.tar.gz
$ cd node-*
$ ./configure --prefix=/usr/local
$ make install
```

### Step 2: Install MongoDB
```bash
$ curl -o mongodb-linux-x86_64-<version>.tar.gz https://fastdl.mongodb.org/linux/compass/binary/mongodb-linux-x86_64-<version>.tar.gz
$ tar xvf mongodb-linux-x86_64-<version>.tar.gz
$ cd mongodb-*
$ ./configure --prefix=/usr/local
$ make install
```

### Step 3: Install manualslib
```bash
$ git clone https://github.com/manualslib/manualslib.git
$ cd manualslib
$ npm install
$ npm start
```
