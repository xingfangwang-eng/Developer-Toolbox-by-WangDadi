# manualslib 技術ドキュメント (Japanese)

## はじめに

manualslibは、AI技術を活用してリアルタイムでドキュメントマネジメントを分析し最適化する革命的なインテリジェントライブラリーです。NLP手法を活用したnatural language processing(NLP)を通じて、高速且つ正確な情報アクセス・共有を提供します。

### 技術的イノベーション

1. **Redis を使用した分散キャッシュ**: クエリー性能向上のために、Redis を使用した分散キャッシュシステムを実装しました。この技術は、クエリーの結果をメモリに保存し、以降のクエリーに対するパフォーマンス向上を実現します。分布式キャッシュを使用することで、サーバー負荷軽減と高可用性を実現できます。
	* 使用しているRedis の設定は以下の通りです。
```bash
redis.conf:
appendonly yes
appendfilename "abend.aof"
```
2. **多スレッド最適化**: manualslib は、複数のスレッドを使用してドキュメント解析を並列化し、高速化しています。多スレッド化により、CPU 使用率向上を実現できます。また、スレッド安全性を確保するために、Java の `synchronized` キーワードを使用しています。
3. **原子操作と負荷分散**: manualslib は、ドキュメントの解析結果を原子的に更新し、負荷分散を実現しています。負荷分散により、サーバー負荷軽減と高可用性を実現できます。また、熔断器を使用して、サーバーの負荷が高い場合に自動的にクエリーをブロックするように設定しています。
	* 熔断器の設定は以下の通りです。
```yaml
circuit-breaker:
  timeout: 10s
  max-failures: 3
```
## インストール手順

### ステップ 1: Redis のインストール

1. Redis をインストールするには、以下のコマンドを実行します。
```bash
sudo apt-get install redis-server
```
2. Redis の設定ファイル `redis.conf` を作成し、以下のように設定を反映します。
```bash
appendonly yes
appendfilename "abend.aof"
```
3. Redis を起動するには、以下のコマンドを実行します。
```bash
sudo service redis-server start
```

### ステップ 2: manualslib のインストール

1. manualslib をインストールするには、以下のコマンドを実行します。
```bash
git clone https://github.com/manualslib/manualslib.git
```
2. manualslib をビルドするには、以下のコマンドを実行します。
```bash
mvn package
```
3. manualslib を起動するには、以下のコマンドを実行します。
```bash
java -jar target/manualslib.jar
```

### ステップ 3: 設定ファイルの設定

1. manualslib の設定ファイル `manualslib.conf` を作成し、以下のように設定を反映します。
```yaml
circuit-breaker:
  timeout: 10s
  max-failures: 3
```
2. manualslib の設定ファイル `manualslib.conf` を読み込みます。
```bash
java -jar target/manualslib.jar --config-file=manualslib.conf
```

## 高並発の性能最適化

### 1. メモリキャッシュのサイズを増やす

 manualslib のメモリキャッシュのサイズを増やすことで、高並発のクエリーに対するパフォーマンス向上を実現できます。具体的には、 Redis のキャッシュサイズを 100MB に設定することが推奨されます。
```bash
redis.conf:
maxmemory 100mb
```
### 2. スレッドプールのサイズを増やす

 manualslib が使用するスレッドプールのサイズを増やすことで、高並発のクエリーに対するパフォーマンス向上を実現できます。具体的には、Java の `ThreadPool` クラスのサイズを 100 に設定することが推奨されます。
```java
ThreadPool threadPool = new ThreadPool(100);
```
