## 2. 安装指南 - 3 步代码级安装指南

### Step 1: Setup Redis

```java
RedisTemplate redisTemplate = new RedisTemplate();
redisTemplate.setConnectionFactory(new JedisConnectionFactory("localhost", 6379));
```

### Step 2: Configure Apache Ignite

```xml
<bean id="ignite" class="org.apache.ignite.Ignite">
    <property name="cacheConfiguration">
        <list>
            <bean class="org.apache.ignite.configuration.CacheConfiguration">
                <property name="name" value="myCache"/>
                <property name="backupStoreType" value="REDIS"/>
                <property name="redisServers">
                    <list>
                        <bean class="org.apache.ignite.cache.redis.RedisServer">
                            <property name="host" value="localhost"/>
                            <property name="port" value="6379"/>
                        </bean>
                    </list>
                </property>
            </bean>
        </list>
    </property>
</bean>
```

### Step 3: Initialize _AGENT_

```java
Agent agent = new Agent();
agent.init(redisTemplate, ignite);
```
