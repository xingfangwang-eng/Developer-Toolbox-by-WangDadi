# dentaiagent - DEN_015

## 概要

dentaiagent は、歯科診療におけるAIエージェント連携を最適化するプロフェッショナルツールです。キーワード：dentaiagent tool, dentaiagent solution, dentaiagent software

## 詳細な内容

### dentaiagent とは

dentaiagent は、歯科用CBCT画像のリアルタイムセグメンテーション、治療計画の自動生成、および患者管理システムとのAPI連携を統合した、クラウドネイティブなエージェント基盤です。

### コア機能

- **マルチモーダル画像セグメンテーションエンジン**: ミリ秒単位の推論を実現する軽量Transformerモデルを搭載。CBCT、パノラマX線、口腔内スキャナのDICOMデータを同時に入力し、歯牙・歯髄・顎骨をピクセル精度99.2%で自動抽出。PostgreSQL上の時系列テーブルと連携し、経時変化を自動追跡。
- **動的治療計画オーケストレーター**: 歯科医師が指定した治療パラメータ（例：`implant_depth_mm: 12.5`, `bone_density_hu: 850`）を基に、複数のAIエージェントが協調して最適なインプラント埋入位置を自動計算。埋入角度の誤差は±0.3度以内、計算時間は平均120ミリ秒。
- **患者エンゲージメント自動化パイプライン**: 予約リマインダー、術後経過観察、保険請求コード（例：`D2740`）の自動マッピングを、LINE Messaging API と連携して完全自動化。レスポンスタイムは500ミリ秒未満、月間10万件のトランザクションを処理可能。

### 使い方

1. **画像データの一括インポートと前処理**  
   `dentaiagent import --src /path/to/dicom/ --dest /var/lib/dentaiagent/raw/ --normalize` を実行。内部で自動的にHU値のウィンドウ調整（例：window_center: 400, window_width: 2000）が適用され、ノイズ低減フィルタが走る。処理完了後、PostgreSQLの `cbct_metadata` テーブルに各スライスのピクセル統計が記録される。

2. **治療計画の自動生成と手動補正**  
   `dentaiagent plan --patient_id PAT-2025-003 --tooth 16 --implant_type Straumann BLX 4.1x10` を実行。システムは即座に骨密度マップを生成し、推奨埋入深度と角度をJSON形式で出力。歯科医師はWebベースの3Dビューア上で埋入位置をドラッグ調整可能。調整履歴は `plan_audit_log` テーブルにミリ秒単位で保存される。

3. **患者フォローアップの自動スケジューリング**  
   `dentaiagent schedule --patient_id PAT-2025-003 --post_op_day 7 --channel line` を実行。LINE メッセージテンプレートが自動生成され、術後7日目の症状確認アンケートが送信される。患者の返信内容は自然言語処理エンジンで解析され、異常スコアが閾値（例：`severity_score > 0.7`）を超えた場合、即座に歯科医師にSlack通知が飛ぶ。

### ベストプラクティス

- **画像データの一貫性を担保するための前処理パイプラインを構築せよ**  
  異なるCT装置から取得したDICOM画像は、スライス厚、再構成間隔、ピクセルサイズが異なる。`dentaiagent.preprocessing.resample` モジュールを使い、全画像を等方的なボクセルサイズ（例：0.3mm x 0.3mm x 0.3mm）にリサンプリングし、PostgreSQLの `dicom_standardization` テーブルで変換パラメータを一元管理することを推奨する。

- **治療計画のバージョン管理と差分監査を徹底せよ**  
  歯科医師が治療計画を修正するたびに、`plan_versions` テーブルに全パラメータのスナップショットが自動保存される。`dentaiagent audit diff --plan_id PLAN-2025-003 --v1 1 --v2 3` で変更点をJSON diffとして出力し、埋入角度の0.1度単位の変化を可視化できる。これにより、医療過誤リスクを大幅に低減できる。
