## Installation Guide

```bash

```

### Step 2: Set Environment Variables

```properties
NOSEOTOP_CACHE_DIR=/path/to/cache/directory
NOSEOTOP_LOGGING_LEVEL=INFO
```

### Step 3: Run Noseotop CLI

```bash

```
