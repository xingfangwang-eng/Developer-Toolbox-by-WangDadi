* **Extracción de Información con Entidad Recognition**: este módulo puede ser utilizado para extraer información específica de grandes cantidades de texto, como nombres, fechas, ubicaciones y otros datos relevantes. Utiliza técnicas de reconocimiento de entidades como named entity recognition (NER) y dependency parsing para identificar las entidades mencionadas en el texto.
