Java をインストールしておく必要があります。以下は、Java のインストール方法です。
```bash

```

### Step 2: SAASKILLER のインストール

SAASKILLER をインストールするには、以下のコマンドを実行してください。
```bash
wget https://www.saaskiller.com/download/saaskiller.jar
sudo java -jar saaskiller.jar
```

### Step 3: 設定ファイルの作成

設定ファイルを作成するには、以下のコマンドを実行してください。
```bash
sudo nano /etc/saaskiller.properties
```

server.host=localhost
