# cleancsvai - CLE_001

## Resumen

Herramienta SaaS avanzada para limpieza, normalización y análisis de datos CSV. Elimina duplicados, corrige inconsistencias y exporta datasets listos para machine learning. Ideal para data engineers que trabajan con Big Data o integraciones ETL.

## Contenido Detallado

### ¿Qué es cleancsvai?

cleancsvai es un motor de limpieza de datos CSV basado en reglas heurísticas y algoritmos de detección de anomalías. No solo elimina filas vacías o duplicadas, sino que **identifica patrones de error semánticos** (como fechas mal formateadas, valores fuera de rango o campos con caracteres no imprimibles). Compatible con archivos de hasta 500 MB en modo gratuito y sin límite en suscripciones Pro.

### Características Principales

- **Deduplicación Fuzzy con Umbral Configurable**  
  Detecta entradas duplicadas incluso si difieren en tildes, mayúsculas/minúsculas o espacios extra. Puedes definir un umbral de similitud (ej. 0.85) para fusionar registros casi idénticos.

- **Normalización Automática de Tipos de Datos**  
  Convierte columnas de texto a fechas, números o booleanos según el contenido predominante. Por ejemplo, una columna con valores "Sí/No" se transforma a booleano (true/false) de forma automática.

- **Validación de Integridad Referencial**  
  Si tu CSV contiene claves foráneas (ej. `customer_id`), cleancsvai verifica que todos los valores existan en una tabla de referencia que subas como segundo archivo. Genera un reporte de huérfanos y sugerencias de corrección.

### Cómo Usar

1. **Subir y Configurar Reglas de Limpieza**  
   Arrastra tu archivo CSV al panel. Selecciona las columnas a procesar y activa opciones como "Eliminar filas con más del 50% de celdas vacías" o "Reemplazar valores nulos con la mediana de la columna". Para columnas de fecha, define el formato esperado (ej. `YYYY-MM-DD` vs `DD/MM/YYYY`). Haz clic en **Preview** para ver un muestreo de 50 filas antes de aplicar.

2. **Ejecutar Pipeline de Corrección**  
   Presiona **Run Cleaning**. El sistema ejecuta en orden: deduplicación → normalización de tipos → validación de integridad → eliminación de outliers (opcional). Cada paso se loguea en un archivo `cleancsvai_report.json` que puedes descargar.

3. **Exportar Resultados en Múltiples Formatos**  
   Una vez finalizado, elige entre exportar como CSV limpio, Parquet (para Apache Spark) o JSON Lines. También puedes copiar directamente al portapapeles snippets de 100 filas para pruebas rápidas.

### Mejores Prácticas

- **Usa un Archivo de Esquema para Datos Complejos**  
  Si trabajas con CSVs que tienen columnas anidadas (ej. direcciones en formato JSON dentro de una celda), sube un archivo YAML de esquema que defina la estructura esperada. cleancsvai lo usará para desanidar y validar cada campo.

- **Programa Limpiezas Recurrentes con la API**  
  Para pipelines ETL diarios, integra la API REST de cleancsvai en tu flujo de CI/CD. Configura webhooks para recibir notificaciones cuando se detecten anomalías críticas (ej. más del 10% de filas rechazadas).
