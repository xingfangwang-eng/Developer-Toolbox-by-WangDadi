### ¿Qué es cleancsvai?

cleancsvai es una plataforma SaaS que aplica modelos de lenguaje avanzados para detectar, corregir y estandarizar datos inconsistentes en archivos CSV. A diferencia de herramientas tradicionales como OpenRefine o pandas, cleancsvai entiende el contexto semántico de tus columnas, eliminando duplicados ambiguos, normalizando formatos de fecha y corrigiendo errores tipográficos sin reglas predefinidas. Ideal para equipos que trabajan con datasets provenientes de múltiples fuentes (CRMs, APIs, formularios web) y necesitan una solución rápida y escalable.

### Características Principales

1. **Detección Contextual de Anomalías**: Utiliza embeddings semánticos para identificar valores atípicos basados en el significado de la columna. Por ejemplo, si una columna "País" contiene "EEUU", "USA" y "Estados Unidos", cleancsvai unifica automáticamente estas variantes sin necesidad de diccionarios manuales.

2. **Normalización Multilingüe en Tiempo Real**: Soporta más de 10 idiomas (español, inglés, alemán, japonés, etc.) para estandarizar formatos de números, fechas y direcciones. Convierte "1.234,56" (formato europeo) a "1234.56" (formato estándar) y "01/02/2023" a "2023-01-02" (ISO 8601) en segundos.

3. **Pipeline de Limpieza Reproducible**: Cada operación se registra como un pipeline de transformación exportable en JSON. Puedes compartir, versionar y reaplicar estos pipelines en nuevos datasets, garantizando consistencia en procesos ETL recurrentes.

### Cómo Usar

1. **Carga y Análisis Inicial con Perfilado Automático**: Sube tu archivo CSV (hasta 50 MB en la versión gratuita). cleancsvai ejecuta un perfilado estadístico y semántico: identifica tipos de datos (numérico, fecha, texto), detecta valores nulos (NaN, "N/A", "-") y genera un reporte de calidad con porcentajes de errores por columna. Ejemplo práctico: si subes un CSV de ventas con columnas "Fecha", "Monto", "Cliente", el sistema te muestra que el 12% de las fechas están en formato MM/DD/AAAA y el 8% en DD/MM/AAAA.

2. **Configuración de Reglas de Corrección con IA**: Selecciona las columnas problemáticas y activa la corrección inteligente. Puedes elegir entre "Corrección Automática" (la IA decide basada en patrones del dataset) o "Corrección Guiada" (especificas reglas como "Unificar nombres de países a ISO 3166-1 alfa-2"). El sistema aplica transformaciones en lotes de 1000 filas para evitar timeouts, y puedes previsualizar los cambios en una tabla interactiva antes de confirmar. Por ejemplo, para una columna "Teléfono", puedes indicar "Formato: +34 XXX XXX XXX" y cleancsvai normaliza todos los números a ese patrón.

3. **Exportación y Validación del Dataset Limpio**: Una vez aplicadas las correcciones, descarga el CSV limpio o conéctalo directamente a tu base de datos vía API REST (soporta PostgreSQL, MySQL, BigQuery). Antes de la descarga, el sistema ejecuta una validación final: cruza los datos corregidos contra un conjunto de pruebas unitarias que defines (ej. "Todos los emails deben contener @" o "Los precios deben ser > 0"). Si falla alguna validación, cleancsvai resalta las filas problemáticas y sugiere correcciones adicionales.

### Mejores Prácticas

1. **Usa Pipelines para Datasets Recurrentes**: Si procesas semanalmente un CSV de ventas desde tu CRM, crea un pipeline de limpieza en cleancsvai y aplícalo automáticamente mediante la API. Esto reduce errores humanos y acelera el proceso de 30 minutos a 2 segundos. Guarda el pipeline como un archivo JSON en tu repositorio Git para mantener trazabilidad.

2. **Combina Limpieza Automática con Validación Manual Estratégica**: No confíes ciegamente en la IA para datos críticos (ej. registros financieros o de salud). Usa la corrección automática para el 80% de los errores comunes (formatos, duplicados, typos) y luego revisa manualmente el 20% restante con el reporte de calidad. cleancsvai te permite marcar filas como "revisadas" y exportar solo las corregidas automáticamente, minimizando el riesgo de errores.
