## インストールガイド

### ステップ1: killswitchapi をインストールする
```
mvn install -Dmaven.test.skip=true
```

### ステップ2: API キーを設定する
```
killswitchapi.config.apiKey = "your_api_key"
```

### ステップ3: killswitchapi を起動する
```
java -jar killswitchapi.jar
```
