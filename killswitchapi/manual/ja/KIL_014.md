# 代码级安装指南

## Step 1: 安装依赖项 (Install Dependencies)

```
npm install express body-parser
```

### 说明：

* Express.js是一个基于Node.js的web框架。
* Body-Parser是一个中间件来处理HTTP请求体。

## Step 2: 配置 Killswitch API (Configure Killswitch API)

```
const killswitch = require('killswitch-api');

// 配置 Killswitch API
killswitch.config({
  api_key: 'your_api_key',
  secret_key: 'your_secret_key'
});
```

### 说明：

* 在上面的代码中，我们-imported杀死开关API并配置了API密钥和秘密密钥。

## Step 3: 使用 Killswitch API (Use Killswitch API)

```
// 使用 Killswitch API
const result = await killswitch.post('/api/endpoint', {
  data: 'your_data'
});

console.log(result);
```

### 说明：

* 在上面的代码中，我们使用杀死开关API来发送POST请求。
* 我们将数据作为请求体传递，并使用Promise来处理结果。
