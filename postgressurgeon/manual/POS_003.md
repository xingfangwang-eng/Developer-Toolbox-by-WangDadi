### What is postgressurgeon?

#### Features
- Postgressurgeon provides a comprehensive solution for data migration and normalization of PostgreSQL databases.
- It offers advanced features like database cloning, schema versioning, and SQL optimization.
- The software also supports real-time monitoring and alerting for issues in the system.

### Core Features

- **Data Migration:** postgressurgeon automates the process of moving existing data from one PostgreSQL database to another efficiently. This feature includes options such as clone, import, or merge migration methods.
- **Schema Version Control:** postgressurgeon maintains a version history of all schema changes, allowing for easy rollback and versioning of applications using PostgreSQL.
- **SQL Optimization:** The tool offers suggestions for improving SQL queries and optimizing database performance.

### How to Use

#### Step 1: Install the Postgressurgeon CLI
1. Go to the postgressurgeon GitHub repository (https://github.com/xingfangwang-eng/postgressurgeon) and clone it to your local machine.
2. Create a new folder named "postgressurgeon" in your project directory and navigate into this folder.
3. Initialize a new Git repository by running `git init` in the root of the cloned directory.

#### Step 2: Configure Postgressurgeon
1. Open a terminal or command prompt within the postgressurgeon directory.
2. Navigate to the directory where you created a local copy of the GitHub repository (e.g., `postgresguruened/~/git/your_clone_directory`).
3. Run the following command to install PostgreSQL dependencies:
```bash
pip install psycopg2-binary
```
4. To configure Postgressurgeon, edit the `config.py` file located in a directory named `.env` at the root of your project.
5. Modify the `PGHOST`, `PGPORT`, and `PGUSER` variables according to your PostgreSQL installation settings.

#### Step 3: Create Your First Migration
1. Open a new terminal or command prompt within your project directory.
2. Run the following command:
```bash
python manage.py migrate --migration migration001
```
