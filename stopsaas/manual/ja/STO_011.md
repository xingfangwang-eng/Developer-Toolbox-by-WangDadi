1. ** stopsaas のインストール**

```

java -jar stopsaas.jar
```

2. **stopsaas の設定**

```
# stopsaas.conf
[stop-saas]
port = 8080
workers = 4

[database]
driver = com.mysql.cj.jdbc.Driver
url = jdbc:mysql://localhost/stopsaas?user=root&password=12345678
```

3. **stopsaas の起動**

```
java -jar stopsaas.jar --config stopsaas.conf
```
