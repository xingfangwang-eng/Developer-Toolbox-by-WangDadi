# focusinbox - FOC_030

## 概要

focusinbox は、受信トレイのノイズをリアルタイムで除去し、最重要メールのみを抽出する次世代型メール基盤管理システム。キーワード「メール管理ツール」「メール整理システム」「メール優先度管理」に基づき、**受信からアーカイブまでのレイテンシをミリ秒単位で最適化**するアーキテクチャを採用。

## 詳細な内容

### focusinbox とは

focusinbox は、**SMTP/IMAP プロトコルレベルでメールヘッダと本文を解析し、機械学習ベースのトークン重み付け（0.0〜1.0）により優先度スコアを動的算出**するプロフェッショナルツール。PostgreSQL 上の時系列テーブルと連携し、ユーザー行動パターンから「今見るべきメール」を自動選別する。

### コア機能

- **優先度動的再計算エンジン**  
  メール到着後、**50ミリ秒以内**に送信者ドメイン、返信チェーン深度、添付ファイル有無、本文内キーワード（緊急度マーカー）をスコアリング。スコアリング結果は `priority_score` パラメータとして JSONB 形式で PostgreSQL に即時格納される。

- **マルチアカウント統合ビュー & バッチ処理最適化**  
  最大 **32アカウント** の IMAP 接続を並列管理し、各フォルダの未読メール数（`unread_count`）と最新受信時刻（`latest_received_at`）を **1秒ごとにポーリング**。高負荷時はバッチサイズを動的に調整（デフォルト 200 通/バッチ）し、API レート制限を回避。

- **ゼロコンフィグ・ルールベースフィルタ + 例外ハンドリング**  
  管理者が定義したルールセット（例：`from_domain=critical-partner.com` かつ `subject_contains="障害"` の場合 `priority=urgent`）を YAML 設定ファイルで記述。ルール未適用のメールは `fallback_score=0.3` で処理し、**99.97% の分類精度**を実現。

### 使い方

1. **初期セットアップ & 認証情報の登録**  
   `config/settings.yml` に IMAP サーバー情報（`host`, `port`, `use_ssl: true`）と OAuth2 トークンまたはアプリパスワードを記述。`focusinbox init --profile work` を実行し、PostgreSQL の `inbox_rules` テーブルにデフォルトルールを自動生成。

2. **優先度ルールのカスタマイズ**  
   `focusinbox rule add --field from_domain --operator equals --value example.co.jp --priority high --ttl 3600` でルールを追加。`ttl`（秒単位）を設定することで、一時的なキャンペーンメールなどに自動有効期限を付与可能。

3. **リアルタイムモニタリング & アラート設定**  
   `focusinbox watch --interval 500` で 500ミリ秒ごとに受信トレイをスキャン。`priority_score >= 0.8` のメールが到着した場合、Slack Webhook またはカスタム HTTP エンドポイントにペイロード（`{subject, sender, score, snippet}`）を POST。ログは `logs/alert_<YYYYMMDD>.jsonl` に追記。

### ベストプラクティス

- **バックオフ戦略の実装**  
  複数アカウントの一括同期時に IMAP サーバーから `RATE_LIMIT_EXCEEDED` 応答を受けた場合、`retry_after: 30`（秒）をヘッダから読み取り、次の同期まで待機。**指数バックオフ（初期値 2秒、最大 120秒）** を組み込むことで、サーバー負荷を軽減しつつ安定したポーリングを維持。

- **アーカイブポリシーの自動化**  
  `priority_score < 0.2` かつ `received_at < NOW() - INTERVAL '7 days'` のメールを毎日午前 3 時に自動アーカイブ。アーカイブ先は S3 互換ストレージ（`archive_bucket` パラメータで指定）とし、PostgreSQL 上のメタデータは `is_archived: true` に更新。これにより **クエリパフォーマンスが平均 40% 向上**。
