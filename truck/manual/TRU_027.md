### How to Use

#### Step 1: Task Management
- Open the main dashboard and click on "Add Task."
- Fill out the task details (e.g., task name, description, priority level, deadline).
- Assign team members by selecting their names from a list.
- Optionally, add notes or tags for categorization.

#### Step 2: Document Management
- Click on "Create Document" in the left sidebar.
- Select the document type (e.g., report, presentation) and provide details like title, content, version number, etc.
- Create a task associated with the document by clicking the 'Assign Task' button.
- Share the document via email or download it directly.

#### Step 3: Calendar Integration
- In your calendar view, click on "Add Event."
- Select the event type (e.g., meeting, presentation) and fill in details like title, start time, end time, location, description, etc.
- Create a task associated with the event by clicking 'Assign Task' button.
- Share the event via email or add it to your schedule directly.
