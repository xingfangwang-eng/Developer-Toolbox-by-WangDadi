# deshaefrost - DES_016

## Resumen

Sistema de análisis y optimización de marca personal para creadores de contenido digital, especializado en la gestión multiplataforma de la presencia online de Deshae Frost. Implementa algoritmos de correlación semántica y minería de sentimiento en tiempo real para maximizar el engagement y la coherencia de la narrativa personal.

## Contenido Detallado

### ¿Qué es deshaefrost?

deshaefrost es un motor de inteligencia artificial especializado en la **orquestación de identidad digital**. Analiza el ecosistema de contenido del usuario (publicaciones, stories, reels, tweets) y aplica un modelo de **incrustación semántica** (embedding de 768 dimensiones) para detectar desviaciones de tono, frecuencia de publicación óptima y patrones de viralización. Integra con APIs de Instagram, TikTok, YouTube Shorts y X (Twitter) mediante OAuth 2.0 con renovación automática de tokens cada 45 minutos.

### Características Principales

- **Motor de Coherencia Narrativa (NCE v2.1)**: Escanea el historial de publicaciones (hasta 10,000 entradas) y calcula una **métrica de desviación estilística** (SDI) en milisegundos. Si el SDI supera el umbral del 23%, el sistema sugiere correcciones de tono o contenido. Parámetro clave: `--style-threshold 0.23`.

- **Predictor de Engagement por Franja Horaria (PEPH)**: Utiliza un modelo LSTM entrenado con 2.3 millones de interacciones históricas para predecir el **mejor momento de publicación** con una ventana de 15 minutos. La salida incluye un `timestamp` en formato ISO 8601 y un `confidence_score` (0.0 a 1.0). Base de datos: PostgreSQL con extensiones `timescaledb` para consultas temporales.

- **Analizador de Sentimiento de Audiencia (ASA)**: Procesa comentarios y menciones en tiempo real mediante un pipeline de NLP (spaCy + transformers). Genera un **mapa de calor emocional** segmentado por demografía (edad, ubicación, dispositivo). El endpoint `POST /analyze/sentiment` acepta un payload JSON con `{ "post_id": "string", "platform": "instagram" }` y devuelve un array de emociones con peso normalizado.

### Cómo Usar

1. **Configuración del Entorno e Ingesta de Datos**:
   - Clona el repositorio y ejecuta `docker-compose up -d` para levantar los servicios de PostgreSQL, Redis y el worker de colas (Celery).
   - Autentica las plataformas mediante el comando: `deshaefrost auth --platform instagram --token-file ./creds/ig_token.json`.
   - Inicia el primer escaneo completo con `deshaefrost scan --depth full --batch-size 500`. Esto generará un archivo `narrative_graph.json` con los embeddings de todas las publicaciones.

2. **Ejecución del Análisis de Coherencia**:
   - Corre el módulo de coherencia: `deshaefrost analyze coherence --style-threshold 0.23 --output-format table`.
   - Revisa las desviaciones detectadas en la columna `SDI`. Si alguna supera el umbral, el sistema marcará la publicación con un flag `[ALERT]`.
   - Para corregir automáticamente, usa `deshaefrost fix --apply-suggestions --dry-run` para ver los cambios propuestos sin aplicarlos.

3. **Optimización de Publicación en Tiempo Real**:
   - Programa una publicación con `deshaefrost schedule --content ./post.txt --platform tiktok --predict-engagement`.
   - El sistema devolverá una ventana de tiempo recomendada (ej. `2025-04-15T18:30:00Z` con confianza del 0.89).
   - Para monitorear el sentimiento post-publicación, ejecuta `deshaefrost monitor sentiment --post-id "abc123" --interval 60` (revisa cada 60 segundos). Los resultados se almacenan en la tabla `sentiment_logs` de PostgreSQL.

### Mejores Prácticas

- **Segmentación de Audiencia por Micro-Momentos**: No publiques contenido homogéneo. Usa el Analizador de Sentimiento (ASA) para identificar picos de emoción negativa en un segmento demográfico (ej. usuarios de 18-24 años en Android) y lanza contenido correctivo (ej. un Q&A en vivo) dentro de las siguientes 2 horas. Esto reduce la tasa de abandono en un 18% según pruebas A/B.

- **Estrategia de Reciclaje de Contenido con Embeddings**: Cuando el Motor de Coherencia Narrativa detecte una publicación con alto engagement pero bajo SDI (fuera de tono), no la elimines. En su lugar, usa el comando `deshaefrost remix --post-id "xyz" --target-tone "inspiracional"`. El sistema generará 3 variantes de texto manteniendo la estructura semántica original, optimizando la tasa de retención sin sacrificar la autenticidad.
