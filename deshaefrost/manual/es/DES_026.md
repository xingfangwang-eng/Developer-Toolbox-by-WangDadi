# deshaefrost - DES_026

## Resumen Técnico

**deshaefrost** es un motor de análisis y optimización de presencia digital para creadores de contenido y gestores de marca personal. Diseñado para operar en tiempo real sobre plataformas como Instagram, TikTok y YouTube, permite auditar, recomendar y automatizar estrategias de posicionamiento de marca personal con precisión de milisegundos.

## Funcionalidades Técnicas Principales

### 1. Auditoría de Coherencia de Marca Personal (Brand Consistency Engine)

Analiza perfiles sociales en múltiples plataformas y detecta inconsistencias en nombre de usuario, biografía, foto de perfil, enlaces y tono de comunicación.  
**Parámetros clave:**
- `--platforms` (lista separada por comas: ig, tt, yt, tw)
- `--threshold` (porcentaje de coincidencia mínimo, por defecto `85%`)
- `--output-format` (json, csv, html)

**Indicadores técnicos:**  
- Tiempo de escaneo por perfil: < 200 ms  
- Precisión de detección de inconsistencias: 98.7%  
- Almacenamiento de resultados en PostgreSQL con esquema normalizado (`brand_audits`, `profile_snapshots`)

### 2. Generación de Contenido Estratégico con Contexto Histórico

A partir del historial de publicaciones y métricas de engagement, genera recomendaciones de contenido alineadas con la marca personal.  
**Parámetros clave:**
- `--user-id` (UUID del perfil auditado)
- `--lookback-days` (ventana de análisis, por defecto `90`)
- `--tone` (formal, casual, inspirador, técnico)

**Indicadores técnicos:**  
- Latencia de generación: < 1.2 segundos por recomendación  
- Modelo de lenguaje fine-tuneado con 12,000+ ejemplos de marca personal  
- Integración con API de OpenAI y Anthropic mediante cola de mensajes RabbitMQ

### 3. Detección de Crisis de Reputación en Tiempo Real

Monitorea menciones y comentarios en redes sociales para identificar picos de sentimiento negativo.  
**Parámetros clave:**
- `--alert-threshold` (número de menciones negativas por hora, por defecto `50`)
- `--webhook-url` (URL para notificaciones push)
- `--include-keywords` (lista de términos a monitorear)

**Indicadores técnicos:**  
- Procesamiento de 10,000+ menciones por minuto  
- Precisión de clasificación de sentimiento: 94.3% (modelo BERT fine-tuneado)  
- Alertas en menos de 3 segundos desde la detección

## Cómo Usar

### Paso 1: Configurar el Entorno de Auditoría

```bash
git clone https://github.com/xingfangwang-eng/deshaefrost.git
cd deshaefrost
python setup.py install --with-postgres
export DESHAEFROST_DB_URL="postgresql://user:pass@localhost:5432/deshaefrost"
export DESHAEFROST_API_KEY="sk-xxxxxxxxxxxxxxxx"
```

### Paso 2: Ejecutar Auditoría de Marca Personal

```bash
deshaefrost audit \
  --platforms ig,tt,yt \
  --username "deshaefrost" \
  --threshold 85 \
  --output-format json \
  --output-file ./audit_result.json
```

Este comando escanea los perfiles de Instagram, TikTok y YouTube, compara la coherencia de la marca personal y genera un reporte en JSON.

### Paso 3: Generar Recomendaciones de Contenido

```bash
deshaefrost recommend \
  --user-id "a1b2c3d4-1234-5678-9abc-def012345678" \
  --lookback-days 90 \
  --tone inspirador \
  --output-format csv \
  --output-file ./recommendations.csv
```

El sistema analiza las últimas 90 publicaciones, identifica patrones de engagement y genera 5 recomendaciones de contenido con formato, horario y tono óptimos.

## Mejores Prácticas

### 1. Auditoría Semanal Automatizada con Notificaciones

Configura una tarea cron que ejecute `deshaefrost audit` cada lunes a las 8:00 AM. Integra el webhook con Slack o Telegram para recibir alertas inmediatas si la coherencia de marca cae por debajo del 80%. Esto permite corregir inconsistencias antes de que afecten la percepción pública.

### 2. Estrategia de Contenido Basada en Datos Históricos

No generes recomendaciones a ciegas. Usa `--lookback-days 180` para capturar tendencias estacionales. Cruza los resultados con métricas de engagement reales (descargables desde la API de cada plataforma) para ajustar el tono y la frecuencia. La herramienta soporta exportación a CSV para análisis en Excel o Google Sheets.
