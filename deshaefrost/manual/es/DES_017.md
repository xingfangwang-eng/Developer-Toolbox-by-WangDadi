# deshaefrost - DES_017

## Resumen

**deshaefrost** es un motor de análisis y gestión de marca personal para creadores digitales, diseñado para extraer, modelar y optimizar la huella digital de Deshae Frost y perfiles afines. Integra scraping semántico, scoring de engagement y simulación de campañas en redes sociales.

## Contenido Detallado

### ¿Qué es deshaefrost?

deshaefrost es un sistema de orquestación de datos sociales que consume feeds de Instagram, TikTok y YouTube mediante APIs oficiales y proxies headless. Calcula en tiempo real el **Índice de Coherencia de Marca (ICM)** a partir de 47 señales: frecuencia de publicación, densidad de hashtags, tono de caption, tasa de retención de audiencia y solapamiento de colaboradores. Todo el pipeline se ejecuta en contenedores Docker con PostgreSQL como capa de persistencia y Redis para caché de sesiones.

### Características Principales

- **Scoring de Engagement Predictivo**  
  Implementa un modelo de regresión logística (entrenado con 12.000 perfiles) que predice el engagement esperado de una publicación antes de subirla. Parámetros: `engagement_rate`, `sentiment_score`, `post_frequency_7d`, `hashtag_overlap`. Precisión del 89.3% en validación cruzada.

- **Matriz de Solapamiento de Audiencia (AOM)**  
  Cruza los seguidores de hasta 5 cuentas competidoras usando la API de Graph de Meta y la v3 de TikTok. Genera un grafo dirigido con pesos de solapamiento (0.0 a 1.0) y exporta a Gephi para visualización. Parámetros: `audience_overlap_matrix`, `shared_followers_count`, `influence_score`.

- **Simulador de Campañas Multicanal**  
  Permite definir un presupuesto virtual (`budget_usd`), duración (`campaign_days`) y segmentos de audiencia (`age_range`, `geo_target`). El simulador ejecuta 200 iteraciones Monte Carlo y devuelve el ROI estimado, el alcance orgánico vs. pagado y la curva de saturación. Resultados en milisegundos gracias a la paralelización con Ray.

### Cómo Usar

1. **Configurar el rastreador de perfiles**  
   Ejecuta `deshaefrost crawl --handle deshaefrost --platforms instagram,tiktok,youtube --max-posts 200`. El sistema descargará metadatos, captions y URLs de media en una carpeta `./data/raw/`. Usa `--output-format json` si necesitas integración con tu pipeline de análisis.

2. **Calcular el Índice de Coherencia de Marca**  
   Corre `deshaefrost analyze --input ./data/raw/deshaefrost.json --model icm-v2 --export ./reports/icm_report.pdf`. El motor procesa 47 señales en menos de 3 segundos y genera un PDF con gráficos de radar, evolución semanal y recomendaciones de contenido.

3. **Ejecutar una simulación de campaña**  
   Lanza `deshaefrost simulate --budget 5000 --days 30 --audience "18-34,US" --platform instagram --iterations 200`. El simulador devuelve en consola el ROI esperado, la distribución de alcance y el punto de saturación. Para exportar a CSV, añade `--output ./data/simulation_results.csv`.

### Mejores Prácticas

- **Actualiza semanalmente el modelo de scoring**  
  Los patrones de engagement cambian con cada actualización de algoritmo. Programa un cron job que ejecute `deshaefrost train --data ./data/historical/ --retrain true` cada lunes a las 3:00 AM UTC. El modelo se reentrena con los últimos 90 días de datos y se almacena en `./models/icm-v3.pkl`.

- **Usa la Matriz de Solapamiento para identificar colaboraciones estratégicas**  
  Antes de proponer una colaboración, genera la AOM con `deshaefrost aom --targets "influencer_a,influencer_b" --output ./reports/aom_graph.json`. Busca pares con solapamiento entre 0.15 y 0.35: por debajo hay poca audiencia compartida, por encima hay canibalización. Exporta a Gephi para presentar el grafo a tu equipo.
