# deshaefrost - DES_004

## Resumen

Sistema de análisis y optimización de marca personal para creadores de contenido en redes sociales, con enfoque en métricas de engagement, coherencia de mensaje y crecimiento orgánico. Diseñado específicamente para perfiles de alto impacto como Deshae Frost.

## Contenido Detallado

### ¿Qué es deshaefrost?

deshaefrost es una plataforma de inteligencia de marca personal que analiza en tiempo real la huella digital de un creador, detecta patrones de audiencia y sugiere ajustes estratégicos para maximizar el alcance y la autenticidad del mensaje. Integra datos de múltiples plataformas (Instagram, TikTok, YouTube, X) y genera un índice unificado de coherencia de marca.

### Características Principales

- **Motor de Coherencia Semántica Cross-Platform (MCSCP)**  
  Analiza el contenido textual, visual y de tono en todas las plataformas vinculadas. Calcula un *Brand Alignment Score* (BAS) de 0 a 100, basado en 47 variables como frecuencia de publicación, uso de hashtags, paleta de colores dominante y densidad de palabras clave. El motor procesa hasta 12.000 publicaciones por minuto con una latencia de <200 ms por análisis.

- **Predictor de Engagement Viral (PEV)**  
  Modelo de machine learning entrenado con 2.3 millones de interacciones de creadores de entretenimiento y estilo de vida. Predice la tasa de engagement esperada (likes, shares, comentarios) para un contenido propuesto, con un margen de error del ±3.8%. Acepta parámetros como `tipo_de_formato` (reel, carrusel, hilo), `hora_programada` (timestamp UTC), `longitud_de_texto` (caracteres) y `frecuencia_de_publicación` (publicaciones/día). Se conecta a PostgreSQL para almacenar históricos de predicción.

- **Generador de Narrativa de Marca Personal (GNMP)**  
  Crea una línea de tiempo de contenido estratégico basada en los objetivos de crecimiento (seguidores, engagement, colaboraciones). Utiliza un algoritmo de programación dinámica que optimiza la secuencia de publicaciones para maximizar la retención de audiencia. Acepta `meta_de_seguidores` (entero), `dias_para_meta` (1-365), `recursos_disponibles` (lista de activos: fotos, videos, textos) y `restricciones_de_estilo` (array de strings como "no humor negro", "no política").

### Cómo Usar

1. **Configuración Inicial del Perfil Multired**  
   - Ejecuta `deshaefrost init --platforms instagram,tiktok,youtube --creator "@deshaefrost"` desde la CLI.  
   - El sistema solicitará las claves de API de cada plataforma (almacenadas en variables de entorno `DESHAEFROST_IG_TOKEN`, `DESHAEFROST_TT_TOKEN`, `DESHAEFROST_YT_TOKEN`).  
   - Tras la autenticación, se ejecutará un escaneo completo de los últimos 90 días de actividad. Recibirás un informe en JSON con el BAS inicial y una lista de 10 publicaciones con mayor y menor rendimiento.

2. **Análisis de Coherencia y Generación de Recomendaciones**  
   - Usa `deshaefrost analyze --depth full --output-format html` para obtener un dashboard interactivo.  
   - El dashboard muestra gráficos de evolución del BAS, mapa de calor de palabras clave y una matriz de estilo visual.  
   - Las recomendaciones se presentan como tareas priorizadas (ej: "Aumentar frecuencia de publicaciones en TikTok a 3/día para mejorar el BAS en 12 puntos en 14 días").

3. **Programación Inteligente de Contenido**  
   - Ejecuta `deshaefrost schedule --goal followers=500000 --days=90 --assets ./carpeta_activos/`  
   - El sistema generará un archivo `.ics` con las fechas y horas óptimas de publicación, junto con un informe de predicción de engagement para cada pieza.  
   - Cada entrada incluye el `engagement_predicho`, el `horario_recomendado` (en tu zona horaria local) y un `texto_sugerido` de 150 caracteres max.

### Mejores Prácticas

- **Segmentación de Audiencia por Plataforma**  
  No trates a todas las plataformas como iguales. Usa `deshaefrost segment --platform instagram --audience "seguidores_activos"` para identificar qué contenido funciona mejor en cada red. Por ejemplo, los reels de humor funcionan mejor en TikTok (engagement +40%), mientras que los carruseles educativos tienen mejor rendimiento en Instagram (engagement +25%). Ajusta el tono y formato según la plataforma, no según el creador.

- **Ciclo de Revisión Quincenal de BAS**  
  Programa una revisión automática cada 15 días con `deshaefrost cron --interval 15d --action analyze --alert-on-drop 5`. Si el BAS cae más de 5 puntos, recibirás una notificación con las tres publicaciones que más contribuyeron a la caída. Esto permite corregir desviaciones de marca antes de que afecten el crecimiento a largo plazo.
