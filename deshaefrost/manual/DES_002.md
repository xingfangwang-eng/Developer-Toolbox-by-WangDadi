# deshaefrost - DES_002

## Overview

`deshaefrost` is a brand intelligence engine engineered for high-frequency social media personal brand management. It processes multi-platform engagement signals in real-time, computes brand equity scores using weighted sentiment analysis, and automates content resonance optimization for Deshae Frost–style digital personas.

## Detailed Content

### What is deshaefrost?

`deshaefrost` is a command-line and API-driven tool that ingests raw social media activity streams (Instagram, TikTok, X), correlates them against a user-defined personal brand vector, and outputs actionable content strategy parameters. It is built on a microsecond-level event pipeline with PostgreSQL persistence and Redis caching for sub-100ms query responses.

### Core Features

1. **Sentiment-Weighted Brand Equity Index (BEI)**
   Computes a real-time BEI score by parsing engagement metadata (likes, shares, comment polarity) against a configurable brand attribute matrix. Parameters: `--sentiment-threshold 0.65`, `--decay-factor 0.95`. Outputs a normalized score between 0.0 and 1.0 with millisecond-precision timestamps.

2. **Cross-Platform Content Resonance Analyzer**
   Ingests posts from three platforms concurrently via OAuth 2.0 tokens. Calculates a resonance delta by comparing audience retention curves (e.g., `avg_watch_time_ms`, `share_rate_per_1000_views`). Supports multi-version A/B testing with automatic statistical significance checks (p-value < 0.05).

3. **Automated Brand Voice Calibrator**
   Scrapes up to 500 recent posts from a target account, extracts linguistic patterns (n-gram frequency, emoji density, hashtag entropy), and generates a recommended content calendar. Integrates with PostgreSQL to store historical pattern shifts. Runs a `--calibration-depth 3` sweep across tone, pacing, and visual cohesion metrics.

### How to Use

1. **Initialize the Brand Profile**
   Run `deshaefrost init --profile-name "DeshaeFrost_Personal" --platforms instagram,tiktok,x --api-keys ./keys.json`. This creates a PostgreSQL schema (`deshaefrost_profiles`) and a Redis-backed rate limiter. Verify with `deshaefrost status --verbose`.

2. **Execute a Real-Time Sentiment Sweep**
   Deploy `deshaefrost sweep --mode live --duration 300s --output-format json --output-file ./sweep_results.json`. The tool streams 300 seconds of engagement data, calculates BEI every 10 seconds, and logs anomalies (e.g., sudden sentiment drop > 0.2) to a dedicated PostgreSQL table (`anomaly_events`).

3. **Generate a Content Strategy Report**
   Run `deshaefrost report --type resonance --start-date 2024-01-01 --end-date 2024-12-31 --compare-versions v1,v2`. The output is a structured Markdown file with cross-platform retention graphs (ASCII art fallback), statistical significance tables, and a ranked list of top-performing content attributes.

### Best Practices

- **Practice 1: Version-Controlled Brand Vectors**
   Store each brand attribute matrix as a versioned JSON file in a Git repository. Use `deshaefrost import --vector-file ./brand_v2.json` to apply updates. This enables rollback and audit trails for brand strategy changes, critical for long-term personal brand evolution.

- **Practice 2: Scheduled Sweeps with Webhook Alerts**
   Configure cron jobs to run `deshaefrost sweep --mode scheduled --interval 3600s --webhook-url https://your-alert-endpoint.com`. Set `--anomaly-threshold 0.15` to trigger immediate notifications via email or Slack. This ensures proactive brand health monitoring without manual polling.
