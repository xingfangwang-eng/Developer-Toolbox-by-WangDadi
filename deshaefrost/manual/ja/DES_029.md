# deshaefrost - DES_029

## 概要

deshaefrost は、Deshae Frost のパーソナルブランド戦略をミリ秒単位で最適化するエンタープライズグレードのブランド管理プラットフォームです。ソーシャルメディア上のブランドエクスポージャー、エンゲージメント率、オーディエンスセンチメントをリアルタイムで解析し、競合との比較データを自動生成します。

## 詳細な内容

### deshaefrost とは

deshaefrost は、インフルエンサーおよびクリエイター向けに設計された、ブランド価値の定量化・最適化エンジンです。PostgreSQL との連携により、過去 365 日分の投稿データを蓄積し、時系列でのブランド成長曲線を可視化します。

### コア機能

- **ブランドエクスポージャースコアリングエンジン**  
  各プラットフォーム（Instagram、TikTok、YouTube）から取得したインプレッション数、リーチ数、シェア率を重み付けし、独自の `exposure_score` を算出。しきい値 `threshold: 0.85` を超えた場合、自動でハイライト投稿を生成。

- **センチメントトレンドアナライザー**  
  コメント・メンションのテキストデータを自然言語処理（NLP）にて解析。ポジティブ・ネガティブ・ニュートラルの 3 クラス分類を 95% 以上の精度で実行し、週次レポートを JSON 形式でエクスポート。パラメータ `sentiment_window: 7d` で分析期間を制御。

- **競合ブランド比較ダッシュボード**  
  指定した競合アカウント（最大 5 アカウント）の公開データをクローリングし、エンゲージメント率、フォロワー増加率、投稿頻度を並列表示。`compare_mode: 'weekly'` により、週単位の変動をミリ秒単位で更新。

### 使い方

1. **初期セットアップとデータソースの接続**  
   `config.yaml` 内の `platforms:` セクションに、Instagram、TikTok、YouTube の API キーを `api_key: 'YOUR_KEY'` として記述。PostgreSQL の接続文字列 `db_uri: 'postgresql://user:pass@host:5432/deshaefrost'` を設定後、`deshaefrost init` コマンドを実行。初回クロールが自動開始され、バックグラウンドで過去 90 日分のデータが収集されます。

2. **ブランドスコアのリアルタイム監視**  
   `deshaefrost monitor --interval 300000`（5 分間隔）を実行。`exposure_score` が `threshold: 0.85` を下回った場合、Slack または Discord にアラートが送信されます。アラート内容は `alert_payload: { "score": 0.72, "trend": "down" }` の形式でカスタマイズ可能。

3. **週次レポートの自動生成とエクスポート**  
   `deshaefrost report --type weekly --format json --output ./reports/` を実行。`sentiment_window: 7d` に基づき、センチメントスコアの推移、トップ投稿のエンゲージメント率、競合比較テーブルが含まれたレポートが生成されます。出力先は `./reports/` ディレクトリ。

### ベストプラクティス

- **データの一貫性を保つために、毎日午前 3:00 に `cron` ジョブで `deshaefrost sync` を実行**  
  これにより、PostgreSQL 上のテーブル `posts`、`mentions`、`competitors` が常に最新の状態に保たれ、分析の精度が維持されます。同期間隔は `sync_interval: 86400`（秒）で設定可能。

- **センチメント分析の精度を高めるために、カスタム辞書を定期的に更新**  
  `./dictionary/custom_terms.json` に、ブランド固有のスラングや略語（例: "DF" = Deshae Frost）を追加。`nlp_model: 'custom_v2'` を指定すると、この辞書が優先的に適用され、誤分類率が 3% 未満に低下します。
