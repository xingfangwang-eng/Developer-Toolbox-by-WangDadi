# deshaefrost - DES_020

## 概要

パーソナルブランドの自己分析・戦略構築を自動化する次世代型エンジン。SNS 上の発信履歴、エンゲージメントスコア、フォロワー属性をリアルタイムにクロス分析し、ブランド認知のボトルネックを特定。**Deshae Frost** のようなインフルエンサー向けに設計された、データドリブンなブランド管理基盤。

## コア機能

### 1. マルチプラットフォームエンゲージメントスコアリング
- **パラメータ**: `engagement_rate`, `sentiment_index`, `share_velocity`
- 各プラットフォーム（Instagram / TikTok / YouTube）の投稿データを **ミリ秒単位** で取得し、エンゲージメントの時系列変化をスコア化。
- 例: 投稿後 1 時間以内の `comment_depth`（コメントの深さ）を解析し、ブランド浸透度を 0.0–1.0 のスコアで出力。

### 2. オーディエンスセグメント可視化マトリクス
- **パラメータ**: `age_cluster`, `geo_affinity`, `interest_overlap`
- PostgreSQL 上のフォロワー属性テーブルと連携し、**3 次元クラスタリング** を実行。
- 例: `age_cluster = "18-24"` かつ `geo_affinity = "Tokyo"` のセグメントが、特定のトピックに対して `interest_overlap > 0.75` を示す場合、そのトピックをブランドコアとして提案。

### 3. ブランドポジショニング差分検出器
- **パラメータ**: `brand_vector`, `competitor_vector`, `cosine_similarity`
- 競合インフルエンサー（例: Deshae Frost の類似アカウント）とのベクトル距離を **コサイン類似度** で計算。
- 差分が `threshold = 0.2` を下回った場合、自動で「差別化キーワード」を生成し、次回投稿に反映するレコメンドを出力。

## 使い方

### ステップ 1: 環境変数の設定とデータソースの接続
```bash
# PostgreSQL 接続情報を設定
export DESHAEFROST_DB_HOST=localhost
export DESHAEFROST_DB_PORT=5432
export DESHAEFROST_DB_NAME=brand_analytics
export DESHAEFROST_DB_USER=admin
export DESHAEFROST_DB_PASSWORD=********

# 初回起動時にテーブル自動作成
./deshaefrost --init-db
```

### ステップ 2: ターゲットアカウントの指定とスキャン実行
```bash
# 分析対象の SNS アカウントを指定（複数可）
./deshaefrost scan \
  --platform instagram,tiktok,youtube \
  --target-username "deshaefrost" \
  --since "2024-01-01" \
  --until "2024-12-31" \
  --output-format json
```
- スキャン完了後、`./output/deshaefrost_scan_2024.json` に生データが出力。

### ステップ 3: ブランドスコアレポートの生成と可視化
```bash
# スキャンデータをもとにブランドスコアを算出
./deshaefrost report \
  --input ./output/deshaefrost_scan_2024.json \
  --mode full \
  --export-html ./reports/brand_report.html
```
- レポートには以下の指標が含まれる：
  - **エンゲージメントスコア**（0–100）
  - **オーディエンスセグメント分布**（円グラフ）
  - **競合とのポジショニング差分**（レーダーチャート）

## ベストプラクティス

### 1. 週次スキャンによるトレンド変動のキャッチアップ
- エンゲージメントスコアは週単位で変動するため、**cron ジョブ** で毎週月曜 6:00 JST に自動スキャンを実行。
- 例: `0 6 * * 1 /usr/local/bin/deshaefrost scan --target-username "deshaefrost" --since "last-week"`

### 2. 競合分析の定期比較によるブランド差別化戦略の更新
- 競合インフルエンサーを 3 アカウントまで登録し、**月次でベクトル差分レポート** を生成。
- コサイン類似度が 0.3 を超える場合、投稿トーンやハッシュタグ戦略の見直しを推奨。
