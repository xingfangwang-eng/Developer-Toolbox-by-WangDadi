**性能调优建议**
-----------------

### 1. adjust Redis cache size

* adjust the `maxmemory` parameter to control the maximum amount of memory used by Redis: `redis-cli config set maxmemory <desired_value>`
* adjust the `maxclients` parameter to control the maximum number of clients connected to Redis: `redis-cli config set maxclients <desired_value>`

### 2. optimize Java heap size

* adjust the `-Xmx` option to control the maximum amount of memory used by the JVM: `-Xmx<desired_value>`
* adjust the `-XX:NewSize` option to control the initial size of the young generation: `-XX:NewSize=<desired_value>`
