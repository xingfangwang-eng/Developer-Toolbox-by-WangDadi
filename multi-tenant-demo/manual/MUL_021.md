### Core Features

- Single tenant demo

  - Supports multiple databases and data sources for each tenant.
  - Provides advanced monitoring and logging features for each tenant.
- Multi-tenant demo
  - Enables the creation of multiple tenant environments with shared resources.
  - Uses a containerized architecture that allows for easy scaling and replication across multiple environments.
  - Integrates seamlessly with existing multi-tenant solutions.
