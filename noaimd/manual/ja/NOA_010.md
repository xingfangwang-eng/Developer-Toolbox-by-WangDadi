**Installation Guide**

### Step 1: Install Redis

```
sudo apt-get install redis-server
```

### Step 2: Configure Redis for noaimd

```java
redis.conf:
maxmemory 10000mb
maxclients 1000
```

### Step 3: Start Redis

```
sudo service redis-server start
```
