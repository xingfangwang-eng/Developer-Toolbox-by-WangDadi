# champions-sp-calc - CHA_016

## Resumen Técnico

**champions-sp-calc** es una herramienta SaaS de cálculo de puntos de campeón optimizada para SEO y rendimiento, diseñada para desarrolladores que necesitan precisión milimétrica en la asignación de recursos dentro de entornos competitivos. Esta solución elimina la complejidad de los cálculos manuales y ofrece una integración directa con pipelines de CI/CD.

## Características Principales (Hardcore Features)

### 1. Motor de Cálculo Asíncrono con Web Workers
Implementamos un motor de cálculo que ejecuta operaciones complejas de puntuación en segundo plano utilizando **Web Workers**. Esto permite procesar simultáneamente hasta **10,000+ combinaciones de campeones** sin bloquear el hilo principal del navegador, garantizando una latencia de respuesta inferior a 50ms incluso en dispositivos de gama baja.

### 2. Algoritmo de Ponderación Dinámica con Regresión Lineal
El sistema aplica un modelo de **regresión lineal multivariable** que ajusta automáticamente los pesos de cada estadística (daño, resistencia, movilidad, etc.) en función del meta actual. Los datos se actualizan cada 24 horas desde una API RESTful, lo que asegura que los cálculos reflejen las tendencias del juego en tiempo real.

### 3. Exportación de Datos en Formato JSON/CSV con Schema Validado
Cada cálculo genera un objeto JSON que sigue un esquema **OpenAPI 3.0** estricto, con campos como `champion_id`, `score_total`, `confidence_interval` y `timestamp`. Además, se puede exportar directamente a CSV para su uso en herramientas de análisis como **Tableau** o **Google Sheets**, facilitando la auditoría y el reporting.

## Cómo Usar (Pasos Detallados)

### 1. Configuración del Entorno Local
Clona el repositorio y ejecuta `npm install` para instalar las dependencias. Luego, inicializa el archivo `.env` con las variables de entorno necesarias: `API_BASE_URL`, `CACHE_TTL=3600` y `LOG_LEVEL=debug`. Asegúrate de tener Node.js v18+ y un navegador compatible con ES2022.

### 2. Ejecución de un Cálculo Personalizado
En la consola del navegador, llama a la función `window.championsSPCalc.run({ champions: ['Aatrox', 'Zed', 'Lux'], weights: { damage: 0.4, defense: 0.3, utility: 0.3 }, metaVersion: '14.10' })`. Esto devolverá una promesa con el resultado en milisegundos. Puedes modificar los pesos dinámicamente sin recargar la página.

### 3. Integración en un Pipeline de CI/CD
Agrega el siguiente script a tu archivo `package.json`:
```json
"scripts": {
  "calc:batch": "node scripts/batch-calc.js --input data/champions.json --output results.csv"
}
```
El script leerá un archivo JSON con una lista de campeones, ejecutará el motor de cálculo y generará un CSV listo para ser consumido por tu sistema de análisis.

## Mejores Prácticas para Desarrolladores

### 1. Cachea Resultados con Service Workers
Implementa un **Service Worker** que almacene en caché los resultados de cálculos frecuentes (por ejemplo, combinaciones populares de campeones). Esto reduce la latencia en un 70% y disminuye la carga en el servidor. Usa `Cache-Control: max-age=3600` en las respuestas de la API.

### 2. Monitoreo con Prometheus y Grafana
Expón métricas personalizadas como `champions_sp_calc_requests_total`, `champions_sp_calc_duration_seconds` y `champions_sp_calc_errors_total` a través de un endpoint `/metrics`. Conecta esto a **Prometheus** y crea dashboards en **Grafana** para visualizar picos de uso y cuellos de botella en tiempo real.
