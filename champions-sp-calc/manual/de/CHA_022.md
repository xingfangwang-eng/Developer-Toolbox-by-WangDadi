# champions-sp-calc - CHA_022

## Übersicht

Dies ist ein professionelles Tool zur Berechnung und Optimierung von Champion-Skill-Punkten in kompetitiven Spielumgebungen. Es unterstützt Entwickler und Spieler dabei, die effizienteste Skill-Punkt-Verteilung für maximale Kampfkraft zu ermitteln. Das Tool eignet sich besonders für Meta-Analysen und Patch-Planung.

## Detaillierter Inhalt

### Was ist champions-sp-calc?

champions-sp-calc ist ein spezialisiertes Berechnungstool, das auf Basis von Echtzeit-Meta-Daten und Champion-Statistiken die optimale Skill-Punkt-Verteilung berechnet. Es ermöglicht die simulation von Build-Pfaden und analysiert die Auswirkungen von Patches auf die Effektivität einzelner Fähigkeiten. Das Tool ist ideal für Spielentwickler, die Balancing-Änderungen testen, sowie für Spieler, die ihre Spielstrategie optimieren möchten.

### Kernfunktionen

- **Meta-Adaptive Skill-Punkt-Optimierung**: Berechnet dynamisch die optimale Skill-Punkt-Verteilung basierend auf aktuellen Champion-Statistiken, Patch-Notizen und Spiel-Meta. Das Tool analysiert automatisch, welche Fähigkeiten in welcher Reihenfolge maximiert werden sollten, um die höchste Winrate zu erzielen.

- **Patch-Delta-Analyse**: Vergleicht Skill-Punkt-Verteilungen zwischen verschiedenen Patch-Versionen und zeigt exakte Änderungen in der Effektivität einzelner Fähigkeiten an. Dies ermöglicht eine präzise Anpassung an Balancing-Änderungen und neue Meta-Trends.

- **Multi-Szenario-Simulation**: Simuliert verschiedene Spielszenarien (Early Game, Mid Game, Late Game) und berechnet die optimale Skill-Punkt-Verteilung für jede Phase. Das Tool berücksichtigt Faktoren wie Gegner-Konter, Team-Komposition und Spielstil.

### So verwenden Sie es

1. **Champion-Daten importieren und Meta-Analyse durchführen**:  
   Importieren Sie die Champion-Daten über die API-Schnittstelle oder laden Sie eine JSON-Datei mit den aktuellen Statistiken. Das Tool führt automatisch eine Meta-Analyse durch und identifiziert die relevantesten Fähigkeiten basierend auf Winrate, Pickrate und Banrate. Verwenden Sie den Befehl:  
   `champions-sp-calc --import champion_data.json --meta-analyse`

2. **Skill-Punkt-Verteilung berechnen und simulieren**:  
   Wählen Sie den gewünschten Spielmodus (z.B. Solo Queue, Competitive) und die Phase (Early, Mid, Late). Das Tool berechnet die optimale Verteilung und zeigt eine detaillierte Tabelle mit den erwarteten Effektivitätswerten. Nutzen Sie die Simulation, um verschiedene Build-Pfade zu testen:  
   `champions-sp-calc --simulate --mode solo-queue --phase mid-game`

3. **Ergebnisse exportieren und in das Spiel integrieren**:  
   Exportieren Sie die berechnete Verteilung als JSON oder CSV und integrieren Sie sie in Ihre Spiel-Config oder Analyse-Tools. Das Tool bietet auch eine direkte API-Anbindung für Echtzeit-Updates. Exportieren Sie mit:  
   `champions-sp-calc --export --format json --output optimal_build.json`

### Beste Praktiken

- **Patch-Tage nutzen**: Führen Sie die Meta-Analyse immer am Tag eines neuen Patches durch, um die aktuellsten Daten zu verwenden. Das Tool kann automatisch Patch-Notizen abrufen und die Analyse entsprechend anpassen. Dies stellt sicher, dass Ihre Skill-Punkt-Verteilung immer auf dem neuesten Stand ist.

- **Multi-Szenario-Vergleich**: Vergleichen Sie immer mehrere Szenarien (Early, Mid, Late) und wählen Sie die Verteilung, die für Ihre bevorzugte Spielphase am effektivsten ist. Wenn Sie z.B. auf Early Game fokussieren, priorisieren Sie Fähigkeiten mit hohem Burst-Schaden, während für Late Game eher Scaling-Fähigkeiten wichtig sind.
