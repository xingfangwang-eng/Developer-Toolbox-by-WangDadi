## How to Use

1. **Configure data sources and fraud rules**  
   Create a `config.yaml` file with your API tokens and fraud parameters:
   ```yaml
   sources:
     - type: twitter
       bearer_token: "your_bearer_token"
       query: "#giveaway -is:retweet"
   fraud:
     min_account_age_minutes: 1440
     max_entries_per_minute: 3
   ```
   Run `giveawaytracker --config config.yaml --validate` to test connectivity.

2. **Launch the ingestion pipeline and monitor live metrics**  
   Start the daemon with:
   ```bash

   ```
   Access the Prometheus‚Äëcompatible metrics endpoint at `http://localhost:8080/metrics` to track `giveaway_events_total`, `fraud_flagged_total`, and `winner_selected_total` with millisecond latency histograms.

3. **Trigger winner selection for a specific giveaway**

   ```bash

        -H "Content-Type: application/json" \
   ```
